dbo.ActionStatistics

This table tracks the number of files that have gone through each action, and the current count of them in that action. This allows overal state of the database/workflow to be reported without a long-running query to aggregate the stats of all files. This data is displayed on the summary page of the database administration tool and in the toolbars of various Extract applications.

Note: The data in this table will generally not be entirely up-to-date while processing is active as updates to the rows here will be done in batches. Up-to-date stats can be calculated by aggregating file status changes from the ActionStatisticsDelta table with the data here.

ActionID
Forigen key to Action.
Invisible
Indicates whether the counts in this row are visible to Extract APIs. Invisible files represent files for which deletion was requested via API.
LastUpdateTimeStamp
The last time these stats were updated.
NumDocuments
The total number of documents that have gone through this action.
NumDocumentsPending
The number of documents pending in this action.
NumDocumentsComplete
The number of documents that have been completed in this action.
NumDocumentsFailed
The total number of documents that have failed in this action.
NumDocumentsSkipped
The total number of files that have been skipped.
NumPages
The total number of pages that have been procesed.
NumPagesPending
The total number of pages pending in this action.
NumPagesComplete
The total number of pages that have been completed in this action.
NumPagesFailed
The number of pages that have failed in this action.
NumPagesSkipped
The number of pages that have been skpped in this action.
NumBytes
The number of bytes in this action. A byte is a measurement of size, and we are specifically measuring the file size.
NumBytesPending
The number of bytes pending.
NumBytesComplete
The number of bytes completed.
NumBytesFailed
The number of bytes failed.
NumBytesSkipped
The number of bytes skipped.

dbo.ActionStatisticsDelta

In order to allow ActionsStatistics to be updated when the database is under a heavy processing load without causing contention updating the data in ActionStatistics rows, changes made to file action statuses are collected here where each change will be a separate row (preventing the need to lock any rows for most action status changes here). Periodically, the action status changes collected here are aggregated into the corresponding rows of the ActionStatistics table.

ID
A unique identifier
ActionID
Forigen key to Action.
Invisible
Indicates whether the counts in this row are visible to Extract APIs. Invisible files represent files for which deletion was requested via API.
NumDocuments
Number of new documents in the current action/visibility state as a result of the file action state change; will be 0, 1 or -1
NumDocumentsPending
Number of new pending documents in the current action/visibility state as a result of the file action state change; will be 0, 1 or -1
NumDocumentsComplete
Number of new completed documents in the current action/visibility state as a result of the file action state change; will be 0, 1 or -1
NumDocumentsFailed
Number of new failed documents in the current action/visibility state as a result of the file action state change; will be 0, 1 or -1
NumDocumentsSkipped
Number of new skipped documents in the current action/visibility state as a result of the file action state change; will be 0, 1 or -1
NumPages
Number of new pages in the current action/visibility state as a result of the file action state change; will reflect the number of pages in the affected document
NumPagesPending
Number of new pending pages in the current action/visibility state as a result of the file action state change; will reflect the number of pages in the affected document
NumPagesComplete
Number of new completed pages in the current action/visibility state as a result of the file action state change; will reflect the number of pages in the affected document
NumPagesFailed
Number of new failed pages in the current action/visibility state as a result of the file action state change; will reflect the number of pages in the affected document
NumPagesSkipped
Number of new skipped pages in the current action/visibility state as a result of the file action state change; will reflect the number of pages in the affected document
NumBytes
Number of new bytes in the current action/visibility state as a result of the file action state change; will reflect the number of bytes in the affected document
NumBytesPending
Number of new pending bytes in the current action/visibility state as a result of the file action state change; will reflect the number of bytes in the affected document
NumBytesComplete
Number of new completed bytes in the current action/visibility state as a result of the file action state change; will reflect the number of bytes in the affected document
NumBytesFailed
Number of new failed bytes in the current action/visibility state as a result of the file action state change; will reflect the number of bytes in the affected document
NumBytesSkipped
Number of new skipped bytes in the current action/visibility state as a result of the file action state change; will reflect the number of bytes in the affected document
