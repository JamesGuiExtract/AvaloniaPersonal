Switching Between ZACH and ZEKE

    From ESXI Console

    • Click F12 for shutdown / restart.
    • Provide user name (= root) and password (= <normal local admin password>)
    • Click F11 to confirm the restart.

    Reboot the Server

    • Click Ctrl+C at the appropriate time to enter SAS Configuration pages.
    • Confirm that the page caption is LSI Corporation Configuration Utility. Click Enter.
    • Confirm that the page caption is Select SAS Topology.
    • Select SAS Topology -- SAS 2008. Click Enter.
    • Select the desired drive
      • Select 0 for ESXi and ZEKE.
      • Select 1 for Windows and ZACH.
    • Click Alt+B to make the selected drive the bootable drive. The effect will be one or the other of the following.
      • SATA,Boot to boot ESXi
      • SATA

      • SATA
      • SATA,Boot to boot Windows

    • Click Esc to exit.
    • Select Save changes.
    • Click Esc to exit.
    • Click Esc to exit.
    • Select Exit and reboot the server.
