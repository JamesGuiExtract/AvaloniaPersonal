Eliminate Duplicates Output Handler

Eliminate Duplicates

  • If the same text is found multiple times by the rules, the Eliminate Duplicates output handler will remove all but one instance of the found text.
  • Eliminate Duplicates will only remove attributes that have the same name, value, and type.
  • This output handler does NOT ignore whitespace.

  • Example:
    • Open Phone.rsd and run it on Phone.txt.
    • Notice that the phone number 333-5555 is caught three times.
    • Add an Eliminate Duplicates output handler to the .rsd file and run the rules again. Note the difference in output.
