Remove Characters Output Handler

Uses

  • The Remove Characters output handler is used to trim, remove, or consolidate specified characters in the found data.
  • For example, you can consolidate white space, remove the period from "Mr.", or trim the dollar sign from dollar amounts.

Usage

  • To use the Remove Characters output handler, you must first specify the characters you want to remove.
    • Because removing white space is so common, there are checkboxes to specify different kinds of white space.
    • You can specify other characters by selecting Others and then typing the desired characters into the edit box.
    • You can make the characters case sensitive.
  • Next, you need to select what you want to do with the specified characters.
    • You can remove all of the characters.
    • You can consolidate multiple characters into one.
    • You can trim off leading or trailing characters.

  • Example:

    • Open RemoveCharacters.rsd and run it on RemoveCharacters.txt.
    • This rules file is finding names and dollar amounts.
    • Add a Remove Characters output handler that removes the commas from the dollar amounts. Run the rules again.
    • We want to remove the period after the middle initial. Add a Remove Characters output handler that removes periods. Run the rules again and notice how this output handler also effects the dollar amount.
    • Change the output handler to consolidate the periods. Run the rules again and note the difference in the output.
    • Change the output handler to trim the leading dollar sign from the beginning of the dollar amounts. Run the rules again.
