Run Object On Query Output Handler

Uses

  • The Run Object On Query output handler is used to run an Attribute Modifier, Output Handler, or Attribute Splitter on a specified attribute or sub-attribute.

Usage

  • To use the Run Object On Query output handler, you must first query the attributes that you wish to run the selected object on.
    • To query a single attribute, simply enter that attribute's name into the query box.
    • To query multiple attributes, enter each attribute's name separated by |'s.
      • PhoneNumber|Name will run the selected rule object on both the PhoneNumber and Name attributes.
    • To query a sub-attribute, enter the query as AttributeName/SubattributeName.
      • Name/Middle will run the selected rule object on the Middle sub-attribute of Name.
    • To query an attribute with a certain type, enter the query as AttributeName@AttributeType
      • Name@Buyer will run the selected rule object on the Name attribute if it has a type of Buyer.
    • To query an attribute only if it has a certain sub-attribute, enter the query as AttributeName{/SubattributeName}.
      • PhoneNumber{/AreaCode} will run the selected rule object on the PhoneNumber attribute only if it has an AreaCode sub-attribute.
    • To query all attributes, enter * as the query
      • Name/* will run the selected rule object on the all of the sub-attributes of Name.

Exercises

  1. Open RemoveCharacters.rsd and run it on RemoveCharacters.txt. Note that this .rsd file finds both names and dollar amounts.
    • We want to remove the periods from the names without removing the decimal points from the dollar amounts.
    • Add a Remove Characters output handler that removes periods. Run the rules and note that the periods are removed from both the names and the dollar amounts.
    • Remove the previous output handler and add a Run Object on Query output handler that queries the attribute Name. Have the query execute a Remove Characters output handler that removes periods.
    • Run the rules. Note that the periods are now only removed from the names.

  2. Open Names.rsd and run it on Names.txt.
    • Add a Run Object On Query output handler to the current Select Multiple Output Handlers output handler that will capitalize the data in the First sub-attribute of the Name attribute without capitalizing the data in the Middle or Last sub-attributes.
