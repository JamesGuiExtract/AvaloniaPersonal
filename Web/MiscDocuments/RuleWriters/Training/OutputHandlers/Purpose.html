Output Handler Purpose and Basics

Uses

  • Output handlers are generally used to clean up and format the found data in preparation for final output.
    • You can use the Remove Characters output handler to remove the dollar sign from a dollar amounts.
    • You can use a replace strings to change "Dr." into "Doctor".
  • They are also used to output the found data into a .voa or .xml file.
  • Output handlers are similar to Value Modifiers; however, Value Modifiers only affect one attribute rule, while Output Handlers affect all of the attribute rules in an .rsd file.

Basics

  • Output handlers can be added by clicking on the Commands button and selecting the Select Output Handlers... option.
    • This will bring up a dialog where you can choose which type of output handler you want and give it a description.
  • You can modify an existing output handler using the Commands button.
    • Choosing the Select Output Handler... option will let you change the type of the output handler.
    • Choosing the Configure Output Handler... option will let you change the settings of the output handler.
    • Choosing the Configure description... option will let you change the description of the output handler.
  • You can remove the output handler by changing its type to <None>.
  • If you right click on the output handler, it will bring up a context menu that you can use to Cut, Copy, Paste, and Delete the output handler.
  • You can turn off the output handler by un-checking the checkbox in front of it. This can be useful for testing.
