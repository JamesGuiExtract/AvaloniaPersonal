Create Value Value Finder

Uses

  • Creates an arbitrary value without spatial data.

Usage

  • In the Add Attribute Finding Rule window:
    • Select Create value from the dropdown menu and click Configure....
    • Fill in the desired Value and/or Type in their respective fields.
    • These fields also resolve tags, such as <DocType>, <RSDFileDir>, or <SourceDocName>.

Exercises

  1. Open the RuleSet Editor and load ".\Files\CreateValue.rsd". Open the Rule Tester and load ".\Files\CreateValue.tif". Execute the rules.
  2. Note that a file named ".\Files\CreateValue.tif.voa" was created. Open it using the VOA File Viewer. Note that it contains an entry named 'SSN' with a value of '000-00-0000'.
  3. We would like the .voa file to also contain Document Type data. Make a new attribute named 'DocumentType' that accomplishes this. Note that the Create Value value finder expands tags.
  4. The value of the DocumentType entry should be 'Notice of State Tax Lien'.
