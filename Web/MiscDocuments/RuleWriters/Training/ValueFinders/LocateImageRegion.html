Locate Image Region Value Finder

Uses

  • The Locate Image Region value finder is used to redact or extract text from a specified region of a document.
  • It uses the position of specified clues in the document to find the desired region.

Usage

  • To use the Locate Image Region value finder, you must first specify whether you want to find an image region or text.
    • Image regions contain only the spatial information of the found region.
      • They are primarily used to redact the region in the ID Shield product.
      • When you create an image region, you will have to assign some "placeholder" text for the found spatial information in the Assign the following text to the image region box.
    • If you choose to find text, the Locate Image Region value finder will extract and return all of the text contained in the found region.
      • This option is primarily used to extract data from a certain region of the document for the FLEX Index product.
      • You can choose to whether to include or exclude characters or words that intersect the boundaries of the region.
  • You can choose whether to return only the first region that is found or to return all of the regions that are found.
  • Next, you will need to specify the clues that will be used to find the region as well as where the region is located in relation to the specified clues.
    • Clues can be added, deleted, and modified using the Add..., Remove, and Modify... buttons.
      • The clues can be plain text or regular expressions.
      • The clues can be made case sensitive.
      • You can have up to four separate clue lists that you can use to find the image region.
    • You can specify were the region is relative to the clues by using Region Boundaries box.
      • You will need to define the location of the top, bottom, left, and right boundaries of the region.

      • Example:
        • Look at TestImage012.tif. We want to redact the Social Security Number box.
        • Note that the top of the Social Security Number box is near the top of the text "4. SOCIAL SECURITY NUMBER".
        • Note that the bottom of the Social Security Number box is near the top of the text "7. BIRTH PLACE".
        • Note that the left of the Social Security Number box is near the left of the text "4. SOCIAL SECURITY NUMBER".
        • Note that the right of the Social Security Number box is to the left of the text "5a. AGE".
        • Add a Locate Image Region value finder that finds an Image Region and gives it the placeholder text "000-00-0000".
        • Add the clue "4. SOCIAL SECURITY NUMBER" to Clue List 1.
        • Add the clue "7. BIRTHPLACE" to Clue List 2.
        • Add the clue "5a. AGE" to Clue List 3.
        • We want the top of the region to be the top of the clue "4. SOCIAL SECURITY NUMBER". We can do this by defining the top of the region to be the Top of Clue List 1.
        • We want the bottom of the region to be the top of the clue "7. BIRTHPLACE". We can do this by defining the bottom of the region to be the Top of Clue List 2.
        • We want the left of the region to be the left side of the clue "4. SOCIAL SECURITY NUMBER". We can do this by defining the left side of the region to be the Left of Clue List 1.
        • We want the right of the region to be the left of the clue "5a. AGE". We can do this by defining the right side of the region as the Left of Clue List 3.
        • Run this Locate Image Region rule on TestImage012.tif and examine the results.
        • Change the above rule from an Image Region to Text.
        • Run the rule again and notice that the text within the Social Security Number box has been extracted.

    • You can also expand the boundaries of the region to be a specified distance away from the specified clues.
      • You can expand the top and bottom boundaries up or down by a specified number of lines.
        • A "line" is defined as the average height of the text in the document.
        • You can specify fractional numbers of lines.
      • You can expand the left and right boundaries left or right by a specified number of characters.

      • Example:
        • We can redact the Social Security Number box in TestImage012.tif using only the "4. SOCIAL SECURITY NUMBER" clue.
        • Add a Locate Image Region value finder that finds an Image Region and gives it the placeholder text "000-00-0000".
        • Add the clue "4. SOCIAL SECURITY NUMBER" to Clue List 1.
        • Define the top of the region to be the Top of Clue List 1.
        • Define the bottom of the region to be the Bottom of Clue List 1 expanded Down by 2.5 lines.
        • Define the left of the region to be the Left of Clue List 1.
        • Define the right of the region to be the Right of Clue List 1 expanded Right by 15 characters.
        • Run this Locate Image Region rule on TestImage012.tif and examine the results.

Exercises

  1. Write a Locate Image Region rule that will redact the Social Security Number box on TestImage009.tif.
  2. Write a Locate Image Region rule that will extract the text from the Date of Birth box on TestImage012.tif.
  3. Write rules that will redact both of the social security numbers on TestImage007.tif.
    • HINT: The Locate Image Region value finder will only find one region per page. However, you can find the second region on the page by having a second rule that has an Advanced Replace Strings preprocessor that removes the first "Social Security Number" clue before running the Locate Image Region value finder.
