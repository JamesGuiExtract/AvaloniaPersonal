Value Finder Purpose and Basics

Value Finder Uses

  • Value finders are the rule objects that are used to find the desired data.
  • They are the "work horses" of the rules.

Value Finder Basics

  • Each attribute rule must have a single value finder.
  • You can add a new attribute rule using the Insert... button.
    • Adding a new attribute rule will bring up a dialog where you choose which type of value finder you want to use. You can also give your attribute rule a description.
  • You can use the Delete button to remove the currently selected attribute rule.
  • You can modify an existing attribute rule using the Modify... button.
  • If you right click on an attribute rule, it will bring up a context menu that you can use to Cut, Copy, Paste, and Delete the rule.
  • You can turn off an individual attribute rule by un-checking the checkbox in front of it. This can be useful for testing.
  • If there are multiple attribute rules, they will be executed in order from top to bottom.
  • All attribute rules will be executed unless the Stop searching after first value found check box is checked.
  • You can change the order of the value modifiers by using the up and down arrow buttons.
