Value From List Value Finder

Uses

  • The Value From List value finder is used to find very specific words in the input text.
  • This value finder will only find literal strings. It can not find text using regular expressions.
  • This value finder will only return a single value.

Usage

  • Words can be added, removed, or, modified by using the Add, Remove, and Modify buttons.
  • The word can be made case sensitive.

Exercises

  1. Open a new Rule Set Editor. Add a Value From List value modifier that is configured to find the word John.
    • Run the rule on TestImage001.tif. Note that only the first instance of the word John is found.
    • Change the value modifier to be case sensitive. Run the rules again and note the difference in output.
