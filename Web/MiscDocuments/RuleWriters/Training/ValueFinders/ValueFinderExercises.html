Value Finder Exercises

  1. Write a regular expression that will find the folowing:
    • Any zip code with an optional four digit extension ( A number in the form ##### or #####-#### ).
    • Any telephone number with an area code in the form (###) ###-####.
    • Any date in the form MM/DD/YYYY.

  2. Open a new Rule Set Editor and add an attribute named File Number. Add a regular expression rule that will find the following from FileNumbers.txt:

    • Only file numbers with the format OH-######.
    • Only file numbers that start with OH.
    • Only file numbers that start with a letter.
    • Only file numbers that start with a number.

    Be sure to use the Rule Tester to test your rules!

  3. Open a new Rule Set Editor and add an attribute named Date. Add a regular expression rule that will find both of the dates in paragraph 1 of TestImage001.tif. Be sure to use the Rule Tester to test your rules!

  4. Open TestImage001.tif.

    • Write a Value After Clue rule that finds the body of text after the clue "STIPULATION TO ABATE CURRENT CHILD SUPPORT" and test the rule with the Rule Tester.
    • Write a Value After Clue rule that uses the Up to X lines after clue text option to find the 3 lines of text following the clue "1.".
    • Write a Value After Clue rule that uses the Up to X words option to find the 3 lines of text following the clue "3.".
