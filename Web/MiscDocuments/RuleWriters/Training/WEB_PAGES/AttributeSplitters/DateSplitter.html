Split Date and/or Time Attribute Splitter

Uses

  • The Split Date and/or Time attribute splitter is used to split dates and times into appropriate sub-attributes, such as Month, Day, Year, Hour, Minute, etc.
  • This splitter can also be used to reformat dates and times.
    • For example, you can change the date "May 23, 2008" into "05/23/08".

Usage

  • To use the Split Date and/or Time attribute splitter you must choose which sub-attributes you would like to output:
    • If you choose the Generate Default sub-attributes option, the splitter will output the default date and time sub-attributes.
      • For dates, the default sub-attributes are Day, Month, and Year.
        • If you choose the Number option in the Split Month as: box, the Month sub-attribute will automatically be translated into the appropriate number. (For example, "August" will be translated into "08".)
        • Similarly, if you choose the Name option in the Split Month as: box, the Month sub-attribute will automatically be translated into the appropriate month name. (For example, "08" will be translated into "August".)
      • For times, the default sub-attributes are Hour, Minute, and Second.
        • If you choose the Normal option in the Display Time as: box, the splitter will also output an AMPM sub-attribute that will contain "AM" or "PM" as appropriate.
          • Choosing the Normal option will also automatically convert military times into AM/PM times.
        • If you choose the Military option in the Display Time as: box, AM/PM times will automatically be converted into military time.
    • If you check the Generate Day of Week sub-attribute checkbox, the splitter will output a DayOfWeek sub-attribute that contains the day of the week that the input date occurred or will occur on.
    • If you check the Generate Formatted sub-attribute from specified format string option, the splitter will generate a Formatted sub-attribute that contains the date and/or time reformatted based on a specified format string.
      • You can use the following tags in the format string:
        • %d: day of the month
        • %B: month (full name)
        • %m: month (numerical)
        • %y: year (two-digit)
        • %Y: year (four-digit)
        • %I: hour (AM/PM time)
        • %H: hour (military time)
        • %M: minute
        • %S: second
        • %p: AM or PM
      • For example, the format string %m/%d/%Y would take change the date "July 3, 2008" into "07/03/2008".
      • You can test your format string by clicking the Test button.
