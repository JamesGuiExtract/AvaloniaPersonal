Select Page Region Preprocessor

Uses

  • Limits the pages passed to Value Finders for processing.
  • Can be used either as a page filter, or to re-OCR.

Usage

  • In the Select Document Preprocessor window:
  • Choose Select page region from the dropdown menu. Click Configure...
  • The dropdown menu at the top allows you to either include or exclude the region you will define.
  • 1) Select page(s).
    • All pages includes all pages, as expected.
    • Specific page(s).
      • Selects a single page. eg. "1"
      • Selects a list of pages. eg. "1,4,5"
      • Selects a range of pages. eg. "1-4"
      • Click the help balloon to the right of the field for more detailed info.
    • All/Leading/Trailing pages that contain: selects pages that contain a string.
      • The dropdown menu toggles selector between All/Leading/Trailing.
      • Enter a string into the text field.
      • Activate the checkboxes to interpret your string as a Regular Expression and/or as Case Sensitive.
  • 2) On each page in scope.
    • Checkboxes activate options to restrict pages horizontally and/or vertically.
    • Enter starting and ending percentages to define the ranges.
    • Note: You may want to enable Display X, Y percentages in Spot Recognition Window in the RDT Configuration utility.
  • 3) For each selected region.
    • Activate the checkbox to OCR the included text.
    • Enter an angle (in degrees) to rotate the image being OCR'd.
