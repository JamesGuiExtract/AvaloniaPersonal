Locate Image Region Preprocessor

Uses

  • Can be used to filter the pages or portions of pages that are passed to the rule based on clues.

Usage

  • In the Select Document Preprocessor dropdown menu in the RuleSet Editor:
    • Select Locate image region and click Configure....
  • There are three sections of options in the Configure Locate image region window: Top, Left, and Right.
    • The Top section contains general options. From left to right, they:
      • Toggle between finding Text or an Image Region. For an Image Region, you must assign text to it in the field provided.
      • Toggle the region definition as Inside or Outside the boundaries.
      • Include or Exclude Characters or Words that intersect the region boundaries.
      • Find the region on All pages or only the First page.
    • The Left section of the window contains options to configure the Clue(s).
      • Use the radio buttons to configure up to four lists of clues.
      • Click Add... to add clues to the list.
      • Click Remove to delete a clue.
      • Click Modify... or double click a clue to edit it.
      • Use the Load List... and Save List buttons to load clues from or save clues to a list. The list is a newline delimited text file.
      • Use the checkboxes below the clue list to toggle the clues as Case sensitive, regular expression(s), or to only find clues in the region defined by the previous clue list (Clue Lists 2-4 only).
    • The Right section of the window configures the Region Boundaries.
      • Use the dropdown menus and numerical fields to define the TOP, BOTTOM, LEFT, and RIGHT boundaries of the region.
      • Toggle between either the Top/Bottom or Left/Right end of the Page Containing Clues or one of the Clue Lists.
      • Expand the region Up/Down or Left/Right by a specified number of Spatial lines (1/10 line precision)/Characters (integer amounts only).
