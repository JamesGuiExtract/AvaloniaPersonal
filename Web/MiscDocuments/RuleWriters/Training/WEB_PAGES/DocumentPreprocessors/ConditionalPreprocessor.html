Conditional Document Preprocessor

Uses

  • The Conditional Document Preprocessor is used to conditionally run a preprocessor on a document.
  • Conditional Document Proprocessor allows rule writers to run a preprocessor on a document only if a specified condition is true, or only if a specified condition is false.
  • The specified condition can be a Document Type Condition, a Finding Rule Condition, or a RSD File Condition.
  • Any preprocessor can be called through Condtional Document Preprocessor.

Usage

  • To use the Conditional Preprocessor, click the Commands button and select it as the Document Preprocessor. Then click the Configure button.
  • To specify if the specified preprocessor will run if the condition is true or false, select true or false from the true/false drop menu.
  • To specify the type of condition select if from the drop down menu.
  • Document type condition allows rule writers to specify the type(s) of documents that must be matched for the specified rule to run.
    • Select whether the type of documents selected should be matched or not matched from the drop-down menu.
    • Click the Select button to select the type(s) of documents from the list of document types.
    • Next set the level of probablity of the classification to maybe, probable or sure. This sets the classification level the document must at least have for the condition to pass.
  • RSD file condition allows the output of a seperate RSD file as the condition that is met or not met to apply a specified document preprocessor. The Conditionally find value rule will call a specified RSD file configured by specifying the path name of the RSD file's etf file. The condition is met if the specified rule set finds at least one attribute.
    • A seperate RSD file must exsit for this preprocessor, and is usually created specifically for this purpose. Any appropriate rules should be created in the RSD file set to act as the condition.
    • Note: Always call the etf file of the RSD file you wish to call.
  • Select the preprocessor that will run conditionally. Click the configure button to configure the rule appropriately.
