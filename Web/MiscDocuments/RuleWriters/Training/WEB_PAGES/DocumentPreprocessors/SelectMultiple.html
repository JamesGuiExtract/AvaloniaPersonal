Select Multiple Document Preprocessors

Uses

  • The Select Multiple Document Preprocessors preprocessor is used to apply more than one preprocessor at a time.
  • For example, it can be used to run two Advanced Replace String preprocessors before running the value finder.

Usage

  • You can use the Add and Delete buttons add and delete preprocessors.
  • You can use the Commands button to modify an existing preprocessor.
    • Choosing the Select Document Preprocessor... option will let you change the type of the preprocessor.
    • Choosing the Configure Document Preprocessor... option will let you change the settings of the preprocessor.
    • Choosing the Configure description... option will let you change the description of the preprocessor.
  • The preprocessors will execute in order from top to bottom. You can change the order of the preprocessors by using the up and down arrow buttons.
  • If you right click on the preprocessor, it will bring up a context menu that you can use to Cut, Copy, Paste, or Delete the preprocessor.
  • The checkboxs in the Enabled column allow you to selectively turn preprocessors on or off. This is especially useful for testing and debugging rules.
