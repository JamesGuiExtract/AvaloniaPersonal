Remove Spatial Information Output Handler

Uses

  • The Remove Spatial Information output handler is used to remove the spatial information from the found data.
  • The spatial information is sometimes removed so that the coordinates of the found data are not included in output .xml files.

Usage

  • The Remove Spatial Information output handler requires no configuration.

Exercises

  1. Add a Select Multiple Document Output Handlers output handler to Output.rsd.
    • Add an Output Data to XML File output handler to the Select Multiple Document Output Handlers output handler. Make sure the Output Data to XML File output handler is configured to output a Version 2 XML file to <SourceDocName>.xml.
    • Run the rules on TestImage001.tif to create a .xml file.
    • Add a Remove Spatial Information output handler before the Output Data to XML File output handler.
    • Change the Output Data to XML File output handler to output a .xml file to <SourceDocName>2.xml
    • Run the rules on TestImage001.tif to create a second .xml file.
    • Compare the two .xml files. What is the difference between the files?
