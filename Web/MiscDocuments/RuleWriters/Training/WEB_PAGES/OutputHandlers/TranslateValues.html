Translate Values or Types Output Handler

Uses

  • The Translate Values or Types output handler is used to change an attribute’s value or type.
    • For example, you can change an attribute's type from SocialSecurityNumber to SSN.
    • You could change the value of an attribute from 0 to None.

Usage

  • First you must choose weather you want to translate values or types.
  • Next, you must specify what text you want to replace and what you want to replace it with.
    • You can make multiple translations in a single output handler.
    • The Add, Remove, and Modify buttons can be used to add, delete, and modify translation pairs.
    • The translation pairs can be made case sensitive.

Exercises

  1. Open Translate.rsd and run it on Translate.txt.
    • Add a Translate Values or Types output handler to the existing Select Multiple Output Handlers output handler and configure it to change the types DLN and SSN to SensitiveNumber.
    • Run the rules again and notice the difference in output.
    • Add another Translate Values or Types output handler that changes the value $0 to No Money Owed.
    • Run the rules again and notice the difference in output.
