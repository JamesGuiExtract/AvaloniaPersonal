Select Multiple Output Handlers

Usage

  • This output handler is used to group other output handlers.
  • It is recommended that you use Select Multiple Output Handlers as your primary output handler. This allows for easily adding more output handlers.
  • Multiple Select Multiple Output Handlers can be used to group output handlers by function to make rulesets more readable. For example, you might have a group of output handlers that removes false positives and another for renaming and moving attributes.
  • Note that this also applies to Document Preprocessors
