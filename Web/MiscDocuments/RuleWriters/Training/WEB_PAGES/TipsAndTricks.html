Tips and Tricks

Useful programs

Image viewers

  • Irfanview is an image viewer that can be used to browse all images in a directory.
  • Foxit Reader is a lightweight alternative to Adobe Acrobat Reader

File management

  • Total Commander is an orthodox file manager. Its features include:
    • file filtering (show only *.tif, *.uss, etc.)
    • quick view of files (use Imagine for quickviewing images)
    • multi-rename tool for renaming many files at once
    • built in command line
    • faster loading times than Windows Explorer of directories with a large number of files, e.g. K:\Common\Engineering\Sample Files.
  • Launchy is a program launcher / file opener that gives you quick access to items on your Start menu and Desktop.
  • Console is an enhanced command line prompt with a tabbed interface.
  • Unlocker can show what program is locking a file and often remove the lock, useful if you have trouble deleting/renaming/moving a file.
  • Cygwin is a collection of Unix-like tools for Windows.
  • BareGrep can search filenames and file contents using regular expressions.

Other programs

  • Path Copy is a shell extension for Windows that allows you to copy the path of a file or directory to the clipboard.
  • Ditto is a clipboard manager that keeps a history of items in the clipboard. It can be configured to include items from Extract applications (RuleSet Editor value modifiers, etc.) in its history, which appear as UCLID Objects.
  • AutoHotkey is a utility for creating hotkeys and mouse/keyboard macros. Many hotkeys for Extract applications are included in i:\users\Douglas_Beachy\Public\AutoHotkey\SmokinHotKeys.ahk. Type Ctrl + Alt + Shift + K while running SmokinHotKeys to see an application's hotkeys.
  • The Regex Coach and RegEx Testbed are for testing regular expressions.
