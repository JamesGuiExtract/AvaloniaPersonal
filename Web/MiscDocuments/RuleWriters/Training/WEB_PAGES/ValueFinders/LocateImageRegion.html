Locate Image Region Value Finder

Uses

  • The Locate Image Region value finder is used to redact or extract text from a specified region of a document.
  • It uses the position of specified clues in the document to find the desired region.

Usage

  • To use the Locate Image Region value finder, you must first specify whether you want to find an image region or text.
    • Image regions contain only the spatial information of the found region.
      • They are primarily used to redact the region in the ID Shield product.
      • When you create an image region, you will have to assign some "placeholder" text for the found spatial information in the Assign the following text to the image region box.
    • If you choose to find text, the Locate Image Region value finder will extract and return all of the text contained in the found region.
      • This option is primarily used to extract data from a certain region of the document for the FLEX Index product.
      • You can choose to whether to include or exclude characters or words that intersect the boundaries of the region.
  • You can choose whether to return only the first region that is found or to return all of the regions that are found.
  • Next, you will need to specify the clues that will be used to find the region as well as where the region is located in relation to the specified clues.
    • Clues can be added, deleted, and modified using the Add..., Remove, and Modify... buttons.
      • The clues can be plain text or regular expressions.
      • The clues can be made case sensitive.
      • You can have up to four separate clue lists that you can use to find the image region.
    • You can specify were the region is relative to the clues by using Region Boundaries box.
      • You will need to define the location of the top, bottom, left, and right boundaries of the region.
    • You can also expand the boundaries of the region to be a specified distance away from the specified clues.
    • You can expand the top and bottom boundaries up or down by a specified number of lines.
      • A "line" is defined as the average height of the text in the document.
      • You can specify fractional numbers of lines.
    • You can expand the left and right boundaries left or right by a specified number of characters.
