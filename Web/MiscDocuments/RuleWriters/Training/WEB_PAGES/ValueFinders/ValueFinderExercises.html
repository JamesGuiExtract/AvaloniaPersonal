Value Finder Exercises

    NOTE: Remember to use the Rule Tester to test your rules.

  1. Write a regular expression that will find the following:
    • Any zip code with an optional four digit extension ( A number in the form ##### or #####-#### ).
    • Any telephone number with an area code in the form (###) ###-####.
    • Any date in the form MM/DD/YYYY.

  2. Open a new Rule Set Editor and add an attribute named FileNumber. Add a regular expression rule that will find the following from FileNumbers.txt:

    • Only file numbers with the format OH-######.
    • Only file numbers that start with OH.
    • Only file numbers that start with a letter.
    • Only file numbers that start with a number.
