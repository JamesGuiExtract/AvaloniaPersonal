Find From RSD File

Uses

  • The Find From RSD value finder is used to find attributes by calling another RSD file and returning all of its output as attributes.
  • The RSD file called can have as many rules, preprocessors and output handlers as needed.

Usage

  • Select Find from RSD file rule and click the configure button to enter the Attribute name of the attributes the rule will be finding and specify the path name to etf file of the RSD file to be called.
    • Note: Always call the etf file of the RSD file you wish to call.

Exercises

  1. Open FindFromRSD.rsd and run it on TestImage001.tiff. Observe that it finds a social security number. Notice that it is calling SSN.rsd.
