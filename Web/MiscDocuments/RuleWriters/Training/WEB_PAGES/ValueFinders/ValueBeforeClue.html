Value Before Clue Value Finder

Uses

  • The Value Before Clue value finder is used to find a specified amount of text before a specified clue.
  • Multiple clues can be defined in a single rule.

Usage

  • Clues can be added, deleted, or modified using the Add, Remove, and Modify buttons.
  • The specified clue can be either plain text or a regular expression.
  • You have to specify how much of the text preceding the clue that you want to capture:
    • You can capture all of the text between the beginning of input and the clue.
    • You can capture all of the text between an end of line character and the clue.
    • You can capture all of the text between a second clue and your first clue.
      • The second clue can also be either plain text or a regular expression.
    • You can capture a certain number of lines before the clue.
      • You will need to choose the number of lines you want to capture and whether or not you want to include the line that contains the clue.
    • You can capture a specified number of words before the clue.
      • You will need to choose the number of words you want to capture.
      • You can optionally stop searching if you find a specified character.
      • You must specify what a word is by defining what characters separate words.
  • All of the strings in the Value Before Clue value finder can be made case sensitive.
