Insert Characters Value Modifier

Uses

  • The Insert Characters value modifier is used to insert one or more characters into the found text at a specified position.
  • For example, you could use this modifier to insert a dash after the first five digits of a zip code.

Usage

  • To use the Insert Characters value modifier, you must first specify which characters you would like to insert into the found text.
  • You must then select where you want to insert the specified characters.
    • You can choose to append the specified characters to the end of the found text.
    • Alternately, you can choose to insert the specified characters at a specified position in the found text.
      • For example, inserting a dash at position "6" will change "123456789" into "12345-6789".
  • You also have the option of only executing this modifier on strings of a specified length.
    • To use this option, specify a string length and then choose to whether you want to run this modifier on strings that are equal to, greater than or equal to, greater than, less then or equal to, less then, or not equal to the specified string length.
    • By default, this modifier will run on strings of any length.

Exercises

  1. Open InsertChars.rsd and run it on InsertCharactersEasy.txt. Add several Insert Character value modifiers that will change the format of the found telephone numbers to (###) ###-####.
  2. Now run InsertChars.rsd on InsertCharactersDiffcult.txt. Change your Insert Character value modifiers so that phone numbers without an area code are output in the ###-#### format, while numbers with an area code are output in the (###) ###-#### format.
    • HINT: You can use the string length to determine if the phone number has an area code.
