Change Case Value Modifier

Usage

  • The Change Case value modifer is used to change the case of the found text.
  • You can change the case of the found text to all upper case, all lower case, or title case.

  • Example:
    • Load FileNumber_VAC.rsd to find File Number attributes from within FileNumbers.txt.
    • Add Change case value modifier - use upper case. Show effect.
    • Add Change case value modifier - use lower case. Show effect.
