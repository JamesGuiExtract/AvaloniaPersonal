VOA File Viewer

  • The VOA File Viewer allows you to view and edit .voa files.
  • To create a new attribute:
    • Click the New Attribute button (or press <ENTER>).
    • Click on the icon that looks like a pink piece of paper to open a Spot Recognition Window.
    • From the Spot Recognition Window, open an image file.
    • Use the Any-Angle Zone or Horizontal Zone tool to create a highlight on the image.
    • If text was recognized in the area under the highlight, it will become the attribute's Value, otherwise you will be prompted to enter text.
    • It is a good idea to enter a type for the attribute to simplify testing.
    • Note: When closing the Spot Recognition Window, you will be prompted to save changes you made to this document? This is referring to the highlights, which can be saved but aren't useful for our purposes, and you should click No.
  • To edit an existing attribute:
    • Double-click on an attribute to open a Spot Recognition Window and display it.
    • With a highlighter tool, create a new highlight. This will be the attribute's new value.
    • Notice that, when you create a new value for an attribute, its type will clear and should be re-entered.
    • Tip:
      If you are adding an attribute to an existing .voa file, it is easier to double click on an existing attribute to open the Spot Recognition Window and then add the new attribute in the VOA File Viewer—that way you don't have to open the image.
    • Watch out!
      When using the above tip, make sure that you create the new attribute before creating a highlight or you will change the value of whatever attribute you have selected.

Exercises

  2. Save the file and close the viewer.

  3. Open the file you saved and verify that the attributes you created were loaded and will display on the image.
