UEX Viewer

Usage

  • Use this utility to view a log of exceptions (errors) generated by an Extract Systems component or application.
    • Note: Selecting "View Exception Log" from the Windows Start Menu opens the default log file (UCLIDException.uex) in the UEX Viewer.

  • Viewing an exception (Ctrl-D) or double-clicking on an item displays an Error window. The Details button provides expanded information about the exception.
  • Click the top of any column to sort the list by that column.

  • Menu Commands
    • File Commands
      • Open (Ctrl+O) allows the user to open another UEX file.
      • Open Previous Log File (Ctrl+Left) opens the previous UEX file in the current folder. (Or use the < key at the bottom of the window.)
      • Open Next Log File (Ctrl+Right) opens the next UEX file in the current folder. (Or use the > key at the bottom of the window.)
      • Refresh current log file (F5) reloads the current file from disk.
    • Edit Commands
      • Find performs a case-insensitive search for a specified text string within the listed exceptions.
      • Delete Selection removes the selected items from the display.
      • Invert Selection unselects all currently selected items and selects all currently unselected items.
      • Clear empties the display.
