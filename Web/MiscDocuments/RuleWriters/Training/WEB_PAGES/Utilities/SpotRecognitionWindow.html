Spot Recognition Window

  • The Spot Recognition Window is used to view images and associated attributes, and to create these attributes.
  • The Toolbar
    • The first three buttons, left to right, on the toolbar are for opening and printing images and saving highlights.
    • The next group contains buttons for zooming in and out and moving the viewport around. If you hold your mouse over one of these buttons for a moment, a tooltip will appear and display a brief description of the tool and its keyboard shortcut.
    • The next group is for making highlights/attributes.
      • The first button in this group is the highlighting tool. This tool has two modes, one to make perfectly horizontal, rectangular zones and one to make skewed zones, and will look like either a yellow rectangle or a yellow marker, depending on which variety is selected. If the horizontal zone tool is selected, you will be able to drag a rectangle on the current image. If the any-angle zone tool is selected, you will be able to draw a line that has a rectangle extending out from it. The height of this rectangle can be changed while drawing by using the + and - keys or preset with the next tool.
      • The default height tool looks like an 'A' with a ruler next to it. To use this tool, click and drag to the desired height and the any angle zone tool will be selected and ready to create a highlight of that height.
      • The next tool allows you to alter the text associated with a highlight but is not useful for creating attributes.
    • The last tool in this group allows you to erase highlights by clicking on them.
    • The next tool looks like a printed page and allows you to select an area and recognize the text within it. The recognized text is either displayed or sent to the application that opened the Spot Recognition Window.
    • The last few buttons allow you to rotate the image and to change pages.

Exercises

  1. Open the Spot Recognition Window and load RedactionDemo\Images\TestImage004.tif.

  2. Use the Zoom Window tool to select an area around Janet Jacksons SSN.

  3. Use the Zoom Out button to bring both Jacksons' SSNs into focus.

  4. Use the Pan tool to move around the image

  5. Click the Zoom Previous button to go back into the zoom history until the whole page is in focus or click the Toggle fit to page mode button.

  6. Click the Go to next page button until the fourth page is displayed.

  7. Click on the Recognize text and process menu button and select OCR Image (Process Rectangular Area). Click and drag to create a box around the skewed stamp near the bottom of the page and look at the result.

  8. Click the Recognize text and process menu button again but this time select OCR Image (Process Current Page). Can you find the stamp in all that text? Does it look the same as it did in the previous exercise?
