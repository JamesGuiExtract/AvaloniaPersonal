Basic Rule Set Editor and Rule Tester Exercises

Exercises

  1. Open the Rule Set Editor and add attributes with the below names. Which of the below attribute names are not acceptable?

    • 1_Attribute
    • One_Attribute
    • One Attribute
    • Attribute1
    • _Attribute
    • abc123
    • Add two attributes with the same name

  2. Right clicking on an attribute's name will bring up a context menu. Try using Cut, Copy, and Paste to move attributes back and forth between .rsd files.

  3. Right-click on an attribute's name and select Duplicate several times. How are the new attribute names distinguished from the original?

  4. Select Text from manual input on the Input tab. Add some text to the edit box. Navigate between the different input choices. Is the input text retained across navigations?
