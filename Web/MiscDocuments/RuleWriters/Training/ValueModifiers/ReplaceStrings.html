Replace Strings Value Modifier

Uses

  • The Replace Strings value modifier is used to replace or remove text from the data found by the value finder.
  • For example you can change "Doctor" to "Dr." or remove area codes from telephone numbers.

Usage

  • Replacement pairs can be added, deleted, and modified using the Add, Remove, and Modify buttons.
  • For each replacement pair, you will need to specify what text you want to replace and what you want to replace it with.
  • The replacement pairs can be either plain text or regular expressions.
  • The replacement pairs can be made case sensitive.
  • The replacement pairs will be run in order from top to bottom. You can change the order of the replacement pairs using the up and down arrow buttons.

  • Example:
    • Open Name.rsd and run it on Names.txt.
    • Add a Replace Strings value modifier that makes the following replacements:
      • Ms. -> Miss
      • Mrs. -> Misses
      • Mr. -> Mister
      • Dr. -> Doctor
    • Run the rules again and notice the difference in output.
