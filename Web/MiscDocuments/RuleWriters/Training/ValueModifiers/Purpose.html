Value Modifier Purpose and Basics

Value Modifier Uses

  • Value Modifiers can be used to format data.
    • You can use a Change Case value modifier to change the format of a file number from oh-0012345 to OH-0012345.
    • Advanced Replace String value modifiers can be used to change the format of a phone number from 608-216-7950 to (608) 216-7950.

  • Value modifiers can be used to "pull out" more specific text from the text found by the Value Finder.
    • A Regular Expression rule can be used to grab the last seven digits from a 10-digit phone number.
    • A Replace Strings value modifier can be used to "trim off" extra new line characters.

  • Value modifiers can be used to discard inappropriate attributes.
    • A Regular Expression value modifier can be used to remove empty strings.
    • Regular Expression value modifiers can do rule-specific validation.

Value Modifier Basics

  • An attribute rule can have any number of value modifiers.
  • To add a value modifier to an attribute rule, you must check the Apply the following modifying rules checkbox.
  • You can add a new value modifier using the Add button.
    • Adding a new value modifier will bring up a dialog where you choose which type of value modifier you want and give it a description.
  • You can use the Remove button to remove the currently selected value modifier.
  • You can modify an existing value modifier using the Commands button.
    • Choosing the Select Attribute Modifier... option will let you change the type of the selected value modifier.
    • Choosing the Configure Attribute Modifier... option will let you change the settings of the selected value modifier.
    • Choosing the Configure description... option will let you change the description of the selected value modifier.
  • If you right click on an value modifier, it will bring up a context menu that you can use to Cut, Copy, Paste, and Delete the value modifier.
  • You can turn off an individual value modifier by un-checking the checkbox in front of it. This can be useful for testing.
  • You can turn off all of the value modifiers in an attribute rule by un-checking the Apply the following modifying rules checkbox.

  • Sequence of Value Modifiers
    • If there are multiple value modifiers, they will be executed in order from top to bottom.
    • Because of this, the order of the value modifiers matters.
      • For example, say you have modifierA that changes the text "Breakfast" into "Lunch" and modifierB that changes the text "Lunch" into "Dinner".
      • If you start with the text "Breakfast" and run modifierA and then modifierB you will end up with the text "Dinner".
      • If you start with the text "Breakfast" and run modifierB and then modifierA you will end up with the text "Lunch".
    • You can change the order of the value modifiers by using the up and down arrow buttons.
