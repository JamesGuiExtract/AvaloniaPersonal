Regular Expression Rule Value Modifier

Uses

  • The Regular Expression Rule value modifier applies a regular expression to the text returned by the Value Finder.
  • It can be used to par down, format, or validate the found value.

Usage

  • The Regular Expression Rule value modifier functions exactly like the Regular Expression Rule value finder.
    • Simply type your regular expression into the edit box.
    • You can make your regular expression case sensitive.
    • You can also read your regular expression in from a text file.

  • Example:
    • Open PhoneNumber_Regex.rsd. This .rsd file has a rule that looks for the text after "Phone Number:".
    • Open the Rule Tester and run this rule on PhoneNumbers.txt and look at the output.
    • Create a new a Regular Expression Rule value modifier and add the regular expression \d{3}-\d{4}.
    • Run the rules again and note the difference in the output.
