Conditional Value Modifier

Uses

  • Applies a value modifier based on a specific condition.

Usage

  • In the Add Attribute Finding Rule window:
  • Click the Apply the following modifying rules checkbox.
  • Click Add and select Conditionally modify attribute from the dropdown menu. Click Configure...
  • Toggle whether the condition must be true or false in the first dropdown menu.
  • Select the condition in the second dropdown menu.
  • Click Configure... to configure the condition.
  • Select the rule in the third dropdown menu.
  • Click Configure... to configure the rule.
  • Important: The condition tests the input passed to the value finder, not the value finder's output.

  2. Open the Rule Tester and load ".\Files\FileNumbers.txt"
