Numeric Sequence Expander/Contractor Value Modifier

Uses

  • The Numeric Sequence Expander/Contractor value modifier is used to expand, contract, and sort comma-separated lists of numbers.
  • For example, it can be used to change the text 2, 4, 3, 1 into the text 1-4.
  • Similarly, it can be used to change the text 5-6, 1-4 into the text 1,2,3,4,5,6.

Usage

  • To use the Numeric Sequence Expander/Contractor, you must first specify whether you want to expand or contract the numeric sequence that is input. This is done by clicking the appropriate radio button.
  • When expanding a numeric sequence, checking the Sort sequence checkbox will sort the numbers in the sequence that is input.
    • You can use the drop down list next to the check box to choose whether the numbers will be sorted in ascending or descending order.
    • For example, if the input text is 1,2,3,4 and Descending is selected, the output text will be 4,3,2,1.
    • Similarly, in the input text is 5-6, 1-3 and Ascending is selected, the output text will be 1,2,3,5,6.
  • When contracting sequences, the Sort sequence checkbox is automatically selected.
    • You can still use the drop down list next to the checkbox to determine whether the contracted sequences will be displayed in ascending or descending order.
    • For example, if the input text is 1,2,3,4 and Descending is selected, the output text will be 4-1.
    • Similarly, in the input text is 1,2,3,4,10,11,12 and Ascending is selected, the output text will be 1-4,10-12.
  • Checking the Eliminate duplicates checkbox will remove duplicates from numeric sequence that is input.
    • For example, if the Eliminate duplicates checkbox is checked when contracting the text 1,2,3,2,4,4,4 the output text will be 1-4.
    • Similarly, if the Eliminate duplicates checkbox is checked when expanding the text 1-3,3,3-5 the output text will be 1,2,3,4,5.
