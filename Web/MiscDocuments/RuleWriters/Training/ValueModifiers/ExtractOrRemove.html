Extract Or Remove Value Modifiers

Uses

  • Extract or Remove Characters Value Modifiers can be helpful when you have a broad search, but need to pull out one piece of it.
  • The most common application is to remove the last 4 digits from a Social Security Number.
  • Another common use is to remove unneccessary filing numbers from around needed data.
    • Example: A search picked up A-###-##-####-12. You could use Extract or Remove Middle Characters to get just the Social Security Number.
  • Extract or Remove Characters Value Modifiers are most useful in situations where Regular Expression based modification produces inconsistent results, such as when things OCR poorly.


  • Extract Or Remove Leftmost Characters
    • This extracts or removes characters at the start of a string.
    • This could be used to remove Mr. or Dr. from the start of names, or to grab the first 5 digits of a zip-code.
  • Extract or Remove Middle Characters
    • This allows you to set a starting point and ending point for what you need extracted/removed.
    • This allows you to extract/remove characters from either side without running both a Leftmost and Rightmost modifier.
  • Extract or Remove Rightmost Characters
    • This extracts or removes characters at the end of a string.
    • This is useful for removing things such as the last 4 digits of a Social Security Number, or the year from a Date of Birth.

Usage

  • Extract or Remove Rightmost Characters and Extract or Remove Leftmost Characters work the same, with the exception of operating on different ends of the string.
    • First, select if you will be removing or extracting.
    • If you choose extract, then only the targeted characters will appear from the output.
    • If you choose remove, then the targeted characters will not appear in the output.
    • Next, decide how many characters you need to operate on. For Rightmost, this will start at the end and work to toward the beginning. For Leftmost it will start at the beginning and work toward the end.
    • The Accept if total number of characters found less than specified above checkbox will allow you to remove your desired number of characters regardless of the string length.
      • Example: If the box is checked, then Removing the last 10 letters from Thomas will remove the entire string. If unchecked, then the string will be unaffected.
  • Extract or Remove Middle Characters has the option of Extracting or Removing, as well as theAccept if total number of characters found less than specified above. The main difference is the method of collection.
    • Input the numerical location of the first character you would like to extract or remove, as well as the last. This is the range that will be effected.

Exercises

  1. Open a new Ruleset Editor, and write a Regular Expression rule to find dates.
    • Run your rule on Date.txt to confirm that it works.
    • Add an Extract or Remove Rightmost Characters Value Modifier so that only the month and day are displayed.
    • Uncheck the previous Value Modifier, and add an Extract or Remove Middle Characters Value Modifier to only show the Month's abbreviation (Jan, Feb, etc) and the year.
