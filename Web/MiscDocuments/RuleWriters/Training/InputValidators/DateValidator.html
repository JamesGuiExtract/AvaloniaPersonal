Date Input Validator

Uses

  • The Date input validator is used in conjunction with the Remove Invalid Entries output handler to remove invalid dates from the output of the rules.

Usage

  • The date validator has no configuration options.

Exercises

  1. Open Dates.rsd and run it on Dates.txt. Add a Date input validator and a Remove Invalid Entries output handler to the rules. Run the rules again and note the difference in output.
