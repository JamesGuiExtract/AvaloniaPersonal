Attribute Splitter Purpose and Basics

Purpose

  • Attribute Splitters are used to divide attributes into "sub-attributes".
  • For example, we could split a Name attribute to create FirstName, MiddleName, and LastName sub-attributes.

Basics

  • Attribute splitters can be added by clicking on the Commands button and selecting the Select Attribute Splitter... option.
    • This will bring up a dialog where you can choose which type of attribute splitter you want and give it a description.
  • You can modify an existing attribute splitter using the Commands button.
    • Choosing the Select Attribute Splitter... option will let you change the type of the attribute splitter.
    • Choosing the Configure Attribute Splitter... option will let you change the settings of the attribute splitter.
    • Choosing the Configure description... option will let you change the description of the attribute splitter.
  • You can remove the attribute splitter by changing its type to <None>.
  • If you right click on the attribute splitter, it will bring up a context menu that you can use to Cut, Copy, Paste, and Delete the attribute splitter.
  • You can turn off the attribute splitter by un-checking the checkbox in front of it. This can be useful for testing.
