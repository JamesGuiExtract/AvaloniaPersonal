USS File Viewer

The USS File Viewer

  • The USS File Viewer application is used for viewing .uss files.
  • If you highlight some text in the USS File Viewer, the page the text is on and the positions of the starting and ending characters will be displayed on the bottom of the window.
  • You can search for text in a USS file by bringing up the Find dialog.
    • You can search for text or a regular expression.
    • You can make the search case sensitive.
    • If searching for more than one item, you can use the AND and OR operators.
    • You can search only part of a document.
    • You can start your search at the beginning of the document or at your current position.
