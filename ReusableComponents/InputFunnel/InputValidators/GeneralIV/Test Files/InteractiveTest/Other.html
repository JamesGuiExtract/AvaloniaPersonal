Ruleset Editor - Input Validators

  1. Date

    • Configure button is inactive.
    • Non-valid dates get flagged as invalid.

  2. Double

    • Configure button displays Double settings dialog.
    • Minimum - Has Limit check box is checkable.
      • Selection enables corresponding edit and check box.
      • Edit box is editable.
      • Include Minimum in range check box is checkable.
    • Maximum - Has Limit check box is checkable.
      • Selection enables corresponding edit and check box.
      • Edit box is editable.
      • Include Maximum in range check box is checkable.
    • Include zero in range check box is checkable.
    • Include negative numbers in range check box is checkable.
    • OK button saves changes and exits dialog.
      • If an invalid limit is specified, an error message is displayed indicating that it is invalid.
    • Cancel button ignores changes and exits dialog.
    • X button ignores changes and exits dialog.
    • Non-valid doubles get flagged as invalid.

  3. Float

    • Configure button is inactive.
    • Non-valid floats get flagged as invalid.

  4. Long Integer

    • Configure button displays Long Integer settings dialog.
    • Minimum - Has Limit check box is checkable.
      • Selection enables corresponding edit and check box.
      • Edit box is editable.
      • Include Minimum in range check box is checkable.
    • Maximum - Has Limit check box is checkable.
      • Selection enables corresponding edit and check box.
      • Edit box is editable.
      • Include Maximum in range check box is checkable.
    • Include zero in range check box is checkable.
    • Include negative numbers in range check box is checkable.
    • OK button saves changes and exits dialog.
      • If an invalid limit is specified, an error message is displayed indicating that it is invalid.
    • Cancel button ignores changes and exits dialog.
    • X button ignores changes and exits dialog.
    • Non-valid long integers get flagged as invalid.

  5. Regular Expression

    • Configure button displays Regular Expression settings dialog.
    • Please specify the regular expression that should be used to validate input edit box is editable.
    • Case Sensitive check box is checkable.
    • OK button saves changes and exits dialog.
      • Error message is displayed if regular expression is not specified.
    • Cancel button ignores changes and exits dialog.
    • X button ignores changes and exits dialog.
    • Values that do not match the expression specified get flagged as invalid.

  6. Short Integer

    • Configure button is inactive.
    • Non-valid short integers get flagged as invalid.

  7. Value From List

    • See Value Finders - Value From List
    • Values that does not match the expression specified get flagged as invalid.
