Saving over a read-only .gdd file.

  1. Open the file IMAGENAME???.gdd.


    NOTE: If you need more information on opening .gdd files go to either of the following test cases:
    Opening_gdd_files.html

  2. Right click on the file IMAGENAME???.gdd.

    A floating menu should appear.

  3. Select 'Properties'.

    The 'Properties' dialog box should appear.

  4. Go to the 'General' tab and click the checkbox to the left of 'Read-only'. Click the 'OK' button.

    The file should be read-only now.

  5. Select the file IMAGENAME???.gdd and click the 'Open' button.

    The 'Open' dialog box should disappear. The image GDD FILE??? should open. The title bar should say GDD FILE??? - Image Recognition. All of the toolbar buttons except the highlighter and the page navigation buttons should be enabled. Any saved highlights should also display.

  6. Click the 'Save' button.

    The 'Save' dialog box should appear with the default file name being IMAGENAME???.gdd.

  7. Click the 'Save' button.

    An error should appear indicating that the file is marked as read-only and cannot be saved. You should also be allowed to save the file to a different name.

  8. Change the file name to IMAGENAME???-B.gdd then click the 'Save' button.

    The file should get saved and the 'Save' dialog box should disappear.
