1. Bring up a new instance of the SpotRecognition window using the IRTester application.

2. Note the ID of the window that is displayed.

3. Type in the ID in the IRID field, and click on Disconnect.

The Spot Recognition window should disappear.

4. Repeat steps 1 above, and click on the "X" to close the Spot recognition window.

The Spot Recognition window should disappear.
