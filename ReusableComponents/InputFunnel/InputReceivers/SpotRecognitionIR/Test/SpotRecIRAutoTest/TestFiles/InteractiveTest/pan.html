Using the pan command.

  1. Open the file IMAGENAME???.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Zoom into a small area of the image.


    NOTE: If you need more information on Zooming go to the following test case:
    zoom.html

  3. Click the 'Pan' button.

    The 'Pan' button should appear depressed (or unhappy) and the cursor should have turned into a little hand.

  4. Move the cursor to a spot on the image. Click and hold the left mouse button down. Drag the Mouse up, down, left, right, around and around

    While moving the cursor, the spot of the image that you clicked on should follow the cursor and the rest of the image should follow.
