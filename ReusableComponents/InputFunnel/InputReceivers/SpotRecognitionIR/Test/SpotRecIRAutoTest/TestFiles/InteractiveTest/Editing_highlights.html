Editing highlights

  1. Open the file Test009.tif.gdd.


    NOTE: If you need more information on opening .gdd files go to either of the following test cases:
    Opening_gdd_files.html

  2. Click the 'Edit Highlights' button.

    The 'Edit Highlights' button should appear depressed.

  3. Click on the highlight for the bearing "N46°49'21"E".

    The 'Edit Highlights' dialog will appear with the text associated with the highlight.

  4. Change the text to "ICHANGEDIT" and click the 'OK' button.

    The text associated with the highlight should have changed.

  5. Click on the same highlight again to see if the associated text is the way you changed it.

    The 'Edit Highlights' dialog will appear with the text associated with the highlight.

  6. Click the 'Close' (X) button in the Image Recognition window.

    A warning message should appear telling the user that changes have been made. Do you want to save the changes?

  7. Click the 'No' button.

    The Image Recognition window should close.

  8. Re-open the image recognition test driver.

    A new 'Image Recognition' dialog should appear.

  9. Re-open the file Test009.tif.gdd.


    NOTE: If you need more information on opening .gdd files go to either of the following test cases:
    Opening_gdd_files.html

  10. Click the 'Edit Highlights' button.

    The 'Edit Highlights' button should appear depressed.

  11. Click on the highlight that was changed in step ????.

    The change should not have been saved. The value should still be "N46°49'21"E" and not ICHANGEDIT.

  12. Use the 'Edit Highlights' tool to change the text associated with some of the highlights to ICHANGEDIT.

    The text associated to the highlights should be ICHANGEDIT.

  13. Click the 'Close' (X) button in the Image Recognition window.

    A warning message should appear telling the user that changes have been made. Do you want to save the changes?

  14. Click the 'Yes' button.

    The 'Save' dialog should appear.

  15. Save the file as Test009b.tif.gdd.

    The file should have been saved. And the Image Recognition window should have closed.

  16. Re-open the image recognition test driver.

    A new 'Image Recognition' dialog should appear.

  17. Re-open the file Test009.tif.gdd. Check the strings that you edited to make sure that the changes were saved.


    NOTE: If you need more information on opening .gdd files go to either of the following test cases:
    Opening_gdd_files.html

  18. Click the 'Close' (X) button in the Image Recognition window.

    The Image Recognition window should close.
