Opening unsupported file types.

  1. Open the file IMAGENAME???.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Click the 'Open' button.

    The 'Open' dialog box should appear.

  3. Select the file WRONG FILE TYPE??? and click the 'Open' button.

    The error following error should appear:
    "Unable to load requested image. The referenced image does not exist, is not readable, or is not in a format that IcoMap understands."

  4. Click the 'OK' button.

    The error message should disappear and the 'Image Recognition' window should be empty and only the 'Open' toobar button should be enabled.
    NOTE: I beleive that the original image should be displayed here in the previous state. Currently the window is blank, displays the old image name and has most of the toolbar buttons enabled. When you click on one of the 'zoom' buttons an exception error appears and the buttons become disabled.
