Recognizing text using spot recognition

  1. Open the file Test008.tif.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Zoom into the bearing "S11°26'38" E". It's just below the center of the image.


    NOTE: If you need more information on zooming go to the following test case:
    zoom.html

  3. Set the highlighter size using the 'Set Highlight Height' tool. Using the bearing "S11°26'38" E" for a reference, the first point you pick with the left mouse button should be just above the "S". While holding the mouse button down, drag the mouse to just below the "S" and then let go of it.

    As you use spot recognition to draw, you should see that the height of the highlight is the same as the distance you picked.

  4. Click the "Highlighter" tool.

    The button should appear depressed indicating that it is the current tool and the cursor should look like a highlighter.

  5. Referencing to the bearing "S11°26'38" E" again, move the cursor near the middle and to the left of the "S" and click and hold down your left mouse button and drag the mouse around.

    You should see a Cyan box stretch from the point you picked to the mouse cursor.

  6. While still holding the left mouse button down, press the + and - keys on the keyboard several times each. Make the highlight height just larger than the text height again.

    Each time you press the + key, you should see the highlight hight get larger. Each time you press the - key, you should see the highlight hight get smaller.

  7. While still holding the left mouse button down, drag the mouse so the cursor is near the middle and to the right of the "E".

    The Cyan box should completely cover the bearing "S11°26'38" E". If it doesn't, while still holding the left mouse button down click the right mouse button. This will cancel the highlight so you can do it over without actually recognizing.

  8. Once you have a highlight that entirely covers the bearing "S11°26'38" E" let go of the left mouse button.

    The highlight should change colors to red and the correct value should get recognized and displayed SOMEWHERE???.
