Recognizing text in a window

  1. Open the file IMAGE???.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Click the 'Recognize text in window' tool.

    The button should appear depressed and the cursor should look like a piece of paper with blue lines on it.

  3. Move the cursor to the so it is above and to the left of the area you want to recognize. Press and hold the left mouse button and drag the cursor around.

    You should see a box get drawn from the point that you picked to the cursor.

  4. While still holding the left mouse button down, drag the cursor so that it is below and to the right of the area you want to recognize. The box should entirely contain what you want to recognize. Once the box is the right size, let go of the left mouse button.

    The text contained within the box should get recognized and output to The legal text viewer???.
