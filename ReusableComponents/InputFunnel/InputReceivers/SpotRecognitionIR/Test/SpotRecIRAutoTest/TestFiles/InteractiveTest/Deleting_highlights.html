Deleting highlights

  1. Open the file Test010.tif.gdd.


    NOTE: If you need more information on opening .gdd files go to the following test case:
    Opening_gdd_files.html

  2. Click the 'Delete Highlights' button

    The 'Delete Highlights' button should appear depressed and the cursor should be shaped like the eraser end of a pencil.

  3. Window half of the highlights.

    As soon as you complete the window, all of the highlights touching and contained within the window should disappear.

  4. Click on the 'Open' button.

    The 'open' dialog should appear.

  5. Right click on Test010.tif.gdd.

    A floating menu should appear.

  6. Click on 'Properties'

    The 'Properties' dialog box will appear

  7. Go to the 'General' tab and uncheck the checkbox to the left of 'Read-only'. Click the 'OK' button.

    The file should not be read-only now.

  8. Cancel out of the 'Open' dialog box.

    The 'Open' dialog should disappear.

  9. Click the 'Save' button.

    The 'Save' dialog box should appear with the file name defaulted to Test010.tif.gdd.

  10. Save the file as Test010b.tif.gdd.


    NOTE: If you need more information on saving .gdd files go to the following test case:
    Opening_gdd_files.html

  11. Open the file Test010b.tif.gdd.


    NOTE: If you need more information on opening .gdd files go to either of the following test cases:
    Saving_gdd_files.html

  12. Click the 'Close' (X) button in the Image Recognition window.

    IcoMap should close the Image Recognition window.

The location of the files and any common instructions can be found here.
