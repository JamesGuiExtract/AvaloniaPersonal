Saving .gdd files.

  1. Open the file IMAGENAME???.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Highlight some text in the image.

    After each text zone gets highlighted, the area has a red box around the highlighted zone.

  3. Click the 'Save' button.

    The 'Save' dialog box should appear with the default name IMAGE???.gdd.

  4. Click the 'Save' button.

    The file IMAGE???.gdd should be saved.

The location of the files and any common instructions can be found here.
