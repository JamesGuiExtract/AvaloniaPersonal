Multi-page images.

  1. Open the file IMAGENAME???.

    In addition to the normal toolbar buttons that display after the image opens, the page navigation buttons should also display.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Click the 'move forward' button to cycle forward through the pages until you get to the last page.

    While cycling through the pages, the page number indicator should display the correct page number and the correct image should be displayed. When the last page is displayed the 'move forward' button should be disabled and page #6 should be displayed.

  3. Click the 'move backwards' button to cycle back through the pages until you get to the first page.

    While cycling through the pages, the page number indicator should display the correct page number and the correct image should be displayed. When the first page is displayed the 'move backwards' button should be disabled and page #1 should be displayed.

  4. Edit the value in the page number display field to 3 and press the 'Enter' key.

    Page 3 should be displayed in the image recognition window now.

  5. Edit the value in the page number display field to 5 and press the 'Enter' key.

    Page 5 should be displayed in the image recognition window now.

  6. Edit the value in the page number display field to 8 and press the 'Enter' key.

    You should have gotten an error message telling you that you entered an invallid page number. After you close the error message page 5 should still be the current page.

  7. Click the 'Go to the first page' button.

    Page 1 should be current.

  8. Click the 'Go to the last page' button.

    Page 6 should be current.
