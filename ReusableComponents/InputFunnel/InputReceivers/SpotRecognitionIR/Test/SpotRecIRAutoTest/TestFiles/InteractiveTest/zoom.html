Using the Zoom commands.

  1. Open the file IMAGENAME???.


    NOTE: If you need more information on opening images go to either of the following test cases:
    Opening_files.html

  2. Click the Zoom Window button.

    The Zoom button should appear depressed indicating that it is the active tool and the cursor should be shaped like a magnifying glass.

  3. Move the cursor to an area of the image that you would like to zoom into.
    Click the left mouse button.
    Drag the cursor (while holding the mouse button down) to a different location on the image. A box should appear from the first point you selected to the cursor.
    When the box surrounds the area you want to zoom into, let go of the left mouse button.

    The area you windowed should take up the majority of the Image Recognition window.

  4. Click the zoom extents button.

    The entire image should be visible in the Image Recognition window.

  5. Click the zoom in button a few times.

    Each time you click the zoom in button, the image should appear to get larger thus viewing less of the image at one time.

  6. Click the zoom out button a few times.

    Each time you click the zoom in button, the image should appear to get smaller thus viewing more of the image at one time.
