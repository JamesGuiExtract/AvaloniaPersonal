Read-only files

  1. Open the file Test002.txt.


    NOTE: If you need more information on opening text files go to the following test case:
    Opening.html

  2. Click the 'Open' button in the 'Legal Text' window

    The 'Open' dialog should get displayed.

  3. Right click on Test002.txt.

    A floating menu should appear

  4. Click on 'Properties'

    The 'Properties' dialog box will appear

  5. Go to the 'General' tab and click the checkbox to the left of 'Read-only'. Click the 'OK' button.

    The file should be read-only now.

  6. Cancel out of the 'Open' dialog box.

    The 'Open' dialog should disappear.

  7. Click the 'Save' button.

    The 'Save' dialog box should appear with the default file name being Test002.txt.

  8. Click the 'Save' button.

    An error should appear indicating that the file is marked as read-only and cannot be saved. You should also be allowed to save the file to a different name. Save it as Test002b.txt.
