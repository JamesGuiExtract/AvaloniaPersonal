Opening text files.

  1. Click the 'Open' button.

    The 'Open' dialog box should appear.

  2. Select the file TEXTFILE??? and click the 'Open' button.

    The 'Open' dialog box should disappear. The file TEXTFILE??? should open. The title bar should say TEXTFILE??? - Legal Text. All of the MCRable text should be highlighted yellow
