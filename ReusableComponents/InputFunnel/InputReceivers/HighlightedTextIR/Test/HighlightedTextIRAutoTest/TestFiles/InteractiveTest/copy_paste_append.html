Copy paste append

  1. Open the file Test004.txt.


    NOTE: If you need more information on opening text files go to the following test case:
    Opening.html

  2. Click the 'Copy' button in the 'Legal Text' window.

    All of the text should have been copied to the clipboard.

  3. Open Notepad and paste the text that you copied. Compare this file with what the 'Legal Text' window displays.

    The text in both files should be the same.

  4. Click the 'Close' (X) button in the 'Legal Text' window.

    The 'Legal Text' window should close.

  5. Re-open the Legal Text window from the test driver.

    The 'Legal Text' window should open.

  6. Click the 'Paste' button in the 'Legal Text' window.

    The same text you copied into Notepad should have pasted into the 'Legal Text' window.

  7. Add the following lines to the beginning of the text in notepad:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'

    Copy all of the text in Notepad to the clipboard.

    The text contained in Notepad should have been copied to the clipboard

  8. Click the 'Append' button in the 'Legal Text' window.

    The copied text should append to the end of the open file in the 'Legal Text' window. All of the mathematical text including the appended text should be highlighted yellow.

  9. Click the 'Save' button.

    A 'Save' dialog box should appear.

  10. Name the file Test004b.txt and click the 'Save' button.

    The save dialog should have closed and the file should have been saved.

  11. Click the 'Close' (X) button in the 'Legal Text' window.

    The 'Legal Text' window should close.

  12. Re-open the Legal Text window from the test driver.

    The 'Legal Text' window should open.

  13. re-Open the file Test004b.txt.

    The saved file should contain both the original text and the appended text.


    NOTE: If you need more information on opening text files go to the following test case:
    Opening.html

  14. Go the open file in Notepad and copy the following lines of text to the clipboard:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'
    Click the 'Paste' button in the 'Legal Text' window.

    All of the existing text in the 'Legal Text' window should have been replaced with the lines:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'
    All of the mathematical text should be highlighted yellow

  15. Click the 'Close' (X) button in the 'Legal Text' window.

    A warning message should get displayed telling the user that the current file has not yet been saved. They should then be asked if they want to save it.

  16. Click the 'Yes' button.

    The 'Save' dialog box should appear.

  17. Select Test004b.txt and click the 'save' button.

    The file Test004b.txt should get saved.

  18. Click the 'Close' (X) button in the 'Legal Text' window.

    The 'Legal Text' window should close.

  19. Re-open the Legal Text window from the test driver.

    The 'Legal Text' window should open.

  20. re-Open the file Test004b.txt.

    The saved file should only contain:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'
    All of the mathematical text should be highlighted yellow


    NOTE: If you need more information on opening text files go to the following test case:
    Opening.html
