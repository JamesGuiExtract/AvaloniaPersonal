Opening non-ASCII text files.

MICROSOFT WORD MUST BE INSTALLED FOR THIS TO WORK

  1. Click the 'Open' button.

    The 'Open' dialog box should appear.

  2. Change the 'Files of type:' drop down to 'Word perfect/micorsoft word...'

    Only .doc files should display.
    NOTE: I don't think that Word Perfect files have a .doc extension, I think they have .wpf extensions.

  3. Select the file TEXTFILE???.doc and click the 'Open' button.

    The 'Open' dialog box should disappear. There will be some processing to convert the non-ascii file to ascii. The file TEXTFILE???.doc should open. The title bar should say TEXTFILE???.doc - Legal Text. All of the MCRable text should be highlighted yellow
