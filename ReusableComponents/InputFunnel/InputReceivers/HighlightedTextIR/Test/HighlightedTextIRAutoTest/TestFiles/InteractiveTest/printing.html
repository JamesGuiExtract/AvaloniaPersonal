Printing from the Legal Text dialog.

  1. Open the file Test001.txt.


    NOTE: If you need more information on opening text files go to the following test case:
    Opening.html

  2. Click the ‘Print’ button in the Legal Text window.

    The ‘Legal Text’ window opens the print dialog box. When the print is initiated, the printed output will contain the contents of the specified file and all the text that was highlighted on screen will also be highlighted on paper. If printing to a color printer, the highlighted text that was not used for MCR should be printed yellow and the text that was used for MCR should be printed red.
