Convert FPS File Utility - Testing Checklist

NOTE: This utility is designed to convert pre-5.0 FPS files to the current standard.

NOTE: Referenced FPS files are located at .\UCLIDFileProcessing\Utils\InteractiveTest.

  1. Basic Operation and Error Conditions

    • Confirm that ConvertFPSFile requires a license file.

    • Confirm that an error message is displayed if only an input FPS file is defined.
    • Confirm that an error message is displayed if only an output FPS file is defined.
    • Confirm that a warning message is displayed if the output FPS file already exists.
    • Confirm that a second warning message is displayed if the output FPS file is the same as the input file.
    • Confirm that an error message is displayed if the input FPS file is newer than version 10.

  2. Convert Version 2 FPS File

    • Confirm that an error message is displayed when opening this FPS file in the FAM.
    • Confirm that V2_OCR_Images.fps converts successfully.
    • Confirm that the new version does not define an Action.
    • Confirm that the new version includes a Folder file supplier providing TIF Images from C:\FlexDataEntry\Images.
    • Confirm that the new version includes one OCR document file processor.
    • [LegacyRC #5288] Confirm that the application closes without error.

  3. Convert Version 5 FPS File

    • Confirm that an error message is displayed when opening this FPS file in the FAM.
    • Confirm that V5_Verify_Redactions.fps converts successfully.
    • Confirm that the new version does not define an Action.
    • Confirm that the new version includes a Folder file supplier providing TIF Images from C:\RedactionDemo\Images\4 Mix.
    • Confirm that the new version includes one Verify Redactions file processor.

  4. Convert Version 6 FPS Files

    • Confirm that an error message is displayed when opening this FPS file in the FAM.
    • Confirm that V6_Folder-Skip-USS.fps converts successfully.
    • Confirm that the new version does not define an Action.
    • Confirm that the new version includes a Folder file supplier and a File Existence skip condition.
    • Confirm that the new version includes one OCR document file processor.

    • Confirm that an error message is displayed when opening this FPS file in the FAM.
    • Confirm that V6_OCR_Images.fps converts successfully.
    • Confirm that the new version does not define an Action.
    • Confirm that the new version includes a Folder file supplier and a File Existence skip condition.
    • Confirm that the new version includes one OCR document file processor.

  5. Convert Version 7 FPS File

    • Confirm that an error message is displayed when opening this FPS file in the FAM.

    • Confirm that the new version includes one Copy, Move or Delete File file processor.

  6. Convert Version 8 FPS File

    • Confirm that an error message is displayed when opening this FPS file in the FAM.
    • Confirm that V8_OCR_test.fps converts successfully.
