Action Menu

  • Confirm that Add..., Remove..., Rename..., and Manually set action status... items exist in the menu.
  • Confirm that all menu items are enabled.

  • Connect to the database using Microsoft SQL Server Management Studio Express. Files queued by the File Action Manager can be found under Databases\<DatabaseName>\Tables\dbo.FAMFile. Right click dbo.FAMFile and select Open table to view files added to the database by the File Action Manager. The table will need to be closed and reopened each time an action is added or removed in order to see the changes. The table can be refreshed with Ctrl+R to view changes to an action's status.

  • Add...
    • Default status for action displays Unattempted by default.
    • Adding an one action with the each status adds a column ASC_<ActionName> to dbo.FAMFile with the following values:
      • Unattempted: U
      • Pending: P
      • Processing: R
      • Completed: C
    • Adding an action using Copy from adds a column ASC_<ActionName> to dbo.FAMFile with the expected values:
      • Unattempted: U
      • Pending: P
      • Processing: R
      • Completed: C
