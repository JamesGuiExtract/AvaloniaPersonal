File Action Manager - FAM Tag Manager Functionality

  • Recursively get the folder "FAM Tag Manager Functionality" from SourceSafe. It is located at:
    $\Engineering\ReusableComponents\COMComponents\UCLIDFileProcessing\InteractiveTest\FAM Tag Manager Functionality

  • <FPSFileDir> Functionality
    • Open a blank File Action Manager
      • Select any action, and then check the "Queue files for this action" checkbox.
      • On the Queue setup tab, click the "Add..." button. Select "Files from folder" from the "Select File Supplier" list, and then click the "Configure" button.
      • Use the tag selector or manually enter "<FPSFileDir>" into the "Specify folder from which to supply files" edit box.
      • Click "OK" to exit the "Configure Files from folder" dialog, and then click "OK" again to exit the "Select File Supplier dialog.
      • Run the file supplier by pressing the “Start processing” button on the toolbar.
      • Confirm that an error message appears that says, "There is no FPS File Name available to expand the <FPSFileDir> tag!"and the File Action Manager stops smoothly.
    • Clear the current database, and add an action called "FPSdir".
      • Open the file named Queue_In_FPS_Dir.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue log and confirm that FPSDir01.tif and FPSDir02.tif were both added to the queue.
    • Add an action called "SubDir" to the database
      • Open the file named Queue_In_Sub_Dir.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue log and confirm that FPSSubDir01.tif and FPSSubDir02.tif were both added to the queue.
    • Add an action called "Parent" to the database
      • Open the file named Queue_In_Parent_Dir.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue log and confirm that FPSParentDir01.tif and FPSParentDir02.tif were both added to the queue.
    • Add an action called "Sibling" to the database
      • Open the file named Queue_In_Sibling_Dir.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue log and confirm that FPSSiblingDir01.tif and FPSSiblingDir02.tif were both added to the queue.

  • File Suppliers
    • Files from dynamic list
      • Add an action called "Dynamic" to the Database
      • Open the file named Test_FPSDir_DynamicList.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue log and confirm that FPSDir01.tif, FPSSubDir01.tif, FPSParentDir01.tif, and FPSSiblingDir01.tif were all added to the queue.

  • File Processors
    • Copy, Move, or Delete File
      • Add an action called "Copy" to the database
      • Open the file named Test_Copy.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look in FAM Tag Manager Functionality\Main\Sub and confirm that two files, named FPSDir01COPY.TIF and FPSDir02COPY.TIF have been created.

    • Redact Images (no verification)
      • Add an action called "Redact" to the database
      • Open the file named Test_AutoRedact.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look in FAM Tag Manager Functionality and confirm that a directory name "Redacted" has been created and that it contains files FPSParentDir01_redacted.tif and FPSParentDir02_redacted.tif.

    • Verify Redactions
      • Add an action called "Verify" to the database
      • Open the file named Test_Verify.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Place a manual redaction and save every image that comes up in the ID Shield window.
      • Look in FAM Tag Manager Functionality\Main\Sub
        • Confirm that a directory name "Redacted" has been created and that it contains files FPSDir01COPY_redacted.TIF, FPSDir02COPY_redacted.TIF, FPSSubDir01_redacted.TIF, and FPSSubDir02_redacted.TIF.
        • Confirm that a directory name "MetaData" has been created and that it contains files FPSDir01COPY_metadata.xml, FPSDir02COPY_metadata.xml, FPSSubDir01_metadata.xml, and FPSSubDir02_metadata.xml.
        • Confirm that a directory name "ExpectedVOA" exists and that it contains files FPSDir01COPY.TIF.voa, FPSDir02COPY.TIF.voa, FPSSubDir01.TIF.voa, and FPSSubDir02.TIF.voa.

  • Skip Conditions
    • Skip based on file existance
      • Add an action called "SkipExist" to the database.
      • Open the file named Test_SkipExistance.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue Log and confirm that only FPSParentDir02.tif was added to the queue.

    • Skip based on file name pattern
      • Add an action called "SkipName" to the database.
      • Open the file named Test_SkipName.fps and run it. This FPS file is located in FAM Tag Manager Functionality\Main
      • Look at the "Files added or updated in the queue recently" display area of the Queue Log and confirm that FPSDir01COPY.TIF and FPSDir02COPY.tif was added to the queue.
