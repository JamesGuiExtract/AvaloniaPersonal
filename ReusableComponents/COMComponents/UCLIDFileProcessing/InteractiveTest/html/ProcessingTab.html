FAM File Processing Tabs

Setup
  • Launch File Action Manager.
  • Select a Database Server and a Database.
  • Select an Action.
  • Confirm checking the Process files in the queue for this action checkbox in the Action tab creates two new tabs: Processing Setup and Processing Log.
  • Confirm that clearing the Process files in the queue for this action checkbox makes Processing Setup and Processing Log tabs disappear.
  • Confirm the setting of the checkbox is persistent.

Processing Log Tab

  • Confirm that the Processing Log tab has three separate display areas: Files currently being processed, Files that completed processing recently, Files that failed processing recently.
  • Confirm that the Files currently being processed grid has columns for Date, Start Time, FileID, Filename, Pages, Current Task, Current Task Progress, Priority, Folder.
    • Confirm the Date column displays the date of the files currently being processed in the format of mm/dd.
    • Confirm the Start Time column displays the time of the files currently being processed in the format of HH:MM:SS.
    • Confirm the FileID column displays the file ID of the file currently being processed.
    • Confirm the Filename column displays the filename of the file currently being processed.
    • Confirm the Pages column displays the number of pages in the file.
    • Confirm the Current Task column displays the task(s) that are operating on the files.
    • Confirm the Current Task Progress column displays the overall progress as a numeric progress indicator. Confirm that units are in 0.01%.
    • Confirm the Priority column displays the priority of the file being processed.
    • Confirm the Folder column displays the name of the folder containing the file being processed.

  • Confirm the Files that completed processing recently grid has columns for Date, Start Time, FileID, Filename, Pages, Total Time(s), Priority, Folder.

    • Confirm Start Time column displays the start time of the files that completed processing in the last run in the format of HH:MM:SS.
    • Confirm the FileID column displays the file ID of the file.
