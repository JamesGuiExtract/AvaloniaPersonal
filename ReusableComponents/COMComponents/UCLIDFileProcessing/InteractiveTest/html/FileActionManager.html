File Action Manager UI

Basic User Interface

Command-Line Options
  • Confirm that ProcessFiles.exe /? on the command line results in display of a usage message.
  • Confirm that ProcessFiles.exe /r on the command line registers FPS files to open by default with ProcessFiles.exe.
  • Confirm that ProcessFiles.exe /u on the command line unregisters FPS files to open by default with ProcessFiles.exe.
  • Confirm that ProcessFiles.exe /sd <Server name> <Database name> on the command line opens the File Action Manager with the specified Server and Database.
  • Confirm that additional command-line options cannot be used with /sd.
  • Confirm that ProcessFiles.exe filename.fps /d <Directory name> on the command line opens the File Action Manager and changes the scope from that specified in the FPS file to the specified directory.
  • Confirm that ProcessFiles.exe filename.fps /l <List filename> on the command line opens the File Action Manager and changes the scope from that specified in the FPS file to the specified list.
  • Confirm that the /l command-line option cannot be used with /d.
  • Exercise the /service switch
    • Confirm that ProcessFiles.exe filename.fps /service on the command line opens the File Action Manager and loads the specified FPS file.
    • Confirm that the FPS file is not automatically saved.
    • Confirm that the FPS file stops processing and automatically closes if it receives a close message.
    • Confirm that the FPS file automatically closes after processing is complete.
