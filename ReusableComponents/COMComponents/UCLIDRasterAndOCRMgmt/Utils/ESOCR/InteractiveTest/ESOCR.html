Extract Systems OCR Utility (ESOCR) - Testing Checklist

  1. Basic Operation and Command-Line Arguments

    • Get local copies of test images from ReusableComponents/COMComponents/UCLIDRasterAndOCRMgmt/Utils/ESOCR/InteractiveTest.
    • Confirm that ESOCR requires a license file.
    • Confirm that an error message is displayed if ESOCR executes with only one argument.
    • Confirm that an error message is displayed if ESOCR executes with four arguments.
    • Confirm that ESOCR supports infile and outfile arguments with an optional /se argument.
    • [LegacyRC #4986] Confirm that ESOCR supports both absolute and relative paths for both infile and outfile arguments.

  2. Error Conditions

    • Confirm that TwoPages.tif displays Must be single-page image exception.
    • Confirm that TooTall.tif displays Image is too tall exception.
    • Confirm that TooWide.tif displays Image is too wide exception.

    • Confirm that TooWide.tif does not display an exception if the /se argument is used.
    • Also confirm that TooWide.tif with the /se argument does not create an output file.
    • Confirm that even when exceptions are thrown ESOCR does not throw an MFC error.

  3. Success Case

    • Confirm that SmallImage.tif OCRs successfully.
