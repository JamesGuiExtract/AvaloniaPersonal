New Functions

  • $FullUserName()
    • Confirm that $FullUserName() expands to the multiple word network user name for the logged in user.
  • $Now()
    • Confirm that $Now() without parameters expands to the current date and time in a YYYY-MM-DD-HH-MM-SS-mmm format where mmm is a three-digit, zero-padded millisecond count.
    • Confirm that $Now() properly expands the following parameters
      • %B - Full month name
      • %d - Day of month
      • %H - Hour in 24-hour format
      • %I - Hour in 12-hour format
      • %m - Month as decimal number
      • %M - Minute as decimal number
      • %p - AM / PM format
      • %S - Second as decimal number
      • %y - Year without century digits
      • %Y - Year with century digits
  • $RandomAlphaNumeric()
    • Confirm that $RandomAlphaNumeric() with an nLength parameter expands to a random sequence of alphanumeric characters of the specified length.
  • $UserName()
    • Confirm that $UserName() expands to the single word (first_last) user name for the logged in user.
