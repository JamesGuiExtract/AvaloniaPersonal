Text Function Expander

Test "D:\Engineering\ReusableComponents\BaseUtils\InteractiveTest\Test  Image.tif"
  1. ***CREATE A DRIVE MAPPING TO H*** Several tests require write privileges on drive H:\.

  2. Confirm that you are logged in to the test machine as a user that has a Full Name. Note that the Administrator account does not have a Full Name.
  3. Add the CommonComponents folder to the system path.
  4. Navigate to C:\ProgramData\ExtractSystems\TestingFiles\ReusableComponents\BaseUtils\InteractiveTest\.
  5. Run TestFunctionExpanderTest.bat. The test will create files in the directory C:\ProgramData\ExtractSystems\TestingFiles\ReusableComponents\BaseUtils\InteractiveTest\. and a few files under H:\.
  6. Review testResults.txt and note any not created items.
  7. Review the testing results for the Specific Functions discussed below. These functions are not well suited for automated testing.

    Specific Functions

    • $FullUserName()
      • Confirm that a 15_<FullUserName>.xml file exists for the logged in user.

      • [LegacyRC #5672] $FullUserName() arguments
        • Log in as local admin or as another user that does not have a FullName. The remaining tests can be exercised with TestTextFunctionExpander.exe.
        • Confirm that the $FullUsername() function supports an argument of 0 or 1.
        • Confirm that for 0, an exception is thrown if the full user name cannot be retrieved.
        • Confirm that for 1, an exception is not thrown if the full user name cannot be retrieved - instead the $UserName() text is provided.
        • Confirm that for $FullUsername() without a parameter, functionality is the same as for 0.
    • $Now()
      • Confirm that a 16_Now folder exists in the test directory. Check for the following contents. Note that *** indicates one or more unknown characters.
        • An AMPM_***.xml file.
        • An DayOfMonth_***.xml file.
        • An FullDateTime_***.xml file.
        • An FullMonthName_***.xml file.
        • An Hour_12HourTime_***.xml file.
        • An Hour_24HourTime_***.xml file.
        • An MinuteDecimal_***.xml file.
        • An MonthDecimal_***.xml file.
        • An SecondDecimal_***.xml file.
        • An Year2Digit_***.xml file.
        • An Year4Digit_***.xml file.
      • Confirm that $Now() without parameters expands to the current date and time in a YYYY-MM-DD-HH-MM-SS-mmm format where mmm is a three-digit, zero-padded millisecond count.
    • $ProcessID()
      • Confirm that a 20_Test  Image.tif.PID_??????.xml file exists.
      • Confirm that the $ProcessID() function returns the correct ID (compare with the ID shown in the Task Manager).
    • $RandomAlphaNumeric()
      • Confirm that a 17_???.xml file exists where ??? indicates 3 random alphanumeric characters.
    • $ThreadID()
      • Confirm that a 21_Test  Image.tif.TID_??????.xml file exists.
    • $UserName()
      • Confirm that a 18_<UserName>.xml file exists for the logged in user. The user name is expected to be a single word.

  8. The test is complete. Run clean.bat to remove the files created by the test.
  9. Remove the Test  Image.tif.xml file in the appropriate subfolder under drive H:\ as created by the test.
  10. Remove the drive mapping for H:\.
