New Functions

  • $FullUserName()
    • Confirm that $FullUserName() expands to the multiple word network user name for the logged in user.

    • [LegacyRC #5672] $FullUserName() arguments
      • Confirm that the $FullUsername() function supports an argument of 0 or 1.
      • Confirm that for 0, an exception is thrown if the full user name cannot be retrieved.
      • Confirm that for 1, an exception is not thrown if the full user name cannot be retrieved - instead the $UserName() text is provided.
      • Confirm that for $FullUsername() without a parameter, functionality is the same as for 0.
  • $Now()
    • Confirm that $Now() without parameters expands to the current date and time in a YYYY-MM-DD-HH-MM-SS-mmm format where mmm is a three-digit, zero-padded millisecond count.
    • Confirm that $Now() properly expands the following parameters
      • %B - Full month name
      • %d - Day of month
      • %H - Hour in 24-hour format
      • %I - Hour in 12-hour format
      • %m - Month as decimal number
      • %M - Minute as decimal number
      • %p - AM / PM format
      • %S - Second as decimal number
      • %y - Year without century digits
      • %Y - Year with century digits
  • $RandomAlphaNumeric()
    • Confirm that $RandomAlphaNumeric() with an nLength parameter expands to a random sequence of alphanumeric characters of the specified length.
  • $UserName()
    • Confirm that $UserName() expands to the single word (first_last) user name for the logged in user.
  • $Left()
    • Confirm that $Left("Hello World",5) outputs "Hello"
  • $Right()
    • Confirm that $Right("Hello World",5) outputs "World"
  • $Mid()
    • Confirm that $Mid("Hello World",4,4) outputs "lo W"
    • Confirm that $Mid("Hello World",4,-1) outputs "lo World"
  • $ChangeExtension()
    • This function takes two arguments, a file name and a target extension.
    • Confirm that the function returns the file name with the original extension replaced by the target.
