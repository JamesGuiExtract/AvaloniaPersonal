      • a Lab Code edit box.

        • Test Name
        • Code
        • Value
        • Units
        • Ref. Range
        • Flag
      • a Result Component Comment multi-line edit box.

  2. Validation

    • Confirm that the image cannot be saved and committed while one or more red warning icons remain visible.
    • Confirm that a Medical Record # is required.
    • Confirm that each of the following fields are required for each Selected Order Details item.
      • Accession Number
      • Lab Code
    • Confirm that a valid Accession Number must include: a single alphabetic character followed by exactly 8 digits.
      • Confirm that a single trailing space character is also allowed.
