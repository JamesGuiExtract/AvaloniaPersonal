LabDE 1.0 Beta 4 Testing

Purpose and Details

  • Purpose: Exercise recent fixed SCRs
  • Details: Recently added SCRs generally do not have official test cases present within the collected checklists. One part of validating a Beta 4 release candidate is to explicitly exercise these recent changes.

SCRs

  1. [DataEntry #145, #163, #194, #260, #314] Windows Tooltips For Swipe Problems
    • Exercise a LabDE image.
    • Select a cell and swipe a white area so that no text is recognized.
    • Confirm that a tooltip is displayed describing the problem.
    • Swipe an area that extends off the page.
    • Confirm that a tooltip is displayed describing the problem.

    • Exercise an image that causes an OCR Engine exception.
    • Confirm that a tooltip is displayed describing the problem.

  2. [DataEntry #306, #615] Characters In New Row
    • Exercise a LabDE image.
    • Select the new row of a table.
    • Turn on CAPS LOCK.
    • Type some characters.
    • Confirm that every typed character appears in upper case.
    • Turn off CAPS LOCK.

    • Select the new row of another table.
    • Type some characters.
    • Confirm that every typed character appears in the first cell.

  3. [DataEntry #412] Shortcuts And Frozen Application
    • Exercise a LabDE image in stand-alone mode.
    • Close the document.
    • Test the following shortcuts
      • Ctrl+S - Save
      • Ctrl+P - Print
      • Ctrl+T - Accept Highlight
      • Ctrl+D - Remove Highlight
      • Alt+Z - Zoom Tool
      • Alt+A - Pan Tool
      • Alt+R - Select Tool
      • Alt+S - Swipe Tool
    • Confirm that the application is not frozen after each shortcut.

  4. [DataEntry #440, #512, #589, #590, #609, #610] Various Installation Issues
    • Exercise the Demo Shield installation.
    • Confirm that .NET 3.5 is the first item listed.

    • Confirm that a LabDE icon is used for the installation.

    • Finish the installation.
    • Check TBD and confirm that the installed version of the Corepoint software is 4.0.1.
    • Confirm that Verify extracted data is available as a processing task.

    • Confirm that an associated RDT installation is available with LabDE.

    • Uninstall LabDE.
    • Confirm that the Lead Tools DLLs are removed.
    • Confirm that the Extract Systems Interop DLLs are removed.

  5. [DataEntry #516] Text For Toolbar Buttons
    • Exercise a LabDE image.
    • Confirm appropriate tooltip text for the Review and Select tool.
    • Confirm appropriate tooltip text for the Angular Highlight tool.
    • Confirm appropriate tooltip text for the Rectangular Highlight tool.

  6. [DataEntry #517] Rectangle Selection With Review Tool
    • Exercise a LabDE image.
    • Exercise the Review and select tool.
    • Confirm that rectangle selection is no longer supported.

  7. [DataEntry #532] Unnecessary Zoom After Swipe
    • Exercise the I:\Common\Testing|PVCS_Testing\DataEntry\144 image.
    • Swipe some phone numbers in the top left corner of the image.
    • Confirm that no unexpected zooms occur.

  8. [DataEntry #536] Dragging Row Within Same Table
    • Exercise a LabDE image.
    • Drag a table row to the new row within the same table.
    • Confirm that the table rows are re-ordered as expected and that the dragged row is not duplicated.

  9. [DataEntry #571, #573, #603] Operator Comment Field And Tab Stop
    • Open a LabDE demo image.
    • Confirm that the Operator Comment field is not a tab stop if the field is empty.
    • Confirm that the Operator Comment field is a tab stop if the field is non-empty.
    • Confirm that the Operator Comment field has a vertical scrollbar if enough text is entered.

  10. [DataEntry #572] LabDE Icon
    • Confirm that the DEP uses the LabDE icon.
    • Confirm that DataEntryApplication.exe uses the LabDE icon.

  11. [DataEntry #578] Over Persistent Tooltip
    • Open a LabDE demo image.
    • Select a row in the Test table.
    • Use the Review and select tool to select a value.
    • Click on a Units cell in the table.
    • Confirm that Value highlight disappears and is replaced by the Units highlight.
    • Note that this bug was not always readily reproduced.

  12. [DataEntry #579] Performing Lab Populated Via Lab List
    • Exercise several LabDE images.
    • Confirm that the Laboratory information is provided in the DEP.

  13. [DataEntry #581] Cells Marked As Viewed On Open
    • Exercise a LabDE image.
    • Confirm that the first cells in various tables are not presented as already viewed immediately after opening the image.

  14. [DataEntry #587] Context Menu Paste
    • Exercise a LabDE image.
    • Copy some data to the Clipboard.
    • Paste the copied data to an appropriate location via Context menu.
    • Confirm that the pasted data is selected.

  15. [DataEntry #588, #593] Issues With Swiping Vertical Text
    • Exercise a LabDE image.
    • Swipe some vertical text into a first name field.
    • Confirm that a Cannot create tooltip for empty attribute! exception is not displayed.

    • Rotate the image 90 degrees.
    • Click on a populated field that is associated with a highlight.
    • Confirm that a Cannot create tooltip for empty attribute! exception is not displayed.

  16. [DataEntry #601] Tooltips Displayed For Direct Hints
    • Exercise a LabDE image.
    • Select a cell that should have a value but instead has only a direct hint.
    • Type some text into the cell.
    • Confirm that the typed text is displayed as a tooltip.

  17. [DataEntry #602] Crash Without Pressing Copy Button
    • Exercise a LabDE image using the MatchToPatientAndOrder configuration.
    • Correct all invalid items except the Copy button.
    • Attempt to Save and Commit the document.
    • Confirm that an exception is displayed noting that the Copy button has not been pressed.
    • Confirm that the application does not crash.

  18. [DataEntry #604] Initial Selection And Tab Stop
    • Exercise a LabDE image without pre-defined Operator's Comments content.
    • Confirm that the Operator's Comments field is not the initial selection.

    • Exercise a LabDE image with pre-defined Operator's Comments content.
    • Confirm that the Operator's Comments field is the initial selection.

  19. [DataEntry #606] Alignment of About Box Text
    • Confirm that the text in the About Box is aligned correctly.

  20. [DataEntry #611] Repeated Exceptions With Auto-Update Query
    • Exercise a LabDE image.
    • Select the Name cell of a Test row.
    • Paste the text provided in the SCR into the cell.
    • Confirm that an exception is displayed.
    • Close the exception and confirm that the exception is not displayed again.

  21. [LegacyRC #5388, #5396] Skipped Files Do Not Revert To Pending
    • Exercise several LabDE images.
    • Skip a document and stop processing.
    • Reconfigure the task to process only documents that I skipped.
    • Start processing and skip the file again.
    • Confirm that the skipped document remains in Skipped status.

    • Exercise several LabDE images.
    • Skip some documents and stop processing.
    • Confirm that all skipped documents remain in Skipped status.
