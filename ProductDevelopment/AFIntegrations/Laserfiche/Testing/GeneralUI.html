ID Shield for Laserfiche
General UI Testing Checklist

  • Start Menu

    • Confirm presence of a program group for Extract Systems ID Shield for Laserfiche.

    • Confirm that the program group contains an entry for ID Shield Admin Console.
    • Confirm that the expected Select Repository window is displayed.
    • Confirm that after providing the correct password, the expected ID Shield Administration Console window is displayed.

    • Confirm that the program group contains an entry for ID Shield Background Service.
    • Confirm that the expected ID Shield Background Laserfiche Redaction Sevice window is displayed.

    • Confirm that the program group contains an entry for ID Shield for Laserfiche Help.
    • Confirm that the expected Help file is displayed.

  • Laserfiche Client Toolbar Buttons

    • From the Laserfiche Start menu, start the Laserfiche Client.
    • Open the ID Shield Demo repository.

    • Confirm presence of a toolbar button for ID Shield that resembles a blue shield on a red background.
    • Confirm that the tooltip is Process with ID Shield.

    • Confirm presence of a toolbar button for ID Shield background service that resembles a gold plus sign with a thin blue arrow in front of a computer.
    • Confirm that the tooltip is Process in background with ID Shield.

    • Confirm presence of a toolbar button for ID Shield Verification that resembles a document page with a large green check mark.
    • Confirm that the tooltip is Verify ID Shield redactions.
