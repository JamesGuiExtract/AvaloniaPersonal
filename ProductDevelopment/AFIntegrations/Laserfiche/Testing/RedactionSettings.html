ID Shield for Laserfiche
Redaction Settings and Behavior

  • Testing Setup

    • Create a new virtual machine using the virtual machine disk ( .vhd file ) at: I:\Common\Engineering\VirtualMachines\Laserfiche_72.
    • Close the Laserfiche Client.
    • Install ID Shield for Laserfiche on the virtual machine.
    • Retro-grade to FKB update 8.2. Do not use the FKB update that was installed with ID Shield for Laserfiche.
    • Generate a Laserfiche Integration license file. Copy the license to the usual CommonComponents folder.
    • Launch USB License Key Manager and point to a machine that contains a USB key (usually Lisa).
    • Launch Rule Tester and verify the About Rule Tester button displays the correct version and FKB numbers.
    • Configure the repository for use with ID Shield
      • Launch ID Shield Admin Console.
      • Verify ID Shield Demo appears in the repository dropdown list of the Select Repository dialog.
      • Select ID Shield Demo and log in with the username "admin" and password "a".
      • Click on the About and verify the ID Shield and FKB versions.
      • On the Repository tab, click the Create Tags button and verify the Admin Console reports that the tags were successfully created.
      • Select ID Shield Demo and log in with the username "admin" and password "a".
      • In the Laserfiche client create a new top-level folder called ID Shield.
      • Use the Fields | Tags context menu to apply the ID Shield - Settings tag to the ID Shield folder.
      • In the ID Shield Admin Console, press the Refresh button.
      • Confirm \ID Shield is displayed as the settings folder following the refresh.
      • On the Redaction Settings tab enter "I:\Common\Testing\Product Testing\Laserfiche\Rules\ID Shield for LF.rsd.etf" in the Redaction Rules box
      • Press "OK" to accept the default value for all other settings.

  • Test HC Data Setting

    • Test Setup
      • Open the admin Console
      • Enter "I:\Common\Testing\Product Testing\Laserfiche\Rules\ID Shield for LF.rsd.etf" in the Redaction Rules box
      • Press "OK" to accept the settings.
      • In the Redaction Settings tab use the following settings:
        • Checked Redact high confidence data
        • Unchecked Redact medium confidence data
        • Unchecked Redact low confidence data
        • Checked Automatically tag processed documents for verification
        • Unchecked Tag all documents that are processed
        • Checked Tag only documents that contain evidence of sensitive data or clues
        • Checked Automatically initiate document verification after on-demand processing if one or more documents were tagged for verification
        • Unchecked Upon verification of a document ensure a corresponding text or image redaction exists for all manually added redactions.
      • In the Laserfiche Client, create a new top-level folder called HCData Test.
      • Copy all 10 documents from Documents into this folder

      • When prompted, press Yes to re-process the documents.
      • Confirm that processing completes with the message Processing Complete! and that the documents are now tagged ID Shield - Processed as well as ID Shield - Needs Verification.

  • Test Ensure Redactions Setting

    • Test Setup
      • Open the admin Console
      • In the Redaction Settings tab use the following settings:
        • Checked Redact high confidence data
        • Checked Redact medium confidence data
        • Checked Redact low confidence data
        • Checked Automatically tag processed documents for verification
        • Checked Tag all documents that are processed
        • Unchecked Tag only documents that contain evidence of sensitive data or clues
        • Checked Automatically initiate document verification after on-demand processing if one or more documents were tagged for verification
        • Checked Upon verification of a document ensure a corresponding text or image redaction exists for all manually added redactions.
      • In the Laserfiche Client, create a new top-level folder called Corresponding Redactions Test.
      • Copy document 06 from Documents into this folder
      • OCR The document in the Laserfiche Client
        • Select the document in the Corresponding Redactions Test folder.
        • Select the Action | OCR/Extract Text/Index... menu option (or button)
        • Check OCR / Extract Text
        • Select All image pages
        • Under options, make sure all options are unchecked
        • Press OK to OCR the document.
      • In the Tools menu of the Laserfiche Client, ensure Link Text Annotations is unchecked.

    • Run the test
      • In the Corresponding Redactions Test folder select document 06 and press the On-demand redaction button.
      • Confirm that the document launches for verification once processing is complete.
      • While the document is open for verification, in the image pane, in column (d) carefully add a manual redaction over the date 05/20/2002 using the Laserfiche Client's redaction tool.
      • Press the verify button on the ID Shield verification toolbar.
      • Confirm the status box Validating redacions... is displayed
      • Confirm a Yes/No prompt is displayed indicating that the document needed to be modified to ensure corresponding text and image redactions.
      • Press Yes on the prompt and confirm that the document is re-opened for verification.
      • In the text pane confirm that a text redaction has been added over the date 05/20/2002
      • In the text pane, using Laserfiche's redaction tool, add a text redaction over the date just below (05/26/2003) and be sure not to link the redaction to the image.
      • Press the verify button on the ID Shield verification toolbar.
      • Confirm the status box Validating redacions... is displayed
      • Confirm a Yes/No prompt is displayed indicating that the document needed to be modified to ensure corresponding text and image redactions.
      • Press Yes on the prompt and confirm that the document is re-opened for verification.
      • Press the verify button on the ID Shield verification toolbar without making any further changes
      • Confirm verification ends with the message Verification Complete!

      • Right-click on the document in the Laserfiche Client and select Fields... | Tags.
      • Deselect ID Shield - Verified and select ID Shield - Needs Verification instead.
      • With the document still selected, press the Verify redactions button in the Laserfiche Client.
      • When the document opend for verification, in the image pane add a redaction over the last date in column (d): 11/14/2005.
      • Press the verify button on the ID Shield verification toolbar.
      • Confirm a Yes/No prompt is displayed indicating that the document needed to be modified to ensure corresponding text and image redactions.
      • Press No.
      • Confirm that verification completes.
      • Open the document in the Laserfiche Client and confirm that a text redaction has been added over the date 11/14/2005 in the text pane.
