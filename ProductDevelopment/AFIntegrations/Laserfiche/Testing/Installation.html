ID Shield for Laserfiche
Installation Testing Checklist

  1. Basic Install

    • Confirm that the Extract Systems ID Shield for Laserfiche installation includes an appropriate splash screen.
    • Confirm that the installation uses the product name Extract Systems ID Shield for Laserfiche.
    • Installation screens.
      • Confirm that a Welcome screen is presented.
      • Confirm that a License Agreement screen is presented. Also confirm that the license terms must be accepted before installation will continue.
      • [Integration #57] Confirm that ID Shield for Laserfiche is part of the license agreement instead of ID Shield or IDShield.
      • Confirm that the Choose Destination Location screen is presented.
      • Confirm that the default destination folder is C:\Program Files\Extract Systems\Laserfiche.
      • Confirm that the Setup Status screen displays installation progress.
      • Confirm that the user is prompted to restart their computer after the installation is complete.

    • Verify the Extract Systems ID Shield For Laserfiche Start Menu shortcuts
      • Confirm Field Activation Utility
      • Confirm ID Shield Admin Console
      • Confirm ID Shield Background Service
      • Confirm ID Shield for Laserfiche Help
      • Confirm Rule Tester
      • Confirm USB License Key Manager
      • Confirm User License
      • Confirm View Exception Log

    • Verify ID Shield buttons & add them to the Laserfiche Client's toolbar
      NOTE: Skip this step when testing on machines that do not have the Laserfiche Client installed.
      • Open the Laserfiche Client
      • Right-click anywhere in the toolbar area and select "Customize..."
      • Confirm that there are 4 ID Shield buttons present in the "Available toolbar buttons" list
      • Add all 4 buttons to the "Current toolbar buttons" list.
      • Close the customize window and confirm the presence of all 4 buttons on the toolbar:
      • Confirm the button tooltip is On-demand redaction.
      • Confirm the button tooltip is Submit for background redaction.
      • Confirm the button tooltip is Verify redactions.
      • Confirm the button tooltip is ID Shield help.

      • [Integrations #67] Toolbar registry information
        • Open the Registry Editor and check HKLM\Software\Laserfiche\Client\Profile\Toolbars.
        • Confirm that there are 4 ID Shield buttons present in subfolders Custom42, Custom43, Custom44, and Custom45.
        • Confirm that if any of the Custom subfolders are already present, the numbers will be incremented by one.

    • Verify Laserfiche Integrator's Toolkit 7.2.1 Run-Time Components is listed in the Add/Remove Programs list.

  2. Licensing

    • Confirm that launching the ID Shield Admin Console via the Start menu displays a meaningful error message if no license file is available.
    • Launch User License.
      • Verify Extract Systems ID Shield for Laserfiche is listed in the products dropdown box.
      • Verify and that the correct version is diplayed when Extract Systems ID Shield for Laserfiche is selected.
      • Verify the "Type" button is enabled when ID Shield for Laserfiche is selected.
      • Verify that supported Type items are:.
        • Verification only
        • Desktop redaction
        • Server redaction
      • Verify that the Copy to clipboard button works when all Step 1 information is filled-in and fails if the information is not filled-in.
      • Paste into notepad after using the Copy to clipboard button.
      • Confirm the text includes the license string, product name, version, type, Name, Company, Phone and Email.
      • Verify that the Email Address box displays "flex-license@extractsystems.com" when ID Shield for Laserfiche is selected.
      • Verify that the if all step 1 information is filled in the Create Email Automatically button creates a new email addressed to "flex-license@extractsystems.com", with a subject of "License Request" and with a body containing the same information that was generated by Copy to Clipboard.
      • Use the license string to generate a license file for Laserfiche Integration (and no other products).

    • Testing License Files.
      • Generate a separate Verification only license file.
      • Generate a separate Desktop redaction license file.
      • Generate a separate Server redaction license file.

      • With only the Verification only license file, start the Laserfiche Client.
        • Confirm that an image can be verified.
        • Confirm that a licensing-related error message is presented when trying to do on-demand redaction.
        • Confirm that a licensing-related error message is presented when trying to submit documents for background redaction.
        • Confirm that a licensing-related error message is presented when trying to start the ID Shield Laserfiche Background Redaction service.

      • With only the Desktop redaction license file, start the Laserfiche Client.
        • Confirm that an image can be verified.
        • Confirm that on-demand redaction is successful.
        • Confirm that submitting documents for background redaction is successful.
        • Confirm that a licensing-related error message is presented when trying to start the ID Shield Laserfiche Background Redaction service.

      • With only the Server redaction license file, start the Laserfiche Client.
        • Confirm that an image can be verified.
        • Confirm that on-demand redaction is successful.
        • Confirm that submitting documents for background redaction is successful.
        • Confirm that starting the ID Shield Laserfiche Background Redaction service is successful.

  3. Basic Operation - use Server redaction license

    • Admin Console
      • Start ID Shield Admin Console
      • Confirm that any available repositories on the network are listed in the Laserfiche Repositories dropdown (example: Large Scale Test")
      • Confirm that you can connect to a repository.
      • Press cancel to close the console.

    • Background Service
      • Right-click on My Computer (or Computor) and select Manage.
      • Select Services under either Services and Applications or Configuration.
      • Confirm the presence of Extract Systems ID Shield for Laserfiche in the services list.
      • Start the ID Shield Background Service console via the Windows Start menu.
      • Confirm that the service console launches without error and displays any available repositories on the network in the Laserfiche Repositories dropdown.
      • Press Cancel to close the console.

    • Help File
      • Start ID Shield for Laserfiche Help and confirm that the Help file is opened.

  4. Sanity Testing

    • Add images to repository
      • Open the Laserfiche Client.
      • Create a Documents folder under the main repository.
      • Drag and drop the sample images into the Documents folder from I:\Common\Testing\Product Testing\Laserfiche\SanityImages.
      • Select all the documents that were added to the repository
      • Select the Action | OCR/Extract Text/Index... menu option (or button)
      • Check OCR / Extract Text
      • Select All image pages
      • Press OK to OCR the documents.

    • Background redaction
      • Launch the ID Shield Background Service console from the start menu
      • Specify the repository and login information to be used for testing.
      • Start the Extract Systems ID Shield for Laserfiche service and confirm the service status changes first to Starting... and then to Running.
      • From the Laserfiche Client, select the Documents folder and click Submit for background redaction.
      • Using Windows task manager, confirm that within 1 minute CPU usage increases and SSOCR2.exe process instance(s) appear.

    • On-demand redaction
      • While the background service is still processing documents, select half of the images in the Documents folder and click On-demand redaction. NOTE: Depending upon which documents are being processed in the background, ID Shield will likely report that some documents are locked and/or have already processed. Choose to re-process any unlocked documents.
      • Confirm that verification is launched once on-demand redaction completes.
      • In the first document window that appears during verification, use the Tools or Annotations menu (depending on the Client version) to turn off automatic linking of newly added annotations.
      • While verifying the documents, add occasional image-based or text-based redactions and confirm that ID Shield adds corresponding text/image redactions and prompts accordingly.
      • Confirm all documents processed during on-demand redaction are presented for verification, but not the files processed only in the background.

    • Verification
      • Once the CPU usage goes down, the background processing is complete.
      • Select the Documents folder and click Verify redactions
      • Verify the documents as was done during on-demand redaction, but ensure this time that only the documents processed in by the background service are presented for verification.

    • Service auto-start
      • NOTE: There is a known issue that prevents Extract Systems products from running if the service is running and was launched outside of our service console with the logon for the service set to the local system account. Therefore, as of version 1, it is best to first set the logon to any user account on the test machine or domain before
      • Open the ID Shield Background Service console.
      • Start the service if it is not still running.
      • Check the Run background service automatically on startup box
      • Restart the machine.
      • Once the machine restarts, open the Laserfiche client and re-submit one document for background processing.
      • Use task manager to verify the "ESLaserficheService.exe" is still running and that it begins to process the submitted document within one minute of submission.

  5. Basic Uninstall

    • [Integration #69] Uninstall Extract Systems ID Shield for Laserfiche.
    • Confirm that the uninstall process asks whether to uninstall Laserfiche Integrator's Toolkit 7.2.1 Run-Time Components.
    • Confirm that the message recommends retaining Laserfiche Integrator's Toolkit 7.2.1 Run-Time Components if the Laserfiche Client is to be used.
    • If Laserfiche Integrator's Toolkit 7.2.1 Run-Time Components was uninstalled, confirm that it does not remain in the Add/Remove programs list.
    • If Laserfiche Integrator's Toolkit 7.2.1 Run-Time Components was retained, confirm that it does remain in the Add/Remove programs list.
    • When the uninstall process completes, confirm that Extract Systems ID Shield for Laserfiche does not remain in the Add/Remove programs list.
    • Confirm that installed ID Shield files have been removed.
    • Confirm that Start menu items have been removed.
    • Open the Laserfiche Client and confirm that the ID Shield buttons are missing from the toolbar.
    • Via the Window's service console, confirm that the service Extract Systems ID Shield for Laserfiche no longer exists.
