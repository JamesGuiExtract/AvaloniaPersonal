ID Shield for Laserfiche Testing Checklist

  1. Laserfiche Setup

    • Laserfiche Installs
      • Laserfiche 7.2 installs (both Client & Server) can be found on the Laserfiche PDP DVD.
      • The license files for the PDP installs can be found at \\jake\Applications\Laserfiche\license
      • Laserfiche 8 installs (both Client & Server) can be found at \\jake\Applications\Laserfiche\Laserfiche 8.0
      • Laserfiche 8 demo licenses can be found at \\jake\Applications\Laserfiche\8.0 licenses

    • Laserfiche VMs
      • A VM with the Laserfiche 7.2 Client & Server pre-installed is here: I:\Common\Engineering\VirtualMachines\Laserfiche_72
        • The machine name (used as Laserfiche Server Name) is Laserfiche-72.
        • Has a locally hosted repository ID Shield Demo with the following logins:
          • Username: Admin, Password: a
          • Username: User, Password: a
          • Username: Peon, Password: a
        • The repository has 10 images that are used by several of the checklists below.
        • The Laserfiche Administration Console is installed to allow maintenance of this or other Laserfiche repositories.

      • A VM with the Laserfiche 8 Client & Server pre-installed is here: I:\Common\Engineering\VirtualMachines\Laserfiche_8
        • The machine name (used as Laserfiche Server Name) is Laserfiche-8.
        • Has a locally hosted repository LF8 with the following logins:
          • Username: Admin, Password: a
          • Username: User, Password: a
          • Username: Peon, Password: a
        • The repository has 10 images that are used by several of the checklists below.
        • The Laserfiche Administration Console is installed to allow maintenance of this or other Laserfiche repositories.

    • Laserfiche Large Scale Repository
      • The Endurance Test checklist should be run against a large scale database (one million pages or greater). Large Scale Repository Configuration covers a couple of ways to set up such a repository.
