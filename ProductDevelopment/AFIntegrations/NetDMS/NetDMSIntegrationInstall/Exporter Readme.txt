This package installs a NetDMSExporter assembly which includes the following method that can be called from a NetDMS script to export documents for processing in Extract System's ID Shield:
NetDMSExporter.NetDMSDocumentExporter.Export(taskClient, workItem, exportPath)

Install this package by extracting all files to a local folder on the machine with NetDMS Task Dispatcher installed and running "install.bat".
