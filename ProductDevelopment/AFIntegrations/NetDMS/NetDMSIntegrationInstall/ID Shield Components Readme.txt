What's included in this update:
This patch adds NetDMS integration components to an ID Shield installation.

Install this patch by extracting all files to a local folder on a machine with ID Shield and NetDMS Workflow Console, Workflow Configuration Manger or Task Dispatcher installed and running "install.bat".

The NetDMS integration includes the following components:
- A new FAM task "NetDMS: Add Redactions" which can add redactions to a document in NetDMS.
- A new FAM task "NetDMS: Advance Work Item" which can advance a work item from the node it was exported from using the "Auto Send" NetDMS feature.
