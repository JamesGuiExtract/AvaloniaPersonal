ID Shield for NetDMS
Advanced Testing Checklist

NOTE: Setup and Configuration details are provided on the basic checklist.

  • Scenario: Verification

    • Configure an ID Shield Server test machine that can also be used as a NetDMS Client.
    • Prepare an FPS file so that files are queued and rules are run.
    • Prepare another FPS file to support verification of processed images where redactions are applied after verification is complete.
    • From the WorkFlow User Console, create a work item and use Capture - File pickup to add a TIF image to PROGRESSDOCKET (the node immediately preceding Extract).
    • Confirm that the image is queued and processed as expected.
    • Confirm that the image is presented for verification as expected.
    • Confirm that redactions are applied as expected.
    • Confirm that the work item advances through NetDMS as expected.

  • Scenario: All-In-One Processing

    • Configure a NetDMS Server machine to also be an ID Shield Server.
    • Prepare an FPS file to support automatic processing.
      • Queue files and run rules
      • Apply redactions
      • Advance the work item through NetDMS
    • From the WorkFlow User Console, create a work item and use Capture - File pickup to add a TIF image.
    • Confirm that the image is queued and processed as expected.
    • Confirm that redactions are applied as expected.
    • Confirm that the work item advances through NetDMS as expected.

    • Modify the ID Shield workflow to include verification of sensitive images.
    • Confirm that image are queued and processed as expected.
    • Confirm that images are presented for verification as expected.
    • Confirm that redactions are applied as expected.
    • Confirm that the work item advances through NetDMS as expected.

  • Scenario: ID Shield Processing As Service

    • Configure an ID Shield Server test machine that can also be used as a NetDMS Client.
    • Prepare an FPS file to support the following:
      • Queue files and run rules
      • Apply redactions
      • Advance the work item through NetDMS
    • Prepare the FAM Service to run the FPS file.
    • From the WorkFlow User Console, create a work item and use Capture - File pickup to add a TIF image.
    • Confirm that the image is queued and processed as expected.
    • Confirm that redactions are applied as expected.
    • Confirm that the work item advances through NetDMS as expected.

  • Scenario: Process TIF and PDF Images

    • From the WorkFlow User Console, create a work item and use Capture - File pickup to add one or more TIF images.
    • From the WorkFlow User Console, create a work item and use Capture - File pickup to add one or more PDF images.
    • Confirm that both types of images can be processed via automatic redaction.
    • Confirm that both types of images can be processed via verification before redaction.

  • Scenario: Hybrid Processing

    • Configure an ID Shield Server test machine that can also be used as a NetDMS Client.
    • Prepare an FPS file to queue files and run rules.
    • Prepare an FPS file to automatically apply redactions to HCData items.
    • Prepare an FPS file to verify images that contain one or more of MCData, LCData or Clue items.
    • From the WorkFlow User Console, create several suitable work items.
      • One or more work items that contain only HCData items.
      • One or more work items that contain only MCData, LCData or Clue items.
      • One or more work items that contain non-HCData false positives.
      • One or more work items that do not contain any sensitive data.
    • Confirm that each image is queued and processed as expected.
    • Confirm that an image containing only HCData is processed automatically as expected.
    • Confirm that images containing other data are presented for verification.
    • Confirm that post-verification processing is handled as expected.
    • Confirm that each work item advances through NetDMS as expected.

  • Scenario: ID Shield As Automatic Node

    • Modify the configuration of the Extract node to be an automatic node.
    • Confirm that processing of images for automatic redaction works as expected.
    • Confirm that processing of images for verification before redaction works as expected.

  • Scenario: Processing Parcels Containing Multiple Image Files

    • Configure an ID Shield Server test machine that can also be used as a NetDMS Client.
    • Prepare an FPS file to queue files and run rules.
    • Prepare an FPS file to automatically apply redactions to sensitive items.
    • From the WorkFlow User Console, create several work items that each contain multiple image files.
    • Process a work item and confirm that each image is queued and processed as expected.

    • Create another FPS file to verify each sensitive work item.
    • Process a work item and confirm that the first sensitive image is presented for verification.
    • Confirm that redactions are applied to the verified image even before other images in a particular parcel have completed verification.
    • Confirm that the NetDMS: Advance work item task succeeds for the verified image even before other images in a particular parcel have completed verification.
    • Confirm that the Current Node Title remains unchanged while other images in a particular parcel have not completed verification.
    • Complete verification for the additional images in the parcel.
    • Confirm that redactions are applied correctly.
    • Confirm that the NetDMS: Advance work item task succeeds.
    • Confirm that the Current Node Title for the parcel advances to the subsequent node.

  • Scenario: Three Machine Processing

    • Configure a test environment with three distinct test machines, one for each of the following functions.
      • NetDMS Server
      • ID Shield Server
      • NetDMS Client
    • Prepare an FPS file to support automatic processing.
      • Queue files and run rules
      • Apply redactions
      • Advance the work item through NetDMS
    • From the WorkFlow User Console, create a work item and use Capture - File pickup to add a TIF image.
    • Confirm that the image is queued and processed as expected.
    • Confirm that redactions are applied as expected.
    • Confirm that the work item advances through NetDMS as expected.

    • Modify the ID Shield workflow to include verification of sensitive images.
    • Confirm that image are queued and processed as expected.
    • Confirm that images are presented for verification as expected.
    • Confirm that redactions are applied as expected.
    • Confirm that the work item advances through NetDMS as expected.
