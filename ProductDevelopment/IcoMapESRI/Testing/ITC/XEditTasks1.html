Cut Polygon Features

  1. Click Add Data button on ArcMap Toolbar. Select TestDB1, select both Line and Polygon and click Add.

  2. Select 'Editor > Start Editing'. Select 'Task > Cut Polygon Features'.

  3. Use ArcMap 'Editor Tool' to select the polygon feature in the drawing.

    The polygon feature should be selected.

  4. Click the 'IcoMap' button

    The IcoMap dialog should appear. The highlight frame around IcoMap dialog should appear yellow.

  5. At IcoMap command line, type in '3468,6853' as the starting point, 'W' as the bearing, '23456' as the distance. Finish the sketch.

    The select polygon should be divided into two parts.

  6. Stop editing without saving the changes.
