Copy paste append

  1. Click on the "New Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Open the file Test004.txt from the 'Highlighted Text' window. Make sure the 'Highlight finder' has 'Mathematical Content' selected.

    The 'Highlighted Text' window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  3. Click the 'Copy' button in the 'Highlighted Text' window.

    All of the text should have been copied to the clipboard.

  4. Open Notepad and paste the text that you copied. Compare this file with what the 'Highlighted Text' window displays.

    The text in both files should be the same.

  5. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.

  6. Click on the "New Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  7. Close the 'Open' dialog box.

    The 'Open' dialog box goes away.

  8. Click the 'Paste' button in the 'Highlighted Text' window.

    The same text you copied into Notepad should have pasted into the 'Highlighted Text' window.

  9. Add the following lines to the beginning of the text in notepad:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'

    Copy all of the text in Notepad to the clipboard.

    The text contained in Notepad should have been copied to the clipboard

  10. Click the 'Append' button in the 'Highlighted Text' window.

    The copied text should append to the end of the open file in the 'Highlighted Text' window. All of the mathematical text including the appended text should be highlighted yellow.

  11. Click the 'Save' button.

    A 'Save' dialog box should appear.

  12. Name the file Test004b.txt and click the 'Save' button.

    IcoMap should have closed the save dialog and the file should have been saved.

  13. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.

  14. Click on the "New Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  15. Select Test004b.txt and click the 'open' button.

    The saved file should contain both the original text and the appended text.

  16. Go the open file in Notepad and copy the following lines of text to the clipboard:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'
    Click the 'Paste' button in the 'Highlighted Text' window.

    All of the existing text in the 'Highlighted Text' window should have been replaced with the lines:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'
    All of the mathematical text should be highlighted yellow

  17. Click the 'Close' (X) button in the 'Highlighted Text' window.

    A warning message should get displayed telling the user that the current file has not yet been saved. They should then be asked if they want to save it.

  18. Click the 'Yes' button.

    The 'Save' dialog box should appear.

  19. Select Test004b.txt and click the 'save' button.

    The file Test004b.txt should get saved.

  20. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.

  21. Click on the "New Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  22. Select Test004b.txt and click the 'open' button.

    The saved file should only contain:
    North 52d47'28" East; 101.52'
    South 01d15'14" West; 144.54'
    All of the mathematical text should be highlighted yellow

  23. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.
