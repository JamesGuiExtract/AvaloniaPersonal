Highlight finder

  1. Click on the "Highlighted Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Open the file LegalText8.txt from the 'Highlighted Text' window. Make sure the 'Highlight...' selection is 'Mathematical Content'.

    The 'Highlighted Text' window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  3. Change the 'Highlight...' selection to 'All Words'.

    All of the text should be highlighted using spaces as the delimiter.

  4. Change the 'Highlight...' selection to 'Nothing'.

    None of the text is highlighted.

  5. Change the 'Highlight...' selection to 'Mathmatical Content'.

    Only the mathmatical text should be highlighted, i.e. bearings, distances and angles.

  6. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.
