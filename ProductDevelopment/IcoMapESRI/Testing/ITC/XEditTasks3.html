Modify Features

  • Part I -- Modify Polygon Feature

    1. Click Add Data button on ArcMap Toolbar. Select TestDB1, select both Line and Polygon and click Add.

    2. Select 'Editor > Start Editing'. Select 'Task > Modify Feature'.

    3. Use ArcMap 'Editor Tool' to select the polygon feature in the drawing.

      The polygon feature should be selected and all of its vertices should be shown in solid squares.

    4. Click the 'IcoMap' button

      The IcoMap dialog should appear. The highlight frame around IcoMap dialog should appear yellow.

    5. At IcoMap command line, type in 'W' as the bearing, '12345' as the distance. Finish the sketch.

      The selected polygon now should have one more vertex added.


  • Part II -- Modify Line Feature

    1. Use ArcMap 'Editor Tool' to select the line feature in the drawing.

      The polygon feature should be selected and all of its vertices should be shown in solid squares.

    2. Click the 'IcoMap' button

      The IcoMap dialog should appear. The highlight frame around IcoMap dialog should appear yellow.

    3. At IcoMap command line, type in 'S' as the bearing, '12345' as the distance. Finish the sketch.

      The selected line now should have one more segment added.

    4. Stop editing without saving the changes.
