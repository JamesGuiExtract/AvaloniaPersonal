Extend/Trim Features

  1. Click Add Data button on ArcMap Toolbar. Select TestDB1, select both Line and Polygon and click Add.

  2. Select 'Editor > Start Editing'. Select 'Task > Extend/Trim Features'.

  3. Use ArcMap 'Editor Tool' to select the line feature in the drawing.

    The line feature should be selected.

  4. Click the 'IcoMap' button

    The IcoMap dialog should appear. The highlight frame around IcoMap dialog should appear yellow.

  5. At IcoMap command line, type in '-2413.78,-2063.64' as the starting point, 'N' as the bearing, '12345' as the distance. Finish the sketch.

    The part of the line that was on the right side of the drawing line should be trimmed off.

  6. Stop editing without saving the changes.
