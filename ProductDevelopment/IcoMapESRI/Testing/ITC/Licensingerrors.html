Licensing Errors

NODE LOCKED

Test setup:
  • Install IcoMap.
  • Open the IcoMap License Utility.
  • Select Node Locked licensing mode.
  • Close the IcoMapLicense Utility.

    • There should be 1 .esl file in the C:\Program Files\Extract Systems\CommonComponents folder: ExtractSystems_IcoMap.esl.
  • Start ArcGIS.

    1. Click the IcoMap button.

      An exception dialog box should appear that says: "IcoMap component is not licensed.". [P10: #3111] Confirm that the ArcMap Arrow tool is selected.

    2. Click the Highlighted Text Window button.

      An exception dialog box should appear that says: "IcoMap component is not licensed.".

    3. Click Spot Recognition Window button

      An exception dialog box should appear that says: "IcoMap component is not licensed.".

    4. Open the Extensions dialog from the Tools menu. Click the Extract Systems IcoMap item.

      An Extensions error message should appear that says: "There is no valid IcoMap license file.".

    CONCURRENT

    Test setup:
  • Install Extract Systems LM on the computer that will have the USB Key attached. Do not attach the key.
  • Install IcoMap.
  • Open the IcoMap License utility.
  • Select Concurrent licensing mode.
  • Set LM Server to the name of the computer that will host the USB Key.
  • Close the IcoMap License utility.

    • There should be 1 .esl file in the C:\Program Files\Extract Systems\CommonComponents folder: ExtractSystems_IcoMap.esl.
  • Start ArcGIS on the test computer.

    1. Click the IcoMap button.

      An exception dialog box should appear that says: "Unable to obtain USB Key license.".

    2. Click the Highlighted Text Window button.

      An exception dialog box should appear that says: "Unable to obtain USB Key license.".

    3. Click the Spot Recognition Window button.

      An exception dialog box should appear that says: "Unable to obtain USB Key license.".

    Test setup 2:
  • Install Extract Systems LM on computer that will have the USB Key attached (Server).
  • Attach key with 1 IcoMap license to Server.
  • Install IcoMap on 2 computers.
  • On both computers, Open the IcoMap License Utility.
  • Select Concurrent licensing mode.
  • Set LM Server to the name of the computer that will have the USB Key.
  • Close the IcoMapLicense Utility.
  • Start ArcGIS on both computers.
    •

    1. On Computer 1, click the IcoMap button.

      IcoMap should open without errors.

    2. On Computer 2, click the IcoMap button.

      An exception dialog box should appear that says: "Unable to obtain USB Key license.".

    3. On Computer 2, click the Highlighted Text Window button.

      An exception dialog box should appear that says: "Unable to obtain USB Key license.".

    4. On Computer 2, click the Spot Recognition Window button.

      An exception dialog box should appear that says: "Unable to obtain USB Key license.".
