Drawing a custom curve

  1. Open the 'Curve Genie' tool.

    The 'Curve Genie' dialog box should appear.

  2. Set the 3 parameters to Tangent in bearing, Radius and Tangent out bearing. Click the 'Draw' button.

    The 'Curve Genie' dialog box should disappear and IcoMap should prompt for a start point.

  3. Select a start point

    IcoMap should prompt for the Tangent in bearing, Radius and Tangent out bearing

  4. Draw the curve using the following parameters:
    Tangent in bearing - n72d14'22"e
    Radius - 127.85
    Tangent out bearing - N31d00'26"W
    Make it concave to the left.

    The curve should get drawn in ArcGIS.

  5. Click the 'Finish Sketch' button.

    The curve should get written to the database. The length should be ~230.38.
