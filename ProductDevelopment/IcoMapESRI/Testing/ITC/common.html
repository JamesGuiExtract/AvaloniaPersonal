To start up IcoMap, you will need to open ArcMap, have a layer containing either a polyline or polygon feature class and you will need to start editing.

To create an Annotation layer...

  1. Open ArcMap and select a new empty database.
  2. Open ArcCatalog and create a new Personal Geodatabase.
  3. Right-click in the Contents tab and select New Feature Class. Porvide Name and select Annotation Features as Type. Click Next.
  4. Click Import... and select a Line or Polygon feature from an existing personal geodatabase.
  5. Use the default coordinate range and click Next.
  6. Use 1:1,000 as the Reference Scale with Map Units of Feet. Click Next.
  7. Use the default values for the Annotation classes. Click Next.
  8. Use the defaults for Field Names and Data Types. Click Finish.
