Deleting highlights

  1. Click on the 'Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Open the file Test010.tif.gdd.

    IcoMap opens the associated image along with the highlights.

  3. Click the 'Delete Highlights' button

    IcoMap should prompt you to select the highlights you want to delete

  4. Window half of the highlights.

    As soon as you complete the window, all of the highlights touching and contained within the window should disappear.

  5. Click on the 'Open' button.

    The user should be prompted to save changes before the 'Open' dialog appears.

  6. Right click on Test010.tif.gdd.

    A floating menu should appear.

  7. Click on 'Properties'

    The 'Properties' dialog box will appear

  8. Go to the 'General' tab and uncheck the checkbox to the left of 'Read-only'. Click the 'OK' button.

    The file should not be read-only now.

  9. Cancel out of the 'Open' dialog box.

    The 'Open' dialog should disappear.

  10. Click the 'Save' button.

    The 'Save' dialog box should appear with the file name defaulted to Test010.tif.gdd.

  11. Click the 'Save' button.

    You should get indication that the file is read only and a 'Save' dialog box should appear allowing you to save the file to a different name.

  12. Name the file Test010b.tif.gdd and click the 'Save' button.

    The file Test010b.tif.gdd should get saved and the 'Save' dialog box should close.

  13. Click on the 'Open' button.

    The 'open' dialog should appear.

  14. Select the file Test010b.tif.gdd and click the 'open' button.

    IcoMap opens the associated image along with the highlights. Make sure that the highlights that were deleted did not get saved.

  15. Click the 'Close' (X) button in the Spot Recognition window.

    IcoMap should close the Spot Recognition window.
