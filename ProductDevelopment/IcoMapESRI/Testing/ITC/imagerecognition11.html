OCR and Image Rotation in Spot Recognition Window

  1. Configure IcoMap for Concurrent Licensing. Start ArcMap.

  2. Click on the "Spot Recognition Window" button in the IcoMap dialog.

    IcoMap opens a new Spot Recognition Window and brings up the 'Open' dialog box right away.

  3. [P10 #3125] Select and open a large TIF document from I:\Common\Testing\Product Testing\IcoMap_ArcGIS\Rotation.

    The Spot Recognition window opens the image.

  4. Create a subimage without first correcting the overall image rotation.

    IcoMap opens a new Spot Recognition Window and displays the desired image region with the same orientation as in the original image.

  5. Recognize the text in the subimage via Process Entire Image.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window.

  6. Recognize the text in the subimage via Process Rectangular Area and select a subregion.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window and the SR Window containing the subimage.

  7. Rotate the main image so that the text is right-side up. Create a subimage.

    IcoMap opens a new Spot Recognition Window and displays the desired image region with the same rotated orientation.

  8. Recognize the text in the subimage via Process Entire Image.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window.

  9. Recognize the text in the subimage via Process Rectangular Area and select a subregion.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window and the SR Window containing the subimage.

  10. [P10 #3125] Select and open sedella.pdf from I:\Common\Testing\Product Testing\IcoMap_ArcGIS\Rotation.

    The Spot Recognition window opens the image.

  11. Create a subimage without first correcting the overall image rotation.

    IcoMap opens a new Spot Recognition Window and displays the desired image region with the same orientation as in the original image.

  12. Recognize the text in the subimage via Process Entire Image.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window.

  13. Recognize some text in the subimage via Process Rectangular Area and selection of a subregion.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window and the SR Window containing the subimage.

  14. Rotate the main image so that the text is no longer right-side up. Create a subimage.

    IcoMap opens a new Spot Recognition Window and displays the desired image region with the same rotated orientation.

  15. Recognize the text in the subimage via Process Current Page.

    IcoMap opens a new Highlighted Text Window containing the recognized text.

    Close the HT Window and the SR Window containing the subimage.

  16. [P10 #3082] Select a start point so that IcoMap is ready for a bearing. Open sedella.pdf from I:\Common\Testing\Product Testing\IcoMap_ArcGIS\Rotation if it is not still open.

    The Spot Recognition window opens the image.

  17. Swipe a bearing from the open image.

    Either the bearing will be recognized correctly or the Input Correction dialog will display text that is very close to the actual swiped text.

  18. Click the 'Close' (X) button in the Spot Recognition window.

    IcoMap should close the Spot Recognition window.
