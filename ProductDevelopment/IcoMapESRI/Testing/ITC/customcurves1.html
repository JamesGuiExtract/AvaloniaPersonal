Customizing a curve tool

  1. Open the 'Curve Genie' tool.

    The 'Curve Genie' dialog box should appear.

  2. Set the 3 parameters to Radial in bearing, Delta angle and Radius. Click the 'Save As…' button.

    The 'Save As Curve' dialog box should appear.

  3. Select the radio button next to curve 8 and click 'OK'.

    The parameters defined in step 2 should get saved to the curve definition for curve #8 in the registry. The 'Save As Curve' dialog should disappear.

  4. Click the 'Close' button.

    The 'Curve Genie' dialog should disappear.

  5. Draw a curve using curve 8 on a line layer. Use the following parameters:
    Radial in bearing - n72d14'22"e
    Delta - 76d45'12"
    Radius - 127.85
    Make it concave to the right.

    The curve should get drawn in ArcGIS.
    NOTE: The tooltip and the text next to the button should say "<Radial in bearing><Delta><Radius>"

  6. Click the 'Finish Sketch' button.

    The curve should get written to the database. The length should be ~171.27'
