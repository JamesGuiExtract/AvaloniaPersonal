Basic Concurrent Licensing

Test setup:

  • Install Extract Systems LM on computer that will have the USB Key attached (Server).
  • Attach key with 1 IcoMap license to Server.
  • Install IcoMap.
  • Select Concurrent licensing mode from the IcoMap License Utility.
  • Set LM Server to the name of the computer that will has the USB Key.
  • Close the IcoMap License Utility.

    • There should be only 1 .lic file in the C:\Program Files\Extract Systems\CommonComponents folder: ExtractSystems_IcoMap.lic.
  • Start ArcGIS.
    •

    1. Click on the IcoMap button.

      IcoMap should start without errors.

    2. Click on the Highlighted Text Window button.

      The Highlighted Text Window should open without error.

    3. Click on the Spot Recognition Window button.

      The Spot Recognition Window should open without error.

    4. Open the IcoMap License Utility and click the Server Status button.

      One license should be shown as in use.

    5. Close ArcMap, open the IcoMap License Utility and click the Server Status button.

      No license should be in use.

    The location of the files and any common instructions can be found here.
