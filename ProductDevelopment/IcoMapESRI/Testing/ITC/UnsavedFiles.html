Linked documents and unsaved files

  1. Go to the 'General' tab in the 'IcoMap Options' dialog box and check the option to enable automatic source document linking. Click the 'OK' button to accept the change.

    The option will be enabled and the 'IcoMap Options' dialog box will disappear.

  2. Click on the 'New Image Recognition Window' button.

    IcoMap opens a new 'Image Recognition' dialog and brings up the 'Open' dialog box right away.

  3. Select the file Unsaved.tif and click the 'open' button.

    IcoMap opens the associated image.

  4. Use the 'Recognize Text in Window' tool and draw a window around the legal description.

    The area you windowed should get recognized and output to a new 'Legal Text' window.

  5. Draw the parcel described in the text window

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment. The highlights that are selected will also change to red.

  6. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.

  7. Click on the ArcMap 'Identify' button then select the parcel you just created.

    ArcMap's 'Identify Results' dialog box should appear

  8. Right click on the feature in the left hand window in the 'Identify Results' dialog box.

    There should be a 'Hyperlinks' option in the floating menu. You will notice that the .tif image is listed and not the text file you drew the parcel from. This is because the text file was not saved. This is a change in IcoMap. Now instead of telling the user that the document won't be linked to the feature unless the file is saved it records the parent file (the image name in this case) if available.

  9. Click the 'Close' (X) button in the 'Legal Text' window without saving the file.

    IcoMap should close the 'Legal Text' window.

  10. Click the 'Close' (X) button in the 'Image Recognition' window.

    IcoMap should close the 'Image Recognition' window.
