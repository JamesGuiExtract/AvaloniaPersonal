Enter for last value

  1. Make a polyline layer current and click on the 'IcoMap Line' tool.

    IcoMap should prompt for a start point.

  2. Select a start point.For the first bearing, type n45e and for the first distance, type 100.

    The first line should get drawn and IcoMap should prompt for a bearing again.

  3. Press the 'Enter' key twice (once for the bearing and once for the distance).

    A second line should get drawn.

  4. Press the 'Enter' key four more times.

    Two more lines should get drawn.

  5. Finish the sketch.

    The sketch should get added to the database. The length of the sketch should be 400' long (provided that the units for the feature class are in feet).
