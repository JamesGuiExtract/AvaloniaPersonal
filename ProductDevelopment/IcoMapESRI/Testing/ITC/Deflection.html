These test cases test drawing using deflection angles.

  1. Click on the "Spot Recognition Window" button in the IcoMap dialog.

    IcoMap opens a new 'Spot Recognition' window and brings up the 'Open' dialog box right away.

  2. Open the file DeflectionAngle.tif from the 'Spot Recognition' window.

    The 'Spot Recognition' window displays the image you selected

  3. Go to the 'Direction/Angle Definition' tab in the 'Options' dialog box.

  4. Select the 'Bearing...' option in the 'Specify Direction...' group box.

    The 'Polar Angle...' and 'Azimuth...' options should be disabled.

  5. Select the 'Deflection Angle' option in the 'Specify Angle...' group box and click the 'OK' button.

    The 'IcoMap Options' dialog box should go away. This will change the direction type that IcoMap will be looking for and prompt for deflection angles when you choose the 'internal/deflection angle' line tool.

  6. Draw "PARCEL II" described in DeflectionAngle.txt using whatever means you desire.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment. The highlights that are selected will also change to red.

  7. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~22829.53
    Perimeter = ~727.63

  8. Click the 'Close' (X) button in the 'Spot Recognition' window.

    IcoMap should close the 'Spot Recognition' window.
