This test case tests the various functionalities associated with the Image Recognition dialog.

  1. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Open the image Test008.tif. IcoMap opens the specified file.

  3. Set the highlighter size using the 'Set Highlight Height' tool. Using the bearing "S11°26'38" E" for a reference, the first point you pick with the left mouse button should be just above the "S". While holding the mouse button down, drag the mouse to just below the "S" and then let go of it.

    As you use spot recognition to draw, you should see that the height of the highlight is the same as the distance you picked.

  4. Draw the parcel using spot recognition. Use COGO only if the values are not recognizing.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  5. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~106506.71
    Perimeter = ~1557.39

  6. Click the 'Close' (X) button in the Spot Recognition window.

    IcoMap should close the Spot Recognition window.
