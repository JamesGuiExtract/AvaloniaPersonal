IcoMap - Enhancements and Bug Fixes

  1. Click on the "Set IcoMap options" button in the IcoMap dialog.

    IcoMap opens the IcoMap Options window with the General tab active.

  2. [P10 #3041] Modify the default distance unit. Press the Enter key.

    The IcoMap Options window closes without error and the modification is applied.

  3. [P10 #3041] Reopen the IcoMap Options window and modify the default distance unit again. Press the Esc key.

    The IcoMap Options window closes without error and the modification is ignored.

  4. Reopen the IcoMap Options window and restore the original the default distance unit again. Click the OK button.

    The IcoMap Options window closes without error and the original setting is restored.

  5. [P10 #3102] Test the fix by exercising the provided steps using the image attached to the SCR.

    Confirm that the correct bearing is provided to the Dynamic Input Grid.

  6. [P10 #3078, #3079] Follow the instructions linked below to add an Annotation layer. Start ArcMap with a new, empty map. Click the Add Data button and select the Annotation, Line, and Polygon layers. Start IcoMap and start editing.

    Confirm that with Annotation as the Target layer, IcoMap is disabled.

    Confirm that with Line or Polygon as the Target layer, IcoMap is enabled.

  7. [P10 #3077] Follow the instructions linked below to add an Annotation layer. Start ArcMap with a new, empty map. Click the Add Data button and select the Annotation, Line, and Polygon layers. Start editing and use the ArcMap sketch tool to draw a simple line or polygon feature. Add an Annotation to the feature.

    Start IcoMap and confirm that no IcoMap errors are displayed.

  8. Close ArcMap.
