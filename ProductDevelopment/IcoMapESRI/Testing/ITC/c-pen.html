C-Pen Functionality

  1. Print out the image c-pen.tif.
    If you already have a copy printed that's fine too. It's best printed on an 11x17 sheet of paper but the text you need to swipe should fit fine on 8.5x11.

  2. Draw the parcel using the c-pen. Use COGO only if the values are not recognizing.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  3. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~106506.71
    Perimeter = ~1557.39
