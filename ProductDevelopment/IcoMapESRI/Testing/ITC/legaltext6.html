Hyperlinks and select features

  1. Click on the 'New Text Window' button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Open the file Test006.txt from the 'Highlighted Text' window. Make sure the 'Highlight finder' has 'Mathematical Content' selected.

    The 'Highlighted Text' window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  3. Go to the 'General' tab in the 'IcoMap Options' dialog box and check the option to enable automatic source document linking. Click the 'OK' button to accept the change.

    The option will be enabled and the 'IcoMap Options' dialog box will disappear.

  4. Use the IcoMap drawing tools to draw out the legal description using MCR.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment. The highlights that are selected will also change to red.

  5. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~288950.79'
    Perimeter = ~2316.53'

  6. Click on the ArcMap 'Identify' button then select the parcel you just created.

    ArcMap's 'Identify Results' dialog box should appear

  7. Right click on the feature in the left hand window in the 'Identify Results' dialog box.

    There should be a 'Hyperlinks' option in the floating menu and the file that you drew the parcel from should be listed.

  8. Repeat steps 4 and 5

  9. Click on the 'Select Features' button.

    The 'Select Features' dialog box will appear.

  10. Browse to the file Test006.txt and select it.

    The path and filename should appear in the edit box.

  11. Click the 'OK' button.

    Both of the drawings you created should get selected.

  12. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window
