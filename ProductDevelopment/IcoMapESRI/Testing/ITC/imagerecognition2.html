Editing highlights

  1. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Select the file Test009.tif.gdd and click the 'open' button.

    IcoMap opens the associated image along with the highlights.

  3. Click the 'Edit Highlights' button.

    IcoMap will prompt you to select a highlight.

  4. Click on the highlight for the bearing "N46°49'21"E".

    The 'Edit Highlights' dialog will appear with the text associated with the highlight.

  5. Change the text to "ICHANGEDIT" and click the 'OK' button.

    The text associated with the highlight should have changed.

  6. Click on the same highlight again to see if the associated text is the way you changed it.

    The 'Edit Highlights' dialog will appear with the text associated with the highlight.

  7. Click the 'Close' (X) button in the Spot Recognition window.

    A warning message should appear telling the user that changes have been made. Do you want to save the changes?

  8. Click the 'No' button.

    IcoMap should close the Spot Recognition window.

  9. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  10. Select the file Test009.tif.gdd and click the 'open' button.

    IcoMap opens the associated image along with the highlights.

  11. Click the 'Edit Highlights' button.

    IcoMap will prompt you to select a highlight.

  12. Click on the highlight that was changed in step 5. The change should not have been saved. The value should still be "N46°49'21"E" and not ICHANGEDIT.

  13. Use the 'Edit Highlights' tool to change the text associated with some of the highlights to ICHANGEDIT.

    The text associated to the highlights should be ICHANGEDIT.

  14. Click the 'Close' (X) button in the Spot Recognition window.

    A warning message should appear telling the user that changes have been made. Do you want to save the changes?

  15. Click the 'Yes' button.

    The 'Save' dialog should appear.

  16. Save the file as Test009b.tif.gdd. The file should have been saved. And the Spot Recognition window should have closed.

  17. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  18. Select the file Test009b.tif.gdd and click the 'open' button.

    IcoMap opens the associated image along with the highlights. Check the strings that you edited to make sure that the changes were saved.

  19. Click the 'Close' (X) button in the Spot Recognition window.

    IcoMap should close the Spot Recognition window.
