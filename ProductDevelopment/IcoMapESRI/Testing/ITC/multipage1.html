Multi-page tif basics

  1. Click on the 'Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' window and brings up the 'Open' dialog box right away.

  2. Select the file Test018.tif and click the 'open' button.

    IcoMap opens the associated image to the first page. The 'move backwards' button should be disabled. The page number indicator should be 1

  3. Click the 'move forward' button to cycle forward through the pages until you get to the last page.

    While cycling through the pages, the page number indicator should display the correct page number and the correct image should be displayed. When the last page is displayed the 'move forward' button should be disabled and page #6 should be displayed. The 'move backwards' button should only be disabled on the first page.

  4. Click the 'Close' (X) button in the Spot Recognition window.

    IcoMap should close the Spot Recognition window.
