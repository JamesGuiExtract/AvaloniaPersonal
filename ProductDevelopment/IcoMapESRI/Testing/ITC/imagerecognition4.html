Recognize text in a window

  1. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Select the file Test011.tif and click the 'open' button.

    IcoMap opens the associated image.

  3. Click the Recognize text and process button. Select 'Send text to Highlighted Text Window (Process RECTANGULAR area)' from the pop-up menu.

    The mouse cursor should change into a cross with a blue note-like shape.

  4. Draw a window around some text area.

    The area you windowed should get recognized and output to a new 'Highlighted Text' window.

  5. Click the 'Close' (X) button in the 'Highlighted Text' window without saving the file.

    IcoMap should close the 'Highlighted Text' window.

  6. Click the 'Close' (X) button in the 'Spot Recognition' window.

    IcoMap should close the 'Spot Recognition' window.
