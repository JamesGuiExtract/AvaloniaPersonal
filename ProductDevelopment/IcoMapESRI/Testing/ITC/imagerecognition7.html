Rotate Image

  1. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Open the image ImageRecognitionImage7.tif.

    IcoMap opens the specified file.

  3. Click the 'Rotate image right' button.

    The image should appear upside down.

  4. Click the 'Rotate image left' button.

    The image should appear correctly now.

  5. Draw the parcel using the spot recognition tool.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  6. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected. Area = ~??????? Perimeter = ~???????

  7. Click the 'Save' button.

    The 'Save' dialog box should appear with the file name defaulted to ImageRecognitionImage7.tif.gdd.

  8. Click the 'Save' button

    The file should have saved

  9. Close the 'Spot recognition window' and re-open it with the image ImageRecognitionImage7.tif.

    The image should appear rotated to the right.

  10. Close the 'Spot recognition window' and re-open it with the .gdd file ImageRecognitionImage7.tif.gdd.

    The image should appear rotated correctly and the hightlights should appear in the correct locations.

  11. Close the 'Spot recognition window'.
