Spot Recognition Window - Enhancements and Bug Fixes

  1. Configure IcoMap for Concurrent Licensing. Start ArcMap.

  2. Click on the "Spot Recognition Window" button in the IcoMap dialog.

    IcoMap opens a new Spot Recognition Window and brings up the 'Open' dialog box right away.

  3. [P10 #3014] Select and open a PDF document. Select and OCR a region of the image.

    The Spot Recognition window opens the PDF file and the appropriate text is displayed in a new Highlighted Text Window.

  4. [P10 #3112] Select and open a TIF document. Zoom in very close to some text in the image.

    Confirm that anti-aliasing is enabled.

  5. Click the 'Close' (X) button in the Spot Recognition window.

    IcoMap should close the Spot Recognition window.
