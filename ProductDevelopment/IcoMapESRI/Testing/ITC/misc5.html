Command Shortcuts

  1. Open the 'IcoMap Options' and go to the 'Command Shortcuts' tab. Change the defaults to whatever you want and click the 'OK' button.

    All of the command shortcuts that you changed should be usable. Use the IcoMap commands with the shortcuts you defined.

  2. Open the 'IcoMap Options' and go to the 'Command Shortcuts' tab. Click the 'Restore Defaults' button.

    All of the shortcuts should have changed back to the original defaults. You can see a list by going here.

  3. Click the 'OK' button.

    Now when you use the command shortcuts, the original defaults should work correctly.

The location of the files and any common instructions can be found here.
