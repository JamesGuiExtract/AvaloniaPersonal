Crop image into a new window

  1. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Open the image ImageRecognitionImage8.tif.

    IcoMap opens the specified file.

  3. In the upper left hand corner of the image there are two tables. One curve table and one line table. Create a new spot recognition for each of them using the 'Crop image and display...' tool.

    You should have two more spot recognition windows. One of them for the curve table and one for the line table.

  4. Draw the map using spot recognition. You should need to use all three spot recognition windows while drawing the map.

    You should have linework drawn in the ArcGIS environment and there should be highlights in all three windows.

  5. Close the spot recognition windows. Don't bother saving the .gdd files because there isn't much use as the highlights are spread among the spot recognition windows.
