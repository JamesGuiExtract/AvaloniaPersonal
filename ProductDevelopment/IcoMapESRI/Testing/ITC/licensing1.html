Node Locked Licensing

Test setup:

  • Install IcoMap.
  • Open the IcoMap License Utility.
  • Select Node Locked licensing mode.
  • Close the IcoMap License Utility.

    • There should be 1 .ESL file in the C:\Program Files\Extract Systems\CommonComponents folder: ExtractSystems_IcoMap.esl.
  • Start ArcGIS.
    •

    1. Using the COM License Generator utility, create a Fully Licensed license file for IcoMap locked to the testing PC. Copy this license file to the Common Components folder.

    2. Start up IcoMap.

      No licensing error should be seen.

    3. Open a Spot Recognition Window.

      No licensing error should be seen.

    4. Open a Highlighted Text Window.

      No licensing error should be seen.

    5. Remove or rename all license files (*.LIC) located in the Common Components folder. Start up IcoMap.

      A licensing error regarding IcoMap should be seen.

    6. Create a license file for IcoMap 4.0. Start up IcoMap.

      Since IcoMap 4.0 was the last version that did not support product-level licensing, a licensing error regarding IcoMap should be seen.

    7. Replace or rename any modified license files. Start up IcoMap.

      No licensing error should be seen.

    8. Open the Extensions dialog from the Tools menu. Click the Extract Systems IcoMap item to attempt unchecking the checkbox.

      An Extensions error message should appear that says: "The extension could not be deactivated because it is currently in use.".

      Confirm that the extension remains checked.

    The location of the files and any common instructions can be found here.
