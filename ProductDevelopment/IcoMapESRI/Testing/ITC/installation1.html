Typical Installation

  1. Note the Windows operating system and the version of ArcGIS being tested.

  2. Run setup.exe for IcoMap for ArcGIS

    Confirm that the splash screen displays the proper IcoMap version number and copyright date. The 'Welcome' window will appear.

  3. Click the Next button.

    A dialog box containing the license agreement should appear with proper version information and proper company contact information.

  4. Select "I accept the terms of the license agreement" and click the Next button.

    The window for choosing the product location should appear.

  5. Use the default location and click the Next button.

    The 'Setup Type' window should appear with the options 'Typical', 'Compact' and 'Custom'.

  6. Click the Typical radio button and click the Next button.

    The program will go through the installation. The entire product will be installed including the program files, help and sample files.

  7. Confirm that the Node Locked radio button is selected in the IcoMap License Utility.

    Click OK.

  8. Click the Finish button.

    This will exit the installation.

  9. Do not create a license file.

    Start ArcGIS and confirm that a meaningful error appears for each of the Extract Systems Tools toolbar buttons: Highlighted Text Window, Spot Recognition Window, and IcoMap Window.

  10. Create a suitable license file for the test machine.

    Confirm that each of the Extract Systems Tools toolbar buttons works without error: Highlighted Text Window, Spot Recognition Window, and IcoMap Window.

  11. To exercise basic SRW licensing functionality:
    • Open a TIF image and confirm that anti-aliasing is functional.
    • Recognize text in a portion of the TIF image and confirm that appropriate text is displayed in a Highlighted Text Window.
    • Open a PDF document without error.
    • Recognize text in a portion of the PDF image and confirm that appropriate text is displayed in a Highlighted Text Window.

  12. Delete or rename the license file. Run the IcoMap License Utility application and choose Concurrent. Choose an appropriate LM Server and confirm that the Server Status button updates the User Limit.

    Confirm that each of the Extract Systems Tools toolbar buttons works without error: Highlighted Text Window, Spot Recognition Window, and IcoMap Window.

  13. To exercise basic HTW licensing functionality:
    • Open a text file and confirm that each input finder works correctly.
    • Start editing and draw a simple parcel by clicking on highlighted bearings, distances, and curve parameters within a text file.

  14. To exercise swiping functionality, parcel creation, and Dynamic Input Grid ( DIG ) editing:
    • Open the saybrook_all_new.tif image from the Spot Recognition Window and swipe bearings, distances, and curve parameters to create one of the parcels.
    • Modify one or more parameters via the DIG.
    • Confirm that DIG functionality with respect to in-place editing, cell highlighting, and navigation is as expected.
