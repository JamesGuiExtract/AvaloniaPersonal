IcoMap Grid Tool and Query Feature

  1. To install the Grid Generator:
    • Copy GridGenerator.dll and GridGenerator.ini from the GridGenerator folder ( parallel to IcoMapForArcGIS ) to the CommonComponents folder.
    • Open ArcMap and select Customize... from the Tools menu.
    • Click Add from file... and browse to GridGenerator.dll.
    • Note DrawGrid and QueryFeature as Added Objects.
    • Click the Commands tab and select the Extract Tools category.
    • Individually drag the DrawGrid and QueryFeature items to the Extract Systems Tools toolbar.
    • Close the Customize... tool.

  2. Testing Grid Generator licensing:
    • Create an IcoMap license file that does not include Grid Generator or Query Feature Tool.
    • Open ArcMap.
    • Confirm that both new tools are disabled on the Extract Systems Tools toolbar.
    • Confirm that both new tools include (Not Licensed) within their tooltips.
    • Close ArcMap.
    • Create an IcoMap license file that includes both Grid Generator and Query Feature Tool.
    • Open ArcMap.
    • Confirm that both new tools are still disabled on the Extract Systems Tools toolbar.
    • Confirm that both new tools do not include (Not Licensed) within their tooltips.

  3. Testing Grid Generator operation:
    • Open ArcMap.
    • Click the Add Data button.
    • Navigate to and select the TestGrid.mdb database, selecting all of the features.
    • Edit GridGenerator.ini so that the Layer names match those just added from TestGrid.mdb.
    • Close GridGenerator.ini and start editing.
    • Select the Sections layer and the ArcMap Sketch tool.
    • Draw a shape with four sides and finish the sketch.
    • Select the ArcMap Edit tool and the newly created feature.
    • Click on the IcoMap Draw Grid tool.
    • Fill in County Code, Section, Township, and Range with numbers. Township Direction and Range Direction can be modified if desired.
    • Click the Create Features button.
    • Confirm that squares are drawn for all layers.

  4. Testing Query Feature operation with Quarter_Sections:
    • Uncheck all layers except for Quarter_Sections.
    • Use the ArcMap Edit tool to select each of the four quadrants.
    • Click the ArcMap Attributes button and confirm that the Quarter values are: NE = 1, NW = 2, SE = 3, SW = 4.
    • Click on the IcoMap Query Feature tool.
    • Fill in County Code, Section, Township, and Range with the same numbers used before. Select the Quarter_Sections layer.
    • Test each of { 1, 2, 3, 4 } for Quarter_Section and click the Query button.
    • Confirm that the appropriate square is selected for each Quarter_Section.

  5. Testing Query Feature operation with Quarter_Quarter_Sections:
    • Uncheck all layers except for Quarter_Quarter_Sections.
    • Use the ArcMap Edit tool to select one of the sub-quadrants.
    • Click the ArcMap Attributes button and confirm that for the NE Quarter, the Quarter_Quarter values are: NE = 11, NW = 21, SE = 31, SW = 41 for the appropriate sub-quadrants.
    • Click on the IcoMap Query Feature tool.
    • Fill in County Code, Section, Township, and Range with the same numbers used before. Select the Quarter_Quarter_Sections layer.
    • Test each of { 1x, 2x, 3x, 4x } for Quarter_Quarter_Section where x defines one of the Quarter sections ( i.e. 44 = SW corner of the SW Quarter ), and click the Query button.
    • Confirm that the appropriate square is selected for each Quarter_Quarter_Section.

  6. Testing Query Feature operation with Quarter_Quarter_Quarter_Sections and QQQQ:
    • Follow the above instructions except for using Quarter_Quarter_Quarter_Sections.
    • Confirm that the numbering scheme { 1xy, 2xy, 3xy, 4xy } remains correct for the sub-sub-quadrants.

    • Follow the above instructions except for using QQQQ.
    • Confirm that the numbering scheme { 1xyz, 2xyz, 3xyz, 4xyz } remains correct for the sub-sub-sub-quadrants.

  7. Close ArcMap.
