Highlighted Text Window - Bug Fixes

  1. Click on the "Highlighted Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted Text Window and brings up the 'Open' dialog box right away. Close the Open dialog and confirm that the 'Highlight...' selection is 'Mathematical Content'.

  2. [P10 #3096] Return to the Open dialog and create a new Text Document. Edit the text file to add the following text N875544. Make sure that a trailing period is included. Save the file and open it in the Highlighted Text Window.

    The Highlighted Text window opens the new file and the text is highlighted except for the trailing period.

  3. [P10 #3096] Edit the text to change the trailing period to a trailing comma. Open the modified file in the Highlighted Text Window.

    The Highlighted Text window opens the modified file and the text is highlighted except for the trailing comma.

  4. [P10 #3097] Edit the text to prepend a Registered Trademark symbol ( ® ). Open the modified file in the Highlighted Text Window.

    The Highlighted Text window opens the modified file and the mathematical text is highlighted.

  5. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.
