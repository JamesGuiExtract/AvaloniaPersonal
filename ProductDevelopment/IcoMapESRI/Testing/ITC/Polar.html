These test cases test drawing in Polar angles.

  1. Click on the "Highlighted Text Window" button in the IcoMap dialog.

    IcoMap opens a new 'Highlighted Text' window and brings up the 'Open' dialog box right away.

  2. Open the file PolarAngle.txt from the 'Highlighted Text' window.

    The 'Highlighted Text' window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  3. Go to the 'Direction/Angle Definition' tab in the 'Options' dialog box.

  4. Select the 'Angle in degrees...' option in the 'Specify Direction...' group box.

    The 'Polar Angle...' and 'Azimuth...' options should become enabled.

  5. Select the 'Polar Angle...' option and click the 'OK' button.

    The 'IcoMap Options' dialog box should go away. This will change the direction type that IcoMap will be looking for.

  6. Draw the parcel described in PolarAngle.txt using MCR.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment. The highlights that are selected will also change to red. As you move the cursor from normal text to highlighted text the cursor changes from the 'I-Beam' to the 'Pointing Finger'.

  7. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~2698.84'
    Perimeter = ~210.38'

  8. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.
