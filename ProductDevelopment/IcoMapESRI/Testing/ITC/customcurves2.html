Resetting a curve tool to the original parameters

  1. Open the 'Curve Genie' tool.

    The 'Curve Genie' dialog box should appear.

  2. Click the 'Reset…' button.

    The 'Reset Curves' dialog box should appear.

  3. Check the checkbox next to curve 8 and click 'Ok' and close the 'Curve Genie' dialog box.

    The 'Reset Curves' and 'Curve Genie' dialog boxes should disappear and Curve 8 should be set to it's original defaults.
    NOTE: The tooltip and the text next to the button should say <Radial-in bearing><Radial-Out bearing><Radius>

  4. Draw a curve to make sure the prompts are also correct.
