Read-only files

  1. Click on the "New Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Right click on Test002.txt.

    A floating menu should appear

  3. Click on 'Properties'

    The 'Properties' dialog box will appear

  4. Go to the 'General' tab and click the checkbox to the left of 'Read-only'. Click the 'OK' button.

    The file should be read-only now.

  5. Open the file Test002.txt from the 'Highlighted Text' window. Make sure the 'Highlight finder' has 'Mathematical Content' selected.

    The 'Highlighted Text' window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  6. Click the 'Save' button.

    The 'Save' dialog box should appear with the default file name being Test002.txt.

  7. Click the 'Save' button.

    An error should appear indicating that the file is marked as read-only and cannot be saved. You should also be allowed to save the file to a different name. Save it as Test002b.txt.

  8. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.
