Feature Attributes

  1. Create two different workspaces, and test each workspace follow the same instruction
    • Create a Personal Geodatabase with a line feature class.
    • Create a Shape file with a line feature.

  2. Use IcoMap to draw the following segments on the line layer:
    North 100'
    N45dE 150'
    Curve to the right
    R = 70'
    CB = East
    CL = 100'
    S44d24'55"E 50'
    South 150'
    S85d10'16"W 241.90'

    A polyline feature should get created.
    Perimeter = ~803.28'

  3. Select the 'Edit selected feature' button then select the polyline feature you just created.

    The 'Feature Attributes' dialog should appear.

    Note:

    • For Personal Geodatabase, the selected feature should have both Current and Stored features shown in the dialog.
    • For Shape file, the selected feature should have only Current feature shown in the dialog. The stored feature part should be disabled.

  4. Delete the following segment:
    South 150'
    Click 'OK'

    The second to last segment should have gone away.
    Perimeter = ~653.28

  5. Select the 'Edit selected feature' button then select the same polyline feature that was selected before.

    The 'Feature Attributes' dialog should appear.

  6. Select the last segment and click the 'Add Line' button.

    The 'New line' button should have appeared

  7. Make the line:
    Bearing = South
    Distance = 75

    The new segment data should get added before the last segment

  8. Repeat Steps 5 and 6

  9. Click the 'OK' button.

    The segment should look the same as before. The only difference being the line that was deleted is now two segments.
    Perimeter = ~803.28'

  10. Select the 'Edit selected feature' button then select the same polyline feature that was selected before.

    The 'Feature Attributes' dialog should appear.

  11. Select the 'Edit record' button and select the existing start point. Write down the X, Y values on a piece of paper with a pen or pencil because we will use these values for the new part that we will create. Then cancel out of the 'Edit start point' dialog box.

    The X, Y values should be displayed on the piece of paper that you wrote them on.

  12. Select the first open row and click the 'Append new part' button.

    The 'Start new point' dialog box should appear.

  13. Enter the X, Y location that you specified in step 10 then click the 'OK' button.

    A new start point should have been added

  14. Select the first empty row and add a line segment:
    Bearing = N75d22'14"W
    Distance = 78.22'

    It should get added.

  15. Select the first empty row and add a curve to the left:
    Radius = 62.44'
    Delta = 112d22'54"
    CB = S62d50'02"W

    The curve should be added

  16. Click the 'OK' button.

    A new part of the feature should have been added from the original start point off to the left.
    Perimeter = ~1003.98
    (both parts combined)

  17. Select the 'Edit selected feature' button then select the same polyline feature that was selected before.

    The 'Feature Attributes' dialog should appear.

  18. Select the last Start point you added. Then Click the 'Delete' button.

    A dialog box should appear that says "Delete starting point and all associated line and curve segments from part ###"

  19. Click the 'Yes' button.

    The start point, line and curve should not be listed anymore.

  20. Click the 'OK' button.

    The part that was deleted should no longer appear.
