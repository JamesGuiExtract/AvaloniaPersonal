Other things to check

  1. The correct USAFE version should exist. Should have "USAFE/DSAFE Pair ID: 43014" at the bottom

  2. Start button Icons:'Help' and 'IcoMap for ArcGIS' should be located in 'Start > Programs > IcoMap for ArcGIS' by default.

  3. The 'About IcoMap' dialog should exist and show the correct version #, copyright info and registration key.

  4. The user should be able to access the IcoMap help system from the main IcoMap dialog box

  5. Uninstalling IcoMap should remove everything that the installation originally copied to the computer. If things were changed or added they will not be removed.

The location of the files and any common instructions can be found here.
