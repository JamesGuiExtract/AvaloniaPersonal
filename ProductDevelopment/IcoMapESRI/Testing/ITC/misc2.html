Tooltips and Accelerators checklist

  1. All of the toolbar buttons in the main IcoMap dialog box should have tooltips.

  2. All of the toolbar buttons in the image recognition dialog box should have tooltips.

  3. All of the toolbar buttons in the legal text window should have tooltips.

  4. All of the toolbar buttons in the feature attributes dialog box should have tooltips.

  5. All of the controls in the options dialog box should have tooltips.

  6. The controls for the Options dialog box should have accelerators that don't conflict.

  7. The controls for the Curve Genie dialog boxes should have accelerators that don't conflict.

  8. The menus should have accelerators that don't conflict.

  9. The tab order for accelerators should be logical.

  10. The 'OK' button should either be on top or to the left of the 'Cancel' button.

The location of the files and any common instructions can be found here.
