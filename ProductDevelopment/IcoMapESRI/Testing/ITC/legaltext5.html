Non ASCII files e.g. .doc files, .rtf files, etc.

  1. Click on the "New Text Window" button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Open the file Test005.doc from the 'Highlighted Text' window. Make sure the 'Highlight finder' has 'Mathematical Content' selected.

    The 'Highlighted Text' window calls Microsoft word to copy the text in the non-ASCII file to the clipboard and pastes the text into the 'Highlighted Text' window. All bearings, distance, angles, and numbers are highlighted in yellow.

  3. Click the 'Save' button

    The 'Save' dialog box should appear with Test005.txt as the default name.

  4. Click the 'Save' button.

    The file Test005.txt will be saved.

  5. Click the 'Close' (X) button in the 'Highlighted Text' window.

    IcoMap should close the 'Highlighted Text' window.
