Opening multiple instances of the same image.

  1. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Select the file Test012.tif and click the 'open' button.

    IcoMap opens the associated image.

  3. Click on the 'New Spot Recognition Window' button.

    IcoMap opens a new 'Spot Recognition' dialog and brings up the 'Open' dialog box right away.

  4. Select the file Test012.tif and click the 'open' button.

    The user should not get a warning message telling them that they already have that image open.

  5. Close both Spot Recognition windows.

    They should both close.
