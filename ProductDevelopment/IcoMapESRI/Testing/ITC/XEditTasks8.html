Modify Feature Using DIG

  1. Open IcoMap with an empty map. Use AddData to open the Sample2.mdb geodatabase within the Tutorial folder. Select 'Line' and 'Polygon'.

  2. Change the Map Scale to 1:5000. Click the IcoMap toolbar button and start editing within a Polygon layer.

  3. Open a Highlighted Text Window and open the saybrook_legal.txt file.

  4. Select a starting point and draw the entire parcel without finishing the sketch. The error distance shown in the IcoMap status bar should be 0.00 FEET. The purpose of the following steps is to exercise in-place editing, swiping, and the context menu for the DIG.

  5. From the context menu, duplicate segment #2. The error distance shown in the IcoMap status bar should be 112.47 FEET.

  6. From the context menu, move the new segment (now #3) down. The error distance shown in the IcoMap status bar should be 112.47 FEET.

  7. Edit the distance value of segment #4 to 52.47. The error distance shown in the IcoMap status bar should be 52.47 FEET.

  8. Edit the distance value of segment #2 to 60.00. The error distance shown in the IcoMap status bar should be 0.00 FEET.

  9. Drag segment #2 down one row. The error distance shown in the IcoMap status bar should remain 0.00 FEET.

  10. Drag the new segment #2 down two rows. The error distance shown in the IcoMap status bar should remain 0.00 FEET. Also, the parcel should return to its original shape.

  11. From the context menu, duplicate the curve element (segment #5). The error distance shown in the IcoMap status bar should be 60.00 FEET.

  12. Edit the Chord Length of the first curve to be 30.00. The error distance shown in the IcoMap status bar should be 30.00 FEET.

  13. Select the Chord Length of the second curve and click the 30 foot item in the text file. Navigate to another row to finalize the change. The error distance shown in the IcoMap status bar should be 0.00 FEET.

  14. Edit the Chord Bearing of the first curve from S47D41'18"W to S41D47'18"W. The error distance shown in the IcoMap status bar should be 3.08 FEET.

  15. Correct the recently changed Chord Bearing by clicking the appropriate text in the Text Window. The error distance shown in the IcoMap status bar should be 0.00 FEET.

  16. Stop editing without saving the changes.
