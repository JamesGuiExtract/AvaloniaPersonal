Delete Sketch

  1. Click on the 'Draw Line' tool

    IcoMap should prompt for a point

  2. Draw a couple of lines using whatever bearings and distances you want to use.

    Lines should get displayed in the ArcGIS environment.

  3. Click the 'Delete Sketch' tool

    The lines that were just created should disappear.
