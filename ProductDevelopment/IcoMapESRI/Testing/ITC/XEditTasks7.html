Create 2-Point Line Features

  1. Open ArcCatalog. Create a new Personal Geodatabase on your local drive. Create a new Line Feature class and name it 'Road'.

  2. If you don't have the Create Cogo Field button on a toolbar in ArcCatalog, right click next to a toolbar and select Customize.... Select the Command tab. In the Categories window on the left side of the pane, click Geodatabase tools. Drag and drop the Create COGO Fields tool onto any of the ArcCatalog toolbars.

  3. Select the 'Road' feature class.

    The Create Cogo Fields button should be enabled. Click this button. A message indicating success of creating COGO Fields should appear. Click OK to dismiss the message.

  4. Open ArcMap. Drag the 'Road' feature class from ArcCatelog and drop it into the ArcMap.

  5. Select 'Editor > Start Editing'. Select 'Task > Create 2-Point Line Features'.

  6. Click the 'IcoMap' button

    The IcoMap dialog should appear. The highlight frame around IcoMap dialog should appear yellow.

  7. Use IcoMap drawing tool to draw several line segments and several curve segments on the 'Road' layer. Finish sketch.

  8. Right click on the 'Road' in the Table of Contents pane to bring up the context menu. Select 'Open Attribute Table'.

    The Attribute Table should open. There should be multiple features displayed in the table. Each feature is actually a 'segment'. Notice that all curves should have values for COGO Delta, COGO Radius, COGO Tangent fields and so on.

  9. Stop editing without saving the changes.
