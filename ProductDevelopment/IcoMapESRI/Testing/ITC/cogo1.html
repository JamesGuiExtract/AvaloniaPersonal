10 key

  1. Click the 'IcoMap Window' button

    The 'IcoMap' window should appear

  2. From the menu, go to 'Tools > Options…'.

    The 'Options' dialog box should appear.

  3. Go to the 'Keyboard Input' tab and set the directions so that:
    NW = 7
    N = 8
    NE = 9
    W = 4
    E = 6
    SW = 1
    S = 2
    SE = 3
    Click the 'OK' button

    The settings should get written to the registry and the 'IcoMap Options' dialog should disappear.

  4. Click on the 'New Image Recognition Window' button.

    IcoMap opens a new 'Image Recognition' dialog and brings up the 'Open' dialog box right away.

  5. Select the file Test014.tif and click the 'open' button.

    IcoMap opens the associated image.

  6. Draw the parcel using the 10 key COGO option. E.G. If a bearing is "N45d15'W" type "74515" where the "7" stands for "NW".

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  7. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~859214.93
    Perimeter = ~3926.71

  8. Click the 'Close' (X) button in the Image Recognition window.

    IcoMap should close the Image Recognition window.
