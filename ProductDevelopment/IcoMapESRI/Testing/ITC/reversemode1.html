Reverse mode

  1. Click on the 'New Image Recognition Window' button.

    IcoMap opens a new 'Image Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Select the file Test013.tif and click the 'open' button.

    IcoMap opens the associated image.

  3. Set the reverse mode toggle so the linework gets drawn forward.

    The reverse mode icon should have the blue arrow pointing towards the right indicating forward.

  4. Draw the parcel using spot recognition. Use COGO only if the values are not recognizing. Note that the image is rotated 90° so the linework that you create will be at a different orientation than the image.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  5. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~859214.93
    Perimeter = ~3926.71

  6. Set the reverse mode toggle so the linework gets drawn reverse.

    The reverse mode icon should have the red arrow pointing towards the left indicating reverse.

  7. Draw the parcel using spot recognition. Use COGO only if the values are not recognizing.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  8. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected. The geometry of the second parcel should look identical to the first. Check the area and perimeter of both parcels, they should be:
    Area = ~859214.93
    Perimeter = ~3926.71

  9. Click the 'Close' (X) button in the Image Recognition window.

    IcoMap should close the Image Recognition window.
