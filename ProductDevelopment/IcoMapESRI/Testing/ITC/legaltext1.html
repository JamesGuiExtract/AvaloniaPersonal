This test case tests the various functionalities associated with Printing from the Legal Text dialog.

  1. Click on the “New Text Window” button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Open the file Test001.txt from the Highlighted Text window. Make sure the 'Highlight finder' has 'Mathematical Content' selected.

    The ‘Highlighted Text’ window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  3. Click the ‘Print’ button in the Highlighted Text window.

    The ‘Highlighted Text’ window opens the print dialog box. When the print is initiated, the printed output will contain the contents of the specified file and all the text that was highlighted on screen will also be highlighted on paper. If printing to a color printer, the highlighted text that was not used for MCR should be printed yellow and the text that was used for MCR should be printed red.

  4. Click the ‘Close’ (X) button in the ‘Highlighted Text’ window.

    IcoMap should close the ‘Highlighted Text’ window.
