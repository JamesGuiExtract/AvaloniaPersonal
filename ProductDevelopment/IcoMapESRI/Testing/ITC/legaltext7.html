Document source linking is off

  1. Click on the 'New Text Window' button in the IcoMap dialog.

    IcoMap opens a new Highlighted text window and brings up the 'Open' dialog box right away.

  2. Open the file Test007.txt from the 'Highlighted Text' window. Make sure the 'Highlight finder' has 'Mathematical Content' selected.

    The 'Highlighted Text' window opens the specified file and all bearings, distance, angles, and numbers are highlighted in yellow.

  3. Go to the 'General' tab in the 'IcoMap Options' dialog box and un-check the option to enable automatic source document linking. Click the 'OK' button to accept the change.

    The option will be disabled and the 'IcoMap Options' dialog box will disappear.

  4. Use the IcoMap drawing tools to draw out the legal description using MCR.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment. The highlights that are selected will also change to red.

  5. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~1889645.48
    Perimeter = ~7773.50

  6. Click on the ArcMap 'Identify' button then select the parcel you just created.

    ArcMap's 'Identify Results' dialog box should appear

  7. Right click on the feature in the left hand window in the 'Identify Results' dialog box.

    There shouldn't be a 'Hyperlinks' option in the floating menu indicating that there isn't a hyperlink for the feature.

  8. Click the 'Close' (X) button in the 'Highlighted Text' window. IcoMap should close the 'Highlighted Text' window.
