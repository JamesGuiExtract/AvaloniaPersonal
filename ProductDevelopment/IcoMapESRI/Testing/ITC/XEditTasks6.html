Select Features Using an Area

  1. Click Add Data button on ArcMap Toolbar. Select TestDB1, select both Line and Polygon and click Add.

  2. Select 'Editor > Start Editing'. Select 'Task > Select Features Using an Area'.

  3. Click the 'IcoMap' button

    The IcoMap dialog should appear. The highlight frame around IcoMap dialog should appear yellow.

  4. Use menu item 'Selection > Clear Selected Features' to clear the selection if any.

  5. At IcoMap command line, type in '6876.42,3771.90' as the starting point, then 'W' and '34567', 'S' and '12345'. Finish the sketch.

    Both the polygon and the line should be selected.

  6. Stop editing without saving the changes.
