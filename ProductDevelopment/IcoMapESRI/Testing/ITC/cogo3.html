Not duplicate characters for different directions.

  1. From the menu, go to 'Tools > Options…'.

    The 'Options' dialog box should appear.

  2. Go to the 'Keyboard Input' tab and set the directions so that:
    NW = 1
    N = 2
    NE = 3
    W = 4
    E = 1
    SW = 2
    S = 3
    SE = 4
    Click the 'OK' button

    An error dialog box should appear telling the user that duplicate keys were found.

  3. Click the 'Cancel' button.

    The 'Options' dialog box should go away.
