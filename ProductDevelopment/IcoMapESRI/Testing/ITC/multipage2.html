Drawing and saving highlights with multi-page images.

  1. Click on the 'New Image Recognition Window' button.

    IcoMap opens a new 'Image Recognition' dialog and brings up the 'Open' dialog box right away.

  2. Select the file Test019.tif and click the 'open' button.

    IcoMap opens the associated image to the first page. The 'move backwards' button should be disabled. The page number indicator should be 1

  3. Draw the parcel on page 1 using spot recognition. Use COGO only if the text doesn't recognize.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  4. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~345941.43'
    Perimeter = ~2928.14'

  5. Go to page 2

    Page 2 should be displayed in the image window. The highlights from page 1 should not be displayed in this view.

  6. Draw the parcel on page 2 using spot recognition. Use COGO only if the text doesn't recognize.

    As you pick on line and curve data, the lines should get drawn to the ArcMap environment.

  7. Finish the sketch.

    The lines and curves you drew should get written to the ArcGIS geodatabase and the newly created shape will be selected.
    Area = ~557549.48
    Perimeter = ~3125.92

  8. Go back to page 1.

    The image of page 1 should be displayed in the image window. The highlights from page 2 should not be displayed. The highlights from page 1 should be displayed.

  9. Click the 'Save' button.

    The 'Save' dialog box should appear.

  10. Save the file as Test019.tif.gdd.

    The file Test019.tif.gdd should get created

  11. Click the 'Close' (X) button in the Image Recognition window.

    IcoMap should close the Image Recognition window.

  12. Click on the 'New Image Recognition Window' button.

    IcoMap opens a new 'Image Recognition' dialog and brings up the 'Open' dialog box right away.

  13. Select the file Test019.tif.gdd and click the 'open' button.

    IcoMap opens the associated image to the first page. The 'move backwards' button should be disabled. The page number indicator should be 1. All of the highlights for page 1 should be displayed.

  14. Go to page 2

    Page 2 should be displayed in the image window. The highlights from page 1 should not be displayed in this view. The highlights for page 2 should be displayed.

  15. Click the 'Close' (X) button in the Image Recognition window.

    IcoMap should close the Image Recognition window.
