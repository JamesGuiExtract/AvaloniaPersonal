Command Shortcuts

    By default, You should be able to type /key for the command shortcuts. In the IcoMap command line, type '/' followed by a character corresponding to a command shortcut and make sure that the correct command is activated.

    Shortcut Command
    /1 Curve #1
    /2 Curve #2
    /3 Curve #3
    /4 Curve #4
    /5 Curve #5
    /6 Curve #6
    /7 Curve #7
    /8 Curve #8
       
    /0 Line (Direction)
    /A Line (Internal/Deflection Angle)
    /9 Custom Curve (Curve Genie)
    /* Concave Right
    // Concave Left
    /- Delta < 180° (shallow)
    /+ Delta > 180° (bulbous)
    /f Draw Forward (normal)
       
    /r Draw Reverse
    /. Finish Sketch
    /p Finish Part
    /x Delete Sketch
