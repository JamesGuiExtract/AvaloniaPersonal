  • Config File Support

    • Create or identify at least two distinct DEPs.
      • Modify StandardLabDE.config in the appropriate Solution folder so that DataEntryPanelFileName is set to the first DEP.
      • Confirm that the appropriate DEP is loaded in stand-alone mode.
      • Confirm that the appropriate DEP is loaded in FAM mode.

      • Modify StandardLabDE.config in the appropriate Solution folder so that DataEntryPanelFileName is set to the second DEP.
      • Confirm that the appropriate DEP is loaded in stand-alone mode.
      • Confirm that the appropriate DEP is loaded in FAM mode.

    • Modify StandardLabDE.config in the appropriate Solution folder so that FileProcessingTaskName is set to a unique file processing task.
    • Click on Check for new components in the FAM.
    • Confirm that the specified file processing task can be selected in the FAM.

    • Modify StandardLabDE.config in the appropriate Solution folder so that ApplicationTitle is set to a unique string.
    • Exercise LabDE in stand-alone mode.

    • Exercise LabDE in FAM mode.
    • Confirm that the specified title is visible in the Image Viewer.
