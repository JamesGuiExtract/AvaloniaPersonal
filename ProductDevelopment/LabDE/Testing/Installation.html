LabDE Installation Testing Checklist

NOTE: Items prefaced with an asterisk indicate functionality that is not yet complete.

  1. Installation Options

    • Confirm that the Main Menu for the LabDE installation includes Extract Systems information plus a LabDE logo.
    • Confirm that the Install Products button displays a list of installation packages in the following order.
      • Microsoft .NET 3.5
      • Install LabDE
      • SQL Server 2005 Express
      • Extract Systems License Manager
      • HL7 Integration Engine
    • Confirm that the Other Products button displays only brief descriptions of and logos for ID Shield, FLEX Index, and IcoMap.
    • Confirm that the Contact Us button displays address, phone, fax, website, and email information for Extract Systems.
    • Confirm that the Exit button briefly displays a thank you message plus contact information for Support.

    • Confirm that LabDE will not install if Microsoft .NET 3.5 is not installed.
    • Confirm that selecting SQL Server 2005 Express allows the following choices.
      • MS SQL 2005 Express
      • SQL Manager 32-bit
      • SQL Manager 64-bit
    • Confirm that both MS SQL 2005 Express and the appropriate SQL Manager can be installed.
    • Confirm that Extract Systems License Manager can be installed.
    • Confirm that a local USB key can be accessed.
    • Confirm that the HL7 Integration Engine can be installed.
    • [DataEntry #438, #440] Corepoint version
      • Check the version information for C:\Program Files\Corepoint Health\Corepoint Integration Engine\NIMonSvc.exe.
      • Confirm that the version number is 4.0.1.x.

    • Individually-Licensed Items

      • Create Searchable PDF
        • Use the COM License Generator utility to create a license which includes the Create Searchable PDF package. Copy this file to the CommonComponents folder.
        • Confirm that the Convert to searchable PDF task is available in the File Action Manager.
        • Use this task to convert some TIF images, and confirm that the output image is also available in PDF format.

      • Disable USB Key Serial Numbers
        • Use the COM License Generator utility to create a license which includes the Disable USB Key Serial Numbers package. Copy this file to the CommonComponents folder.
        • Confirm that a customer rules set that is locked to a specific serial number can be run with a different USB Key.

      • Disable USB Counters (locked to PC) WAYNE MUST PERFORM THIS TEST!!!
        • Use the COM License Generator utility to create a license which includes the Disable USB Counters (locked to PC) package. Copy this file to the CommonComponents folder.
        • Confirm that a customer rules set that is locked to a specific serial number can be run without a USB key.

      • Handwriting Recognition
        • Not Tested.

      • Image Cleanup Engine
        • Use the COM License Generator utility to create a license which includes the Image Cleanup Engine package. Copy this file to the CommonComponents folder.
        • Confirm that the Cleanup image task is available in the File Action Manager.
        • Use this task to clean some images ( see the image files associated with the automated test ), and confirm that the output images are cleaner.

      • MICR Finding Engine
        • Not Tested.

      • PDF Read / Write
        • Use the COM License Generator utility to create a license which includes the PDF Read + Write package. Copy this file to the CommonComponents folder.
        • Confirm that PDF documents can be opened in the Image Viewer.
