LabDE Pre-Release Testing

  1. Exercise Demo Application

    • Review the Demo_LabDE instructions.
    • Exercise the Demo_LabDE FPS files and links.
    • Exercise the Demo_LabDE portion of the Sanity Test checklist.
    • Confirm that the Demo_LabDE images are non-sensitive by reviewing the original images and confirming that sensitive data has been changed.

  2. Create Master CD

    • Create an autorun.inf file for the root of the CD with the following lines:
      • [autorun]
      • open=LabDEInstall\Launch.exe
      • icon=LabDEInstall\LabDE.ico
    • Add the following folders and their contents to the root of the CD:
      • DotNet 3.5 Framework
      • Extract Systems LM
      • LabDE
      • LabDEInstall
      • SQLServerExpress2005
    • Create an appropriate ISO image to facilitate creation of copied CDs.

    • Test the LabDE CD by installing each application.
