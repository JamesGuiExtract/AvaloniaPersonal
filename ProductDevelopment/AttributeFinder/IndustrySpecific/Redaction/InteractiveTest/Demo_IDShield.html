Demo_IDShield Testing Checklist

  • Installation

    • Copy the Demo_IDShield folder from the appropriate build folder to the test machine.
    • As Administrator,
      • Run Clean all.bat to create the database.
      • Run SQL Management Studio and exercise CreateDomainLogin_AddForDemoDBs.sql from the I:\Common\Testing\Product Testing\IDShield_Sanity folder. Note that instructions to update other demo databases can be removed before executing the query.

  • AutoRedact

    • Navigate to the .\DemoFiles\Installs\AutoRedact folder.
    • Run install.bat.
    • Confirm that FPS files have been copied to the .\Demo_IDShield\FPS folder.
    • Confirm that image files have been copied to the .\Demo_IDShield\Input folder.
    • Exercise Automatically Perform Redactions.fps.
    • Results
      • Confirm that files successfully queue and process.
      • Confirm that the Input folder contains USS and VOA files for all 14 test images.
      • Confirm that the Output folder contains TIF and XML files for all test images except for TestImage009.tif and TestImage014.tif.
      • Confirm that all of the images in the Output folder contain redactions.
      • Confirm that the redactions include an exemption code with type information.

  • HybridDemo

    • Navigate to the .\DemoFiles\Installs\HybridDemo folder.
    • Run install.bat.
    • Confirm that FPS files have been copied to the .\Demo_IDShield\FPS folder.
    • Confirm that image files have been copied to the .\Demo_IDShield\Input folder.
    • Exercise Standard - Compute and Auto Redact.fps.
    • Results
      • Confirm that files successfully queue and process.
      • Confirm that the Input folder contains USS and VOA files for all 30 test images.
      • Confirm that the Output folder contains 8 each of TIF and XML files.
      • Confirm that each TIF image contains redactions with exemption codes.
    • Exercise Standard - Verify Redactions.fps.
    • Results
      • Confirm that files presented for verification do not contain only HCData items.
        • TestImage013.tif contains HCData but also contains MCData and Clues items.
      • Confirm that all 18 files are now represented in the Output folder.
    • Exercise Legislation Guard - Compute and Auto Redact.fps.
    • Results
      • Confirm that 30 files successfully queue and process.
      • Confirm that the Input folder contains .DOB.VOA files for all 30 test images.
      • Confirm that the Output folder contains 1 .DOB.XML file (TestImage001.tif.dob.xml).
      • Confirm that the associated TIF image contains the original SSN redactions plus one new DOB redaction.
    • Exercise Legislation Guard - Verify DOB Redactions.fps.
    • Results
      • Confirm that the following files are presented for verification.
        • TestImage008.tif - with existing SSN redaction, 1 Clue, 1 MCData
        • TestImage012.tif - with existing SSN redaction, 1 Clue
        • TestImage018.tif - with existing SSN redaction, 1 Clue, 1 MCData
      • Confirm that the Output folder contains .DOB.XML files for each of the above images.
      • Confirm that the associated TIF files also contain DOB redactions.

  • VerifyAllInterestingDocs

    • Navigate to the .\DemoFiles\Installs\VerifyAllInterestingDocs folder.
    • Run install.bat.
    • Confirm that FPS files have been copied to the .\Demo_IDShield\FPS folder.
    • Confirm that image files have been copied to the .\Demo_IDShield\Input folder.
    • Exercise Compute Redactions.fps.
    • Results
      • Confirm that files successfully queue and process.
      • Confirm that the Input folder contains USS and VOA files for all 14 test images.
      • Confirm that the Output folder is empty.
    • Exercise Verify Redactions.fps. Add and resize redactions as appropriate for each image.
    • Results
      • Confirm that each item and each page must be visited.
      • Confirm that Type can be blank for a manual redaction.
      • Confirm that the Output folder contains TIF and XML files for all 14 test images.
      • Confirm that all of the images in the Output folder contain redactions.
      • Confirm that the redactions include an exemption code with type information.
