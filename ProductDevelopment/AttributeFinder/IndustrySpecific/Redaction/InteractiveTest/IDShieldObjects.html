ID Shield Objects Testing Checklist

  1. Redaction Demo Installation

    • Install RedactionDemo or Copy the RedactionDemo folder to the local C: drive.

    • Confirm presence of Automatically Perform Redactions shortcut in RedactionDemo folder.
    • Confirm presence of Compute Redactions shortcut in RedactionDemo folder.
    • Confirm presence of Perform Manual Redactions shortcut in RedactionDemo folder.
    • Confirm presence of Verify Redactions shortcut in RedactionDemo folder.
    • Confirm presence of Clean batch file in RedactionDemo folder.
    • Confirm presence of non-empty FPS, Images,  and Rules subfolders.

  2. Redaction Demo DB Creation

    • Run the Clean.bat file.
    • Run the FAM DB Admin utility.
    • Set the Database Server Name to (local).
    • Set the Database Name to Demo_IDShield.
    • Login to the existing database with the password a.
      • Open the Summary tab.
      • Confirm the presence of two actions: Compute and Verify.


  3. Test Images - Compute Redaction

    • Open and execute the Compute Redactions FPS file.

    • With a local USB Key confirm that a test image opens and processes without error.
    • Remove the local USB Key and restart processing. Confirm that processing the test image does not finish processing and that the UEX contains errors related to USB key connectivity.
    • Connect to a remote USB Key and restart processing. Confirm that the test image opens and processes without error.

    • Confirm that each test image opens and processes without error.
    • Remove or rename the USS file for one test image. Confirm that processing this image runs OCR before finding and redacting Attributes.

    • [FlexIDSCore #2762] Execute the Automatically Perform Redactions FPS file, then execute the Compute Redactions FPS file again. Confirm that the previously created redacted images are NOT supplied and processed.
