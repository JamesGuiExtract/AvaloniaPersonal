ID Shield Features Testing Checklist

  • Redacting Fixed Locations On Document

    • Execute Rules
      • Confirm presence of a Use previously saved OCR results if available checkbox.
      • Confirm presence of an If previous OCR results are not available group box.
      • Confirm presence of a radio button for OCR all pages of the image.
      • Confirm presence of a radio button for OCR specific pages.
      • Confirm presence of an associated edit box to specify page numbers.
      • Confirm presence of a radio button for Do not OCR any of the pages (Use this option only if you're sure the rules don't require OCR results).
      • Confirm presence of a Save OCR results to speed up re-processing in the future checkbox.
      • Confirm presence of a Use cleaned image if available checkbox.

      • Confirm that settings are persistent.
      • Confirm that a previous version of Execute Rules that was saved with Extract data from USS file if the input file is an image and a corresponding USS file exists opens and is properly configured with Use previously saved OCR results if available.
      • Confirm that a previous version of Execute Rules that was saved with If input file is an image, but no corresponding USS file exists, create a USS file, and then extract data from the USS file plus OCR all pages of the image opens and is properly configured with OCR all pages of the image.
      • Confirm that a previous version of Execute Rules that was saved with If input file is an image, but no corresponding USS file exists, create a USS file, and then extract data from the USS file plus OCR specific pages opens and is properly configured with OCR specific pages.

      • Create a basic rule set that does not require full OCR of an image.
      • Exercise the rule set with the Do not OCR any pages option and confirm that rules execute as desired.

    • Select Page Region
      • Confirm the text change for label 3) to Select what should be returned.
      • Confirm presence of a radio button for Return text.
      • Confirm presence of an associated combo box with Include and Exclude.
      • Confirm presence of an associated combo box with characters, words and lines.
      • Confirm presence of a radio button for Re-OCR and return text.
      • Confirm presence of an associated edit box to define the number of degrees of clockwise rotation.
      • Confirm presence of a radio button for Return image region.
      • Confirm presence of an associated edit box for the assigned text.

      • Confirm that settings are persistent.
      • Confirm that a previous version of Select Page Region that was saved without OCR the included text opens and is properly configured with Return text plus Include characters.
      • Confirm that a previous version of Select Page Region that was saved with OCR the included text without rotation opens and is properly configured with Re-OCR and return text plus an empty rotation edit box.
      • Confirm that a previous version of Select Page Region that was saved with OCR the included text with rotation opens and is properly configured with Re-OCR and return text plus the appropriate rotation setting.

      • Confirm proper behavior of a basic rule set
        • to Return text with inclusion of characters.
        • to Return text with inclusion of words.
        • to Return text with inclusion of lines.
        • to Return image region with assigned text.
      • Confirm that this combination of rule object enhancements can successfully redact a fixed location on a document without any OCR operation.
