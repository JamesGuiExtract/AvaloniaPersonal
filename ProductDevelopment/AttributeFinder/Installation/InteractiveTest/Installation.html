Installation Testing Checklist

  1. General

    • Licensing
      • Confirm that an SDK license enables each Start menu item from an SDK installation
      • Confirm that an RDT license enables each Start menu item from an RDT installation
      • Confirm that an SDK license does not enable the Ruleset Editor
      • Confirm that an RDT license does enable the Ruleset Editor

    • Install Options
      • Confirm that not agreeing with the terms of the license agreement halts the installation with no files copied to the system.
      • Confirm that a Custom SDK installation allows installation of the SDK Merge Module.
      • Confirm that Merge Modules for FlexIndexSDK, InputFunnel and VOAClient are copied to the \FlexIndex\SDK\Redist folder.
      • Confirm that a Compact SDK installation does not install Tutorials or Help.

  2. Windows XP

    • Software Development Toolkit (UCLIDFlexIndexSDK or SDK)
      • Confirm that SDK installs successfully
      • Each Shortcut on Start menu "works"
      • EAV Generator is available from the Start menu
      • Feedback Manager is available from the Start menu
      • File Processing Manager is available from the Start menu
      • FlexIndex Help is available from the Start menu
      • Rule Tester is available from the Start menu
      • Spatial String Viewer is available from the Start menu
      • User License is available from the Start menu

      • Confirm that uninstall of SDK is successful

    • Rule Development Toolkit (RDT)
      • Confirm that RDT does not install without SDK installation
      • Confirm that RDT installs successfully after SDK installation
      • Each Shortcut on Start menu "works"
      • EAV Generator is available from the Start menu
      • Feedback Manager is available from the Start menu
      • File Processing Manager is available from the Start menu
      • FlexIndex Help is available from the Start menu
      • Index Converter is available from the Start menu
      • RDT Configuration is available from the Start menu
      • Ruleset Editor is available from the Start menu
      • Rule Tester is available from the Start menu
      • Run Diagnostic Tests is available from the Start menu
      • Spatial String Viewer is available from the Start menu
      • Test Harness is available from the Start menu
      • User License is available from the Start menu

      • Confirm that uninstall of SDK with RDT still installed is prohibited
      • Confirm that uninstall of RDT is successful
      • Confirm that uninstall of RDT retains SDK items
      • Confirm that subsequent uninstall of SDK is successful

    • VOAClient
      • Confirm that VOA Client installs successfully by itself
      • Copy the VOATester EXE to the test machine and confirm that a license error message is seen when no license file is present.
      • Confirm that a VOAClient license file removes the above error message.
      • VOATester
        • Confirm appropriate message for VOA file without Attributes
        • Confirm appropriate message for VOA file with Attributes
        • Confirm appropriate message for VOA file with a defined Splitter
        • Confirm appropriate message for VOA file with a defined Validator

      • Confirm that uninstall of VOA Client is successful
