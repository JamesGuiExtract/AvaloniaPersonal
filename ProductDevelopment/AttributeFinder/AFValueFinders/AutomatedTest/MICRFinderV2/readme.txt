To Run Memory Leak Test (this will create the necessary FAM DB and FPS file and run the test):
  1. Install dotnet core SDK (e.g., download and run https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-2.2.108-windows-x64-installer)
  2. If DB is to be hosted on a different computer set environment variable TESTING_DB_SERVER to the server name, else "(local)" will be used
  3. Execute run_memory_leak_test.bat
