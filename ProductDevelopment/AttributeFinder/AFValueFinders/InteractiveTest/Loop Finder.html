Miscellaneous

  • [FlexIDSCore #3195] Loop Finder exceptions
    • Exercise the Loop Finder such that an exception is thrown by one of the components. One possibility is to include an invalid regular expression.
    • Confirm that descriptive exception information is provided.
    • Create an RSD file test.rsd with a Loop Finder rule configured such that it will reach the maximum number of iterations before it stops searching. Make sure that the option to throw an exception if it reaches the maximum number of iterations is checked.
    • Configure another RSD file to find from test.rsd and test it on an image.
    • Confirm that an exception is thrown and that it contains information about the RSD file containing the loop and the input file that was being processed when the exception was thrown.

  • [FlexIDSCore #3191] Loop Finder UI should open config on doubleclick
    • Open a new RuleSet Editor.
    • Create a Loop Finder rule and configure it with a finding rule, preprocessor, and condition.
    • Confirm that double-clicking on any of the read-only text boxes (Execute rule, Execute preprocessor, or While this condition is) opens the corresponding Configure dialog.
    • Confirm that double-clicking on them while holding down the <Ctrl> key opens the corresponding Select dialog.
    • Confirm that double-clicking on them while holding down the <Alt> key opens the corresponding Configure description dialog.

  • [FlexIDSCore #4227, #4257] Double-clicking read-only text boxes
    • On Server 2008, Server 2008R2, or Windows 7, confirm the following behavior:
      • In the Ruleset Editor, configure a loop finding rule. Click OK.
      • Double-click on the finding rule to reopen the property page.
      • Double-click on one of the read-only text boxes in the property page.
      • Confirm that the text of the read-only label has not been copied to the clipboard.
  • [FlexIDSCore #4259, #4260]
    • In the property page for the loop finder:
      • Click on the Do loop option with the mouse.
      • Move the cursor straight down and click on the While option.
      • Move the mouse down and click on the For loop option.
      • Confirm that only one radio button appears selected at a time.

      • Close and reopen the property page.
      • Hit tab until the Do loop option has the focus box.
      • Press the down arrow.
      • Confirm that the loop setup groupbox and loop type groupbox remain in sync as you hit the arrow keys.
