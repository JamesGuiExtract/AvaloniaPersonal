Ruleset Editor - Value Finders
Box Finder

Box Finder Rule

  • Confirm that Configure... button displays the Configure Box Finder dialog.
  • Clues group box
    • Confirm that a list box for Clues to find box is provided.
    • Confirm that Add..., Remove and Modify... buttons are provided next to the list box.
    • Confirm that the Add... button is always enabled.
    • Confirm that the Remove button is enabled only if one or more clues is selected.
    • Confirm that the Modify... button is enabled only if exactly one clue is selected.
    • Confirm that the Modify... button displays a pre-populated Text dialog.
      • Changes made are persistent.
    • Graphical UP arrow button moves the selected item up in the list box.
    • Graphical DOWN arrow button moves the selected item down in the list box.
    • Graphical UP and DOWN arrow buttons become inactive when reaching top or bottom of the list.
    • [FlexIDSCore #2875] Confirm that double-clicking on a list box entry displays a pre-populated Text dialog.

    • Confirm that the Case sensitive checkbox is always enabled.
    • Confirm that the Case sensitive checkbox is unchecked by default.
    • Confirm that the Above clue is a regular expression checkbox is always enabled.
    • Confirm that the Above clue is a regular expression checkbox is unchecked by default.

    • Confirm that the Select location of clue relative to box image is next to the buttons.
    • Confirm that the image includes Clue in red text and Data in blue text.
    • Confirm that the default position of the red Clue text is in the center cell.
    • Confirm that checking in each of the nine grid cells moves the red Clue text to the associated cell.
    • Confirm that group box settings are persistent.

    • [FlexIDSCore #2876] Refresh and z-order.
      • Add a clue to the list and make sure that it is selected.
      • Use Alt+Tab to activate a different application.
      • Use Alt+Tab to reactivate the RuleSet Editor.
      • Confirm that the balloon help icon does not obscure the selected clue.

    • [FlexIDSCore #2878] Support for clues from file via file://.
      • Clear the Clues to find box: list box and specify a file name beginning with file:// inside the Text dialog and click the OK button.
        • Confirm that the Add... button is still enabled.
        • Confirm that the Remove button is enabled while the entry is selected.
        • Confirm that the Modify... button is enabled while the entry is selected.
        • Confirm that the Remove and Modify... buttons are disabled while the entry is not selected.
      • Add several entries inside the Clues to find box: list box (do not include a file name entry) and double click one of the entries to bring up the Text dialog. Modify the text in the edit box to a file name (including the file:// header) and click the OK button.
      • Confirm that the entry is added to the list.

  • Search Scope group box
    • Confirm that a radio button exists for All pages.
    • Confirm that the All pages radio button is the default selection.

    • Confirm that a radio button exists for First pages.
    • Confirm that an associated edit box exists for the First pages radio button.
    • Confirm that the associated edit box is enabled only if the radio button is selected.
    • Confirm that the associated edit box accepts only numeric characters.

    • Confirm that a radio button exists for Last pages.
    • Confirm that an associated edit box exists for the Last pages radio button.
    • Confirm that the associated edit box is enabled only if the radio button is selected.
    • Confirm that the associated edit box accepts only numeric characters.

    • Confirm that a radio button exists for Specified pages.
    • Confirm that an associated edit box exists for the Selected pages radio button.
    • Confirm that the associated edit box is enabled only if the radio button is selected.
    • Confirm that the associated edit box accepts comma-delimited page numbers and page ranges.
    • Confirm that a validation error is displayed if alphabetic characters are typed in the associated edit box.

    • Confirm that group box settings are persistent.

  • Expected size of box group box
    • Confirm that an edit box exists for minimum page width.
    • Confirm that an edit box exists for maximum page width.
    • Confirm that an edit box exists for minimum page height.
    • Confirm that an edit box exists for maximum page height.

    • Confirm that each edit box accepts only numeric characters for input.
    • Confirm that the limits for each edit box are 0 and 100.
    • Confirm that an error message is displayed if the minimum width is greater than the maximum width.
    • Confirm that an error message is displayed if the minimum height is greater than the maximum height.
    • Confirm that group box settings are persistent.

    • [FlexIDSCore #2874] Maximum page width.
      • Exercise the RSD file and the image associated with the SCR.
      • Confirm that no box is found.
      • Modify the maximum width to 12 percent and confirm that one box is found.

    • [FlexIDSCore #4589] More Precise Box Finder Box Sizes
      • Confirm that decimal percentages are supported in Box Finder configuration.
      • Exercise the image found at I:\Common\Testing\PVCS_Testing\FlexIDSCore\4589.
      • Confirm that the date of birth box can be isolated without including boxes that are significantly smaller.

  • Return Value group box
    • Confirm that a radio button exists for Return spatial information of found box and artificially associated the spatial area with the following text.
    • Confirm that this radio button is selected by default.
    • Confirm that an associated edit box exists for specifying text.
    • Confirm that the default text is 000-00-0000.
    • Confirm that an associated check box exists for Exclude vertical spatial area of clue from box.
    • Confirm that the check box is unchecked by default.
    • Confirm that the edit box and check box are enabled only when the radio button is selected.

    • Confirm that a radio button exists for Return text found within the box.
    • Confirm that an associated check box exists for Include clue text as part of the return value.
    • Confirm that the check box is enabled only when the radio button is selected.

    • Confirm that a separate check box is available for Include all lines that are found in the output.
    • Confirm that the check box is unchecked by default.

    • Confirm that group box settings are persistent.
