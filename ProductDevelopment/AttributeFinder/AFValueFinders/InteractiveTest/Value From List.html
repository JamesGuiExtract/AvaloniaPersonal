Value From List 

  • Add button displays Text dialog.

    Text Dialog

    • Specify the text edit box is editable.
    • OK button saves changes and exits dialog.
      • Text entered appears in Define Clue Text(s) list box.
    • Cancel button ignores changes and exits dialog.
    • X button ignores changes and exits dialog.
  • Remove button displays Confirm Delete dialog.

    Confirm Delete Dialog

    • YES button deletes the finding rule and exits dialog.
    • NO button ignores delete request and exits dialog.
  • Remove removes multiple selected entities.
  • CTRL + Left Click allows multiple selection.
  • SHIFT + Left Click allows group selection.
  • Modify button displays pre-populated Text dialog.
    • Changes made are persistent.
  • Load List button displays Open dialog.

    Open Dialog

    • Specify the file name: edit box is editable.
      • Displays an error message when the file specified does not exist.
  • List box is populated correctly after loading from a file.
  • Load List button displays Confirm, when entries exist in list box.

    Confirm Dialog

    • YES button loads the file and exits dialog
    • NO button ignores the load request and exits dialog.
  • Save List button displays Configuration message box when no replacements are specified.
  • Save List button displays Save dialog.

    Save Dialog

    • Specify the file name: edit box is editable.
    • Displays a confirmation message when the file specified already exists.
    • Saved output file is created correctly.
  • Strings are case sensitive check box is checkable.
  • OK button saves changes and exits dialog.
  • Cancel button ignores changes and exits dialog.
  • X button ignores changes and exits dialog.
