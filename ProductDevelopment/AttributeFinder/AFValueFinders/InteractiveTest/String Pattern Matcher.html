    Text Dialog

    • Specify the text edit box is editable.
    • OK button saves changes and exits dialog.
      • Text entered appears in Define Preprocessors list box.
    • Cancel button ignores changes and exits the dialog.
    • X button ignores changes and exits the dialog.
  • Highlighting an existing clue in the list box entry activates Remove and Modify.
  • CTRL + Left Click allows for multiple selection of list box items.
  • SHIFT + Left Click allows for multiple selection of list box items.
  • Remove button displays Confirm Delete dialog.

    Confirm Delete Dialog

    • YES button deletes the finding rule and exits the dialog.
    • NO button ignores delete request and exits dialog.
  • Modify button displays a pre-populated Text dialog.
    • Changes made are persistent.
  • Select... button displays Select Data Scorer Dialog.
