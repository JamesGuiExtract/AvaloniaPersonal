NOTE: Items prefaced with an asterisk indicate functionality that is not yet complete.

  1. Instructions and Installation

    • Confirm that the latest Demo_FlexIndex folder can be copied from the current build folder to C:\.
    • Confirm that all files under the Rules folder are encrypted.
    • Confirm that DataEntry: Verify extracted data requires only a FLEX Index Client license.
    • Confirm that the blank database associated with the demo has the current schema version.

  2. Demo Images

        • any other situations that come to mind

    • Parcel Number

    • Consideration Amount

      • Confirm that text can be swiped into the Consideration Amount edit box.

    • Document Type

      • Confirm that the Legal description table supports tabbing by row.

  4. [JIRA #12236] Enable Trace

      • Add an entry to the Demo_FLEXIndex\DataEntry Solution\DemoFlexIndex.config file as follows.
        • <Extract.DataEntry.Utilities.DataEntryApplication. Properties.Settings>
        • <setting name="EnableTrace" serializeAs="String">
        • <value>True</value>
        • </setting>
        • </Extract.DataEntry.Utilities.DataEntryApplication. Properties.Settings>
      • Run the Dbgview.exe utility found in the I:\users\Steve_Kurth\Public\Utilities folder.
    • Function
      • Specify * in the Include: field and verify Example01.tif.
      • Swipe some text into the Parcel Number field.
      • Confirm that appropriate entries are logged for FieldUpdated and SwipedText.
      • Swipe a different Zip into the found Return Address cell.
      • Confirm that appropriate entries are logged for SwipedText, FieldUpdated, AutoUpdateQuery and ValidationQuery.

      • Confirm that the logged text also includes the following information:
