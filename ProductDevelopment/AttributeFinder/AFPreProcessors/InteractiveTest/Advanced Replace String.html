Advanced Replace String Dialog

  • String to find edit box is editable.
  • Replace with edit box is editable.
  • Confirm the icon on the right side of String to find text can be clicked to display the balloon help.
  • Confirm the icon on the right side of Replace with text can be clicked to display the balloon help.
  • Confirm the drop down menu on the right side of String to find edit box displays a list of tags and utility functions.
    • Put the cursor at any position inside String to find edit box and select one of the tags, it will be inserted at this position.
    • Select all characters inside String to find edit box and select one of the tags; the selected text will be an argument for the function.
  • Confirm the drop down menu on the right side of Replace with edit box displays a list of tags and utility functions.
    • Put the cursor at any position inside Replace with edit box and select one of the tags, it will be inserted at this position.
    • Select all characters inside Replace with edit box and select one of the tags; the selected text will be an argument for the function.
  • Confirm the browse button on the rightmost side of String to find edit box displays an open file dialog.
    • Select one file and click OK to close the dialog, confirm the file name and its full path is displayed in the String to find edit box, with file:// added to the beginning of the file name.
  • Confirm the browse button on the rightmost side of Replace with edit box displays an open file dialog.
    • Select one file and click OK to close the dialog, confirm the file name and its full path will be displayed in the Replace with edit box, with file:// added to the beginning of the file name.
  • All occurrences radio button is exclusively selectable.
  • First occurrence radio button is exclusively selectable.
  • Last occurrence radio button is exclusively selectable.
  • Specified occurrence radio button is exclusively selectable.
    • Selecting Specified occurrence radio button allows editing in the corresponding edit box.
    • Deselecting Specified occurrence radio button grays out the corresponding edit box.
    • Corresponding edit box accepts only digits as input.
    • Corresponding edit box removes leading 0's if input is similar to 0001 and ends up as 1. Note that the change occurs on OK.
  • Case sensitive check box is checkable.
  • Above specified strings are treated as Regular Expression check box is checkable.
  • OK button saves changes and exits dialog.
    • OK button throws exception when String to find has not been specified.
    • OK button throws exception when Specified occurrence is selected and edit box is empty or contains 0.
  • Cancel button ignores changes and exits dialog.
  • X button ignores changes and exits dialog.
