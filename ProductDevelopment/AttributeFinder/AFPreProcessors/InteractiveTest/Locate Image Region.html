Text Dialog

  • Specify the text edit box is editable.
  • OK button saves changes and exits dialog.
    • Text entered appears in Define the clue text(s): list box.
    • Clue List # graphic changes from a red X to an almost filled blue circle.
  • Cancel button ignores changes and exits dialog.
  • X button ignores changes and exits dialog.
  • Highlighting an existing clue in the list box entry activates Remove and Modify.
  • Remove button displays a Confirm Delete dialog.
    • YES button deletes the clue text and exits dialog.
    • NO button ignores delete request and exits dialog.

Behavior of a File Name entry

  • Clear the Define the Clue Text(s) list box and specify a file name beginning with file:// inside the Text dialog and click the OK button.
    • Confirm that the Add button is still enabled.
    • Confirm that the Save List... button is still enabled.
    • Click Remove to delete the current file name entry and confirm that the Add and Save List... buttons are enabled.
  • Add several entries inside Define the Clue Text(s) list box (do not include a file name entry) and double click one of the entries to bring up the Text dialog. Modify the text in the edit box to a file name (including the file:// header) and click the OK button.
  • Confirm that the entry is added to the list.
  • Confirm that the Modify button displays a pre-populated Text dialog.
    • Changes made are persistent.
  • Graphical UP arrow button moves the selected item up in the list box.
  • Graphical DOWN arrow button moves the selected item down in the list box.
  • Graphical UP and DOWN arrow buttons become inactive when reaching top or bottom of the list.
  • Load List button opens a file open dialog and correctly loads the list.
  • Save List... button opens a file open dialog and correctly saves the list.
  • Case sensitive check box is checkable.
  • Above clues will be treated as regular expressions check box is checkable.
  • Restrict search to region defined by the results from clue list check box is inactive if there are no previously defined clue lists.
    • Restrict search to region defined by the results from clue list check box is active in a Clue List if a lesser numbered clue list contains clues.
