Behavior of a Filename entry

  • Clear the Define the replacement string pair(s) list box and click the Add button to add a file name beginning with file:// inside the Specify string to be replaced edit box of Enter Replacement String Pair dialog.
    • Confirm the Specify string to replace edit box is disabled.
    • Confirm that removing the file:// inside the Specify string to be replaced edit box will enable the Specify string to replace edit box.
    • Click the OK button and confirm that a message box is displayed to remind you to use ; as a separator followed by a delimiter.
  • Add ; at the end of the file name and a two-character delimiter after ; and click OK.
  • Confirm that a message box is displayed to remind you that only a single-character delimiter can be used.
  • Confirm that clicking OK on the message box will return to the Enter Replacement String Pair dialog and the delimiter will be selected for modification.
  • Change the delimiter to one character and click OK.
  • Confirm the file name with the delimiter becomes the first entry inside the list box.
  • Confirm the Add button is disabled.
  • Confirm the Save List... button is disabled.
  • Click Remove to delete this file name entry.
  • Confirm the Add button is enabled.
  • Confirm the Save List button is enabled.
  • Add several entries to the Define the replacement string pair(s) list box (do not include a file name entry) and double click one of the entries to bring up the Enter Replacement String Pair dialog.
  • Modify the text in the Specify string to be replaced edit box to a file name with delimiter (prefixed with file://) and click the OK button.
    • Confirm that a message box is displayed stating that the file name will be the only entry inside the list box.
    • [FlexIDSCore #3724] Confirm that the confirmation dialog is formatted properly and uses overwritten.
    • Confirm that clicking YES on the message box will clear all the other entries inside the list box and display the newly modified file name with delimiter as the first entry in the list box.
    • Confirm that clicking NO on the message box will do nothing to the list box.

Confirm Delete Dialog

  • Remove button displays Confirm Delete dialog.
  • YES button deletes the replacement string and exits dialog.
  • NO button ignores delete request and exits dialog.

Open Dialog

  • Load List button displays Open dialog.
  • Specify the file name: edit box is editable
    • Displays an error message when the file specified does not exist.
    • Displays an error message when a directory name is entered.
  • Specify a single character delimiter: edit box allows only one character and is editable.
  • Error message displayed if wrong delimiter is defined.
  • ... button brings up the standard Windows Open file dialog.
    • Selecting a file displays the file name in Specify the file name: edit box.
  • List box is populated correctly after loading from a file.

Save Dialog

  • Save List button displays Configuration message box when no replacements are specified.
  • Save List button displays Save dialog.
  • Specify the file name: edit box is editable.
  • Displays a confirmation message when the file specified already exists.
  • Specify a single character delimiter: edit box allows only one character and is editable.
  • ... button brings up the standard Windows Open file dialog.
  • Selecting a file displays the file name in Specify the file name: edit box.
