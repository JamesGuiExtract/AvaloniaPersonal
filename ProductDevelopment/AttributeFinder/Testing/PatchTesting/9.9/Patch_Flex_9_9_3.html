FLEX Index / ID Shield / LabDE 9.9.3 Patch Testing

Purpose and Details

  • Customer: TX - Midland DA
  • SCRs: 12451, 12467, 12470
  • Issues:
    • Redaction: Create redacted text failed for text files that begin with leading whitespace.
    • ImageFormatConverter.exe needs an option to force color preservation when converting from PDF to TIF.
    • Core: Rasterize PDF needs an option to Preserve color depth.
    • Pagination utility does not output a color document when color pages are included if the first page in the document is not in color.

Patch Tests

  1. Redaction: Create redacted text
    • Confirm that correct output is generated for a text file that begins with whitespace.
    • Confirm that correct output is generated for a text file that does not begin with whitespace.

  2. Image Format Converter
    • Confirm that the /color option forces color to be preserved when converting a PDF to a TIF.
    • Confirm that color is not preserved by default when converting a PDF to a TIF.
    • Confirm that the usage message includes the /color option.
    • NOT FIXED Confirm that the Help file reflects the change.

  3. Core: Rasterize PDF
    • Confirm that the property page includes a Preserve color depth option.
    • Confirm that the option is unchecked by default.
    • Confirm that the Preserve color depth option forces color to be preserved when converting a PDF to a TIF.
    • Confirm that color is not preserved by default when converting a PDF to a TIF.
    • NOT FIXED Confirm that the Help file reflects the change.

  4. Pagination
    • Confirm that a color document is produced from all colored input pages.
    • Confirm that a color document is produced from a mix of colored and uncolored input pages.

  5. Other
    • Exercise the Sanity Test - Release Testing checklist.
