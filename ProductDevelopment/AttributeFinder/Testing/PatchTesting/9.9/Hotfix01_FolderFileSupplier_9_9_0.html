Hotfix 01 Testing

Purpose and Details

  • Customers:
    • UW Transplant
    • PA - Allegheny
    • IN - Henry
    • LA - Iberville
    • CA - El Dorado
  • SCRs:
    • ISSUE-12988
    • ISSUE-10317
    • ISSUE-12923
  • Goal:
    • Create a hotfix to be deployed to current 9.9.0.48 customers.
  • Issues:
    • Listening mode of the Folder File Supplier occasionally misses one or more files in queuing.
    • Some files saved out of Pagination did not appear to be picked up by a watching file supplier.

Patch Tests

  • Problem Replication: ISSUE-12988
    • Copy the Testfiles folder from M:\PVCS_JIRA\JIRA\ISSUE-12988.
    • Create a local Test database with a Test action.
    • Open a Powershell command window as administrator.
    • Type Set-ExecutionPolicy RemoteSigned
    • Execute RunTest.ps1 from the Scripts folder.
      • It may be necessary to use the following argument on the command line.
      • RunTest.ps1 C:\Testfiles\Files if the test folder was copied to the root of C:\.
    • Desired Behavior
      • Powershell will create 5000 text files.
      • Listen.fps will queue the text files.
      • After a delay, DeleteFiles.fps will delete the files.
      • If any files remain in the Files folder, the test stops.
      • The test is successful if 10 iterations of the test succeed.

  • Problem Replication: ISSUE-12923
    • First Test
      • Copy 01000pages.tif from I:\Common\Testing\Test_Images\ManyPage Images.
      • Prepare a testing machine with folders for paginated images and processing results.
      • Create an FPS file to listen for paginated files.
      • Run the Pagination utility.
      • Desired Behavior
        • Divide the 1000-page source image into multiple smaller files.
        • Create output files such that several are created at a time.
        • Listen.fps will queue the paginated files.
        • Confirm that the number of paginated files is equal to the number of queued files.

    • Alternate Test
      • Copy single-page TIFs from I:\Common\Testing\Product Testing\IDShield_Sanity\SourceImages to a test folder.
      • Create an FPS file to listen for paginated files.
      • Run the Pagination utility.
      • Desired Behavior
        • Save each of the single-page source images.
        • Generate output files several at a time.
        • Listen.fps will queue the paginated files.
        • Confirm that the number of paginated files is equal to the number of queued files.

  • Patch Installation
    • Unzip FolderFileSupplier_9_9_0_Hotfix_1.zip.
    • Run Install.bat to update the Extract Systems files.

  • Testing
    • Repeat the steps from Problem Replication: ISSUE-12988.
    • Confirm that the test is successful.
    • Repeat the steps from Problem Replication: ISSUE-12923.
    • Confirm that the test is successful.
