  2. Licensing

    • Create license files.
      • Create a full RDT license file.
      • Create a FLEX Index Rule Writing license file using Simple Rules passwords.
      • Create an ID Shield Rule Writing license file using Simple Rules passwords.

      • Note that subsequent test cases should have only one active license file. The other files should be renamed so that they are not loaded.

    • RDT rules cannot be viewed.
      • With the full RDT license file, create an Indexing RSD file using a non-simple rule object like Locate Image Region. Include use of the Indexing counter.
      • Using only the FLEX Index Rule Writing license file, confirm that the Rule Set Editor displays an error message and does not open the above RSD file.

      • With the full RDT license file, create a Redaction RSD file using a non-simple rule object like Locate Image Region. Include use of the Redaction by Pages counter.
      • Using only the ID Shield Rule Writing license file, confirm that the Rule Set Editor displays an error message and does not open the above RSD file.

    • FLEX Index and ID Shield rules are not compatible.
      • With the FLEX Index Rule Writing license file, create an RSD file using one or more simple rule objects.
      • Confirm that the Indexing counter is enabled.
      • Using only the ID Shield Rule Writing license file, confirm that the Rule Set Editor will not open the above RSD file without error.

      • With the ID Shield Rule Writing license file, create an RSD file using one or more simple rule objects.
      • Confirm that the Redaction by Pages counter is enabled.
      • Using only the FLEX Index Rule Writing license file, confirm that the Rule Set Editor will not open the above RSD file without error.

    • Simple Rule Writing compatible with Master rules
      • Enable both the FLEX Index Rule Writing license file with Simple Rules passwords and the FLEX Index Server license file using Default passwords.
      • Confirm that the County.rsd.etf rules from the Flex Data Entry demo work to process the associated demo images.
      • Confirm that the Rule Set Editor will run but will not provide access to the Grantor-Grantee Finder Ver. 2 value finder.

      • Confirm that the Master.rsd.etf rules from the Redaction Demo work to process the associated demo images.
      • Confirm that the Rule Set Editor will run but will not provide access to the Find from RSD value finder.

  4. USB Key Requirements

    • The following tests should use only the specified license file. Other license files should be renamed so that they cannot be loaded.

    • Full RDT License
      • Confirm that the following USB counter choices are available on the Properties dialog.
        • FLEX Index - Indexing.
        • FLEX Index - Pagination.
        • ID Shield - Redaction (By Page).
        • ID Shield - Redaction (By Document).

      • Confirm that a rule set can be saved without specifying a USB key.
      • Save a simple RSD file using the Indexing counter.
      • Save a simple RSD file using the Pagination counter.
      • Save a simple RSD file using the Redaction (By Page) counter.
      • Save a simple RSD file using the Redaction (By Document) counter.

      • Save a simple RSD file using the Indexing counter with the internal use only flag.
      • Save a simple RSD file using the Redaction (By Page) counter with the internal use only flag.

      • Confirm that an error message is displayed when trying to open the Pagination counter RSD file created above.

      • Confirm that an error message is displayed when trying to open the Redaction (By Document) counter RSD file created above.

      • Open the RSD file using the Redaction (By Page) counter and the internal use only flag.

      • Open the new RSD file and confirm that the modifications were saved and that the internal use only flag remains both checked and disabled.

    • FLEX Index Rule Writing License
