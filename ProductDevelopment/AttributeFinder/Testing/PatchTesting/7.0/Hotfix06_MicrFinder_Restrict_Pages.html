FLEX Index / ID Shield 7.0 Hotfix 06 Testing

Purpose and Details

  • Customer: Old Republic
  • SCRs: FlexIDSCore #3522, #3523
  • Issue: Running the original MICR Finder rules on Old Republic images required approximately 90 minutes per document. The major part of this time is spent searching image pages for checks. The hotfix allows the MICR Finder to run on individual pages so that existence of meaningful clues will trigger the MICR Finder.
