FLEX Index / ID Shield 7.0 Hotfix 02 Testing

Purpose and Details

  • Customer: Old Republic
  • SCRs: LegacyRCAndUtils #5298
  • Issue: Chris had locally converted the Old Republic PDF images into TIFs to allow faster analysis. Many of the documents included both legal-size and letter-size pages. In the converted TIFs, the letter-size pages included a large black area at the top of the page when displayed in the Image Viewer. The OCR engine did not see the black areas so the spatial information for the text did not line up with the displayed image.
  • Solution: Code was added to Image Format Converter to require rechecking of page dimensions for each page instead of assuming that all pages had identical dimensions.
