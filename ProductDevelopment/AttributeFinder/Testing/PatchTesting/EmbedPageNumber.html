EmbedPageNumber Testing

Purpose and Details

  • Customer: CA - Contra Costa - AtPac
  • SCRs: DotNetRCAndUtils #355
  • Issue: Contra Costa needs a utility to modify a filename and include the count of pages for the image.
    1. The application shall run from the command line.
    2. The application shall change the filename if and only if:
      • The filename is exactly 200 characters in length.
      • The last 6 characters before the file extension are XXXXXX.
      • The file is a standard supported image type such as TIF, JPG, PDF, etc.
    3. If any of the above conditions are not met, then the application shall do nothing.
    4. If all of the above conditions are met, the application shall change the XXXXXX to nnnnnn to indicate the page count of the image.
    5. If all of the above conditions are met, but the application is unable to modify the filename, then a message shall be displayed and an exception shall be logged.
