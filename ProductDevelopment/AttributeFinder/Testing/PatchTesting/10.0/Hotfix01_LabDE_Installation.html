LabDE 10.0 Hotfix 01 Testing

Purpose and Details

  • Customer: UW Transplant
  • SCRs: ISSUE-12748
  • Issue: Installation of LabDE 10.0 on a 64-bit Windows 7 workstation caused an undesired automatic reboot when the workstation was already waiting for a reboot due to Windows Updates.

Hotfix Tests

  1. Test Setup
    • Prepare a clean test machine.
    • Apply some Windows Updates such that a reboot is requested.
    • Make a local copy of the LabDE 10.0.0.51 and SilentInstalls folders.
    • Update the associated files with the new versions from I:\users\William_Parr\Public\LabDE UW 10.0 Install Hotfix.

  2. Tests
    • Installation
      • Confirm that LabDE 10.0.0.51 installs properly.
        • No automatic restart should be done.
        • A reboot should be recommended after the install is complete.
      • Restart the test machine.

    • Functionality
      • Exercise the Demo_LabDE and confirm that expected results are seen.
      • Confirm that the appropriate secondary installs are correctly installed.
        • Clear Image PDK
        • Crystal Reports 2008
        • SQL Compact 3.5
        • SQL Server Native Client
      • Confirm that all Start Menu items work properly.
