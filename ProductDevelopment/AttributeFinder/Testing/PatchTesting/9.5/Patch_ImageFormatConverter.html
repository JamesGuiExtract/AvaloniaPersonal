FLEX Index / ID Shield 9.5 Patch Testing

Purpose and Details

  • Customer: UW Health
  • SCRs: LegacyRC #6461
  • Issue: Add support for removal of specified pages from a multi-page document.

Patch Tests

  1. Test Setup
    • Prepare a test machine with FLEX Index 9.5.0.35 or ID Shield 9.5.0.35 or LabDE 9.5.0.35.

  2. Tests
    • Installation
      • Confirm that the patch installs properly by adding ImageFormatConverter.exe to CommonComponents.

    • Functionality
      • Confirm that ImageFormatConverter.exe has an updated usage message.
      • Confirm that ImageFormatConverter supports the following /RemovePages options when modifying a TIF image.
        • PageNumber as /RemovePages 4
        • PageNumberRange as /RemovePages 4-6
        • CommaSeparatedPageList as /RemovePages 4,6,8
        • MinusFollowedByANumber as /RemovePages -2
      • Confirm that ImageFormatConverter supports the following /RemovePages options when modifying a PDF image.
        • PageNumber as /RemovePages 4
        • PageNumberRange as /RemovePages 4-6
        • CommaSeparatedPageList as /RemovePages 4,6,8
        • MinusFollowedByANumber as /RemovePages -2
      • Confirm that pages can be removed from a TIF image using the alternate method.
      • Confirm that pages can be removed from a PDF image using the alternate method.
