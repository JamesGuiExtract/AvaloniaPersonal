FLEX Index / ID Shield 9.7.3 Patch Testing

Purpose and Details

  • Customer: Two different customers
  • SCRs: 12034
  • Issues:
    • When performing a 9.7 installation on a clean machine, some COM objects are not completely registered.
      • Install ID Shield 9.7.0.31 on Server 2012.
      • Install LabDE 9.7.0.31 on Windows 7.
    • Exercising RegisterAll.bat is effective but should not be necessary.

  • SCRs: 12021
    • Installing via Launch.exe or silent install on Windows XP results in modal dialogs asking for privilege elevation for installation of external products.
    • Each installation should run without a request for elevation.

  • SCRs: 7255, 11981, 12039
    • Update make files for build to work against a branch.

  • SCRs: 12055
    • OH - Montgomery - Recorder has an ID Shield 9.7 verification machine that periodically crashes.
    • Steve was able to reproduce this crash by exercising Redaction: Verify sensitive data ML test #1 on Cheddar and on his development PC.
    • Wayne was unable to reproduce this error on any of several test machines.
    • Some code was found in FAMTagManager that was not thread-safe. This has been corrected. Steve can no longer reproduce the error with new code.
