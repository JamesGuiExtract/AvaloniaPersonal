FLEX Index / ID Shield 9.7.1 Patch Testing

Purpose and Details

  • Customer: Florida Supreme Court
  • SCRs: N/A
  • Issues:
    • No automatic cleanup of .Processed files if there is no .Redacted file.
    • Document deleted from SharePoint while being processed in the working folder.
    • Error using RedactNow or Verify button if more files are present in the Document Library than the List View Threshold.

Patch Tests

  1. Server Machine Setup
    • Install ID Shield 9.7.0.31.
    • Install and configure the standard SharePoint server software.
    • Install ID Shield for SharePoint 9.7.1.1.

  2. Client Machine Setup
    • Install ID Shield 9.7.0.31.
    • Install ID Shield for SharePoint 9.7.1.1.

  3. Tests
    • Installation
      • Confirm that the patch installs properly on the server.
      • Confirm that the patch installs properly on the client.

    • Automatic Cleanup
      • Create a document library.
      • Add some files to the library including several files that do not contain sensitive data.
      • Process the files.
      • Confirm that each file is processed.
      • Confirm that redacted output files are created for each sensitive file.
      • Confirm that redacted output files are not created for each non-sensitive file.
      • Confirm that all intermediate files are deleted for each file.
        • .uss
        • .voa
        • .Processed
        • .Redacted

    • Document Deletion
      • Create a document library.
      • Add a few documents to the library.
      • Process the documents to confirm that the workflow is correct.
      • Stop the FAM Service processing.
      • Add a few more documents to the library.
      • Confirm that status changes to Queued for Processing.
      • Delete all but one of the new files from SharePoint.
      • Start the FAM Service processing.
      • Confirm that the remaining new file is correctly processed.
      • Confirm that only one ELI31525 exception is logged for each deleted file.

    • List View Threshold
      • Create a document library.
      • Adjust the List View Threshold to the minimum 2000.
        • Open SharePoint Central Administration.
        • Select Application Management.
        • Select Manage web applications.
        • Select SharePoint - 80.
        • Select Resource Throttling under the General Settings dropdown menu.
        • Adjust List View Threshold and List View Threshold for Auditors and Administrators.
      • Add more than 2000 documents to the library.
      • Process some documents.
        • Confirm that RedactNow processes the selected file without error.
        • Confirm that Verify processes the selected file without error.
