FLEX Index / ID Shield Installation Testing
For Each Operating System

Each operating system shall be tested from a clean Clonezilla image or Virtual Machine.

  • FLEX Index Installation
    • Confirm that the following items are available from the Demo Shield install. Also confirm that an appropriate message is provided in the right-hand pane after a mouse-over event on the product name.
      • Microsoft .NET 3.5
      • Install FLEX Index
      • SQL Server Applications
        • Windows Installer 4.5
        • SQL Server 2008
        • SQL Management Studio
      • Extract Systems License Manager

  • FLEX Index Start Menu
    • Confirm that the following items are available and functional from the Extract Systems FLEX Index Start menu.
      • Utilities
        • Configure Email Settings
        • Convert FAM Database
        • Convert FPS File
        • Detect and Report Failure
        • Feedback Manager
        • Image Viewer
        • Rule Tester
        • USS File Viewer
        • View Exception Log
        • VOA File Viewer
      • DB Administration
      • File Action Manager
      • FLEX Index Help
      • USB License Key Manager
      • User License

  • ID Shield Installation
    • Confirm that the following items are available from the Demo Shield install. Also confirm that an appropriate message is provided in the right-hand pane after a mouse-over event on the product name.
      • Microsoft .NET 3.5
      • Install ID Shield
      • SQL Server Applications
        • Windows Installer 4.5
        • SQL Server 2008
        • SQL Management Studio
      • Extract Systems License Manager

  • ID Shield Start Menu
    • Confirm that the following items are available and functional from the Extract Systems ID Shield Start menu.
      • Utilities
        • Configure Email Settings
        • Convert FAM Database
        • Convert FPS File
        • Detect and Report Failure
        • Image Viewer
        • Rule Tester
        • USS File Viewer
        • View Exception Log
        • VOA File Viewer
      • DB Administration
      • File Action Manager
      • ID Shield Help
      • USB License Key Manager
      • User License

  • Extract Systems LM Start Menu
    • Confirm that the following items are available and functional from the Extract Systems LM Start menu.
      • File Activation Utility
      • USB License Key Manager

  • SQL Server Installation
    • Confirm that each of the following items are available and can be installed from the Demo Shield install.
      NOTE: SQL Server 2008 is not supported for 64-bit Windows XP.
      • SQL Server Applications
        • Windows Installer 4.5
        • SQL Server 2008
        • SQL Management Studio

  • FLEX Index RDT Start Menu
    • Confirm that the following items are available and functional from the Extract Systems FLEX Index RDT Start menu.
      • Automated Tests
        • Run All Tests
      • FLEX Index SDK Help
      • FLEX Index UI Help
      • Image Cleanup Settings Editor
      • Index Converter
      • RDT Configuration
      • Rule Set Editor
      • Test Harness

  • Uninstalls - each operating system
    • With ID Shield and Extract Systems LM installed, confirm that Extract Systems LM uninstalls without error and that ID Shield still runs.
    • With FLEX Index and Extract Systems LM installed, confirm that Extract Systems LM uninstalls without error and that FLEX Index still runs.

  • Uninstalls - each non-Server operating system
    • With ID Shield and RDT installed, confirm that ID Shield cannot be uninstalled.
    • Uninstall RDT.
    • Confirm that ID Shield still runs.
    • Uninstall ID Shield.

    • With FLEX Index and RDT installed, confirm that FLEX Index cannot be uninstalled.
    • Uninstall RDT.
    • Confirm that FLEX Index still runs.
    • Uninstall FLEX Index.
    • Confirm that the ESFAMProcess service is uninstalled.
