FLEX Index / ID Shield Installation Testing
For Each Operating System

Each operating system shall be tested from a clean Clonezilla image or Virtual Machine.

The .NET and SQL Server installations can be exercised from either the FLEX Index or the ID Shield DemoShield application.

  • .NET 3.5 and 4.0 Installation
    • Confirm that the Demo Shield text fields do not overlap when selecting other items.
    • Confirm that the installation finishes without error.

  • SQL Server Installation
    • Confirm that each of the following items are available and can be installed from the Demo Shield install.
      • NOTE: SQL Server 2008 R2 is not supported for 64-bit Windows XP.
      • SQL Server Applications
        • Windows Installer 4.5
        • SQL Server 2008
        • SQL Management Studio

  • Create a license file for FLEX Index and ID Shield.
  • Create a local user on the test machine. Do not add the user to the Administrators group.
  • Add the new user to the Remote Desktop Users group.

  • FLEX Index Installation
    • [FlexIDSCore #3963] Confirm that the Demo Shield text fields do not overlap when selecting other items.
    • [FlexIDSCore #4174] Confirm that elevated permissions are requested during installation of Extract Systems software if administrator is not the logged in user.
    • [FlexIDSCore #4737] Note that continuing to install FLEX Index as a non-administrator may lead to issues during uninstallation.
    • [FlexIDSCore #4659] FKB installer
      • Confirm that the ComponentData rules are installed to a named subfolder such as Extract Systems\FlexIndexComponents\ComponentData\11.1.0.123.
      • Confirm that the FKB version and install directory includes a major version, minor version, a patch number, and build number.

    • Confirm that the following items are available from the Demo Shield install. Also confirm that an appropriate message is provided in the right-hand pane after a mouse-over event on the product name.
      • Microsoft .NET 3.5 and 4.0
      • SQL Server Applications
        • Windows Installer 4.5
        • SQL Server 2008
        • SQL Management Studio
      • Install FLEX Index
      • Extract Systems License Manager

    • Confirm that copyright dates include the appropriate year for the following
      • FLEX Index installation splash screen
      • FLEX Index installation licensing page
    • Confirm that FLEX Index can be installed without error.

    • [FlexIDSCore #4779] Install FLEX Index in Custom mode and install redistributable components. Confirm that UCLIDInputFunnel cannot be installed.

    • Confirm via the Control Panel that the following external programs have been installed along with FLEX Index.
      • Crystal Reports 2008 Runtime SP1
      • Microsoft SQL Server Compact 3.5 SP2
    • Confirm that the following files have been installed in the CommonComponents folder.
      • DataEntryApplication.FlexIndex.resources

  • Uninstalls
    • Login as a user that is a member of the Administrators group.
    • Confirm that RDT, FLEX Index, ID Shield and Extract Systems LM are installed.
    • Confirm that the appropriate FKB version is also installed.
    • Confirm that FLEX Index cannot be uninstalled.
    • Confirm that ID Shield cannot be uninstalled.
    • Confirm that the FKB update uninstalls without error.
    • Confirm that RDT uninstalls without error.
    • [LegacyRC #3458] Confirm that Crystal Reports is not uninstalled as part of the RDT uninstall.
    • With FLEX Index, ID Shield and Extract Systems LM installed, confirm that Extract Systems LM uninstalls without error.
      • Confirm that DB Administration still runs.
      • Confirm that File Action Manager still runs.
      • Confirm that USB License Key Manager still runs.

    • If necessary, create and use an evaluation license for ID Shield. Any non-expiring licenses should be renamed or deleted.
    • Confirm that DB Administration still runs.
    • Confirm that ExtractTRP2.exe is running.
    • Close DB Administration.
    • Uninstall ID Shield.
      • [FlexIDSCore #3163] Confirm that ExtractTRP2.exe is killed before the uninstall occurs.
      • Confirm that a prompt is presented about uninstalling Crystal Reports.
      • Confirm that a prompt is presented about uninstalling SQL Compact Edition.

    • [LegacyRC #3616] Confirm that FLEX Index Help is still available from the Start menu.
    • Confirm that the Help file opens.
    • Confirm that File Action Manager still runs.
    • Uninstall FLEX Index.
      • Confirm that a prompt is presented about uninstalling Crystal Reports.
      • Confirm that a prompt is presented about uninstalling SQL Compact Edition.
    • Confirm that when you uninstall all Extract Systems software, all Extract Systems services (FAM service, FDRS service, exception service) are uninstalled.
