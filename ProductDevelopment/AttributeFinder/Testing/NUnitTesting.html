NUnit Testing - Automated and Interactive

Test Machine Setup

  • Confirm that the test machine has the RDT installed.
  • Copy the Nunit-2.5.8.10295.msi file from \\lofton\installs\NUnit to the test machine.
  • Install NUnit, selecting the Complete option.
  • Modify the Start Menu shortcut to use NUnit-x86.exe since Extract Systems software is 32-bit software.
  • Add the following NUnit DLLs to the GAC. The DLLs can be found at C:\Program Files (x86)\NUnit 2.5.8\bin\net-2.0\framework. The GAC folder is C:\Windows\assembly.
    • nunit.framework.dll
    • nunit.mocks.dll

Test Execution

  • From the Categories tab, select the Interactive category and Add it to the Selected Categories group.
  • Also, check the Exclude these categories checkbox.
  • Load the comprehensive Extract-All.nunit testing project from CommonComponents.
  • Confirm that the project loads without error.
  • Exercise the automated tests. Each is expected to complete without error.
