Simple Rule Writing Testing

Goals

  • Exercise installation and licensing for Simple Rule Writing.
  • Confirm that every simple rule object is available.
  • Confirm that every advanced rule object is unavailable.
  • Confirm that the current customer delivery rules can be read, if that is intended for a specific customer.
  • Confirm that complex rules cannot be read.
  • Confirm that complex rules can still be opened via Rule Tester and exercised via File Action Manager.
