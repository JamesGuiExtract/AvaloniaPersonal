  2. Three Distinct Objects For ID Shield Operation (#9)

    • SCRs: TBD
    • Details: Separate ID Shield operations such that each of three objects performs a unique task. The Verify images task will take a VOA file as input and create a VOA file as output. The Create redacted image task will take a VOA file as input and create a TIF or PDF file as output. The Create metadata file task will take a VOA file as input and create an XML file as output.
    • Main Checklists Updated: TBD
    • Tested By:
      • TBD, 7.1.0.xx, MM/DD/YYYY
    • Testing:
      • Confirm that UI behavior is correct for each of the property pages.
      • Confirm that settings are translated properly from older ID Shield FPS files.
      • Use one or more of the three ID Shield tasks to confirm proper functionality for the following scenarios:
        • Auto-redaction creates an output image.
        • Auto-redaction creates an output image and a metadata file.
        • Redaction verification creates an output image.
        • Redaction verification creates an output image and a metadata file.
        • Redaction verification creates an output VOA file.
      • Confirm that XML output format from automated redaction matches the XML output format from redaction verification.
