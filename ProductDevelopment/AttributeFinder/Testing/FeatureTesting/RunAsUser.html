FLEX Index / ID Shield Feature Testing
Run As User

  1. Goals and Setup

    • Install ID Shield as local admin.
    • Create a user account that does not have administrator privileges.
    • Confirm that basic FLEX Index and ID Shield functions work as intended.

  2. SQL Server

    • [LegacyRC #6180] dbo schema issues
      • As local admin, create an empty database in SQL Management Studio.
      • Grant db_owner permissions for the database to the FAM Admins group.
      • Log in as a member of that group and try to log in to the database using DB Admin.
      • Confirm that DB Admin is able to create all the appropriate tables.

  3. UAC and FAM Utilities

    • [LegacyRC #5902, #6037, #6038, #6039, #6040, #6041]
      • Confirm that the logged-in user does not have administrator privileges.
      • Open the UEX Viewer.
      • Confirm that the error log opens without error.
      • Confirm that the Automatically close window checkbox acts and persists as expected.
      • Confirm that USB License Key Manager allows the non-privileged user to set and persist the License Server.
        • This works when utility is run as administrator.
        • This does not work when utility is run as non-administrator.
      • Confirm that Configure Email Settings can be exercised without error.
      • Exercise Compute.fps from the ID Shield demo.
      • Confirm that files process without error.
      • Confirm that processing (while listening) can be stopped without error.
      • Confirm that a TIF image can be double-clicked to open it in the legacy Image Viewer.
      • Confirm that the image can be OCR'd.
      • Confirm that the same image can be opened and OCR'd in ExtractImageViewer.exe.

    • [LegacyRC #6043] FDRS
      • Run FDRS as an unprivileged user.
      • Confirm that settings are read from and written to an INI file in the common application data folder.
      • Confirm that settings can be modified via the Configure > Automatic notifications... context menu option.

    • [LegacyRC #6044] Cleanup Image
      • As an unprivileged user, confirm that image cleanup can be run successfully from the command line.
        • An Inlite Image Cleanup key will need to be plugged in to a USB port.

    • [LegacyRC #6046] Test Harness
      • As an unprivileged user, run all the automated tests and confirm that they work as expected.

  4. Miscellaneous

    • [LegacyRC #5890] UEX log permissions
      • Log in as local admin to a machine with UAC enabled; remove any existing UEX logs.
      • Do something that will cause ExtractException.uex to be created.
      • Log off and log back in as an unprivileged user.
      • Confirm that you have access to the UEX log and exceptions are logged correctly.
      • Repeat the test in the other direction (i.e., confirm that local admin has access to a log created by another account).
