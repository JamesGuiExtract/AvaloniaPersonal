  2. Slide Show (#3, #18)

    • SCRs: FlexIDSCore #4470, #4475, #4477, #4478, #4479, #4480, #4482, #4483, #4484, #4521, #4528, #4572, #4573, #4587, #4592, #4596, #4607, #4616, #4635
    • Details: A slideshow feature has been added for redaction verfication which will automatically advance document pages after a specified number of seconds. The slideshow shall automatically pause upon user interaction, but resume as soon as the current document is saved (or the user unpauses). The user will be able to specify the number of seconds each page should display (default will be 4) and whether the slideshow should start automatically. The redaction verificaton task will have configuration settings that determine whether the slideshow feature is to be available, to cause it to be automatically paused on certain documents, and to allow for tags or file action status to be set for document which were auto-advanced.
    • Main Checklists Updated: TBD
    • Tested By:
      • TBD, 9.0.0.xx, MM/DD/YYYY
    • Testing:
      • Confirm that the slideshow works in an intuitive way.
      • Confirm that the slideshow is not available if the task setting is disabled.
      • Confirm proper behavior of user alertness using a run key.
      • Confirm proper behavior of user alertness using a random question.
      • Confirm that the selected document tag is applied when a document is automatically advanced.
      • Confirm that the selected file action status is applied when a document is automatically advanced.
      • Confirm that the slideshow is automatically stopped when the selected condition is met.
      • Confirm that the slideshow is automatically stopped when the screen saver is activated.
