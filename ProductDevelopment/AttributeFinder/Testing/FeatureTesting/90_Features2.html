FLEX Index / ID Shield 9.0 Features Guide

  • Purpose

    • To test individual features as well as combinations of features.
    • To allow for individual experimentation during feature testing.
    • To document the tested aspects of various features.
    • Numbered items reference Arvind's 9.0 wish list email from 06/07/2011.
    • Additional numbered items are defined in 90_Testing_Progress.xls located in the I:\Common\Testing\Product Testing\FLEX_Index_90 folder.

    1. Document Classifier - (#16 A)
      • Confirm that document classification no longer searches Probable and Maybe rules if a Sure rule was triggered.

    2. Rule Set Editor Enhancements - (#16 BCDFG)
      • Confirm that hyphenated serial number ranges are accepted for USB keys.
      • Confirm that an FKB update can be specified for use with a ruleset.
      • Confirm that multiple rulesets with different FKB requirements can run successfully on the same machine.
      • Confirm that a new column exists allowing generated exceptions to just be logged.
      • Confirm that rule sets containing the following rule objects can be run but cannot be saved.
        • Entity Name Splitter
        • Grantor-grantee Finder V2
        • Legal Description Finder
        • Legal Description Splitter

    3. Support for Fuzzy Regular Expressions - (#16 E)
      • Confirm that fuzzy regular expressions are supported for all appropriate rule objects.

      • Confirm that Locate image region supports fractional inches in the region bounds.
