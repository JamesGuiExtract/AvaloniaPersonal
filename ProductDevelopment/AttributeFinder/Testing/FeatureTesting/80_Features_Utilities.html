  7. Split Multi-Page Image

    • SCRs: LegacyRC #5519
    • Details: This utility shall divide either a folder of TIF images or a specific TIF image into associated single-page images. Single page files shall be named using a three-digit file extension such as 123.001, 123.002, etc. The single required command-line argument is either a folder name or a file name. Optional arguments include:
      • /ef <LogFile> - to write exceptions to the specified log file
      • /d - to delete the multi-page image after splitting pages
      • /o - to overwrite the output files if necessary
      Annotations should be retained if annotation support is licensed. Failure during processing of a single image should retain any previously split pages. Failure during processing of a folder of images should retain previously split images.
    • Main Checklists Updated: Yes
