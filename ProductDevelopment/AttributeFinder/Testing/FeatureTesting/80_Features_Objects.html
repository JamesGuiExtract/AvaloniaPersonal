  5. PDF/A Converter

    • SCRs: LegacyRC #5528
    • Details: Modify the Core: Convert to searchable PDF task to include an option to create PDF/A-compliant output files. Adobe Preflight or a similar validation utility should be used to examine the output images.
    • Main Checklists Updated: TBD
    • Tested By:
      • TBD
    • Testing:
      • User Interface
        • Confirm that the property page includes a Make output PDF/A compatible (this will increase the size of the PDF) check box.
        • Confirm that the check box is unchecked by default.
        • Confirm that the task settings are persistent.

      • Basic Testing
        • Confirm that TIF images can be converted to searchable PDFs without being PDF/A-compliant.
        • Confirm that output filename is the input filename less the file extension plus PDF.
        • Confirm that the input TIF images are deleted.

        • Confirm that TIF images can be converted to searchable PDFs with being PDF/A-compliant.
        • Confirm that output filename is the input filename less the file extension plus PDF.
        • Confirm that the input TIF images are deleted.

        • Confirm that PDF images can be converted to searchable PDFs without being PDF/A-compliant.
        • Confirm that output filename is the same as the input filename.

        • Confirm that PDF images can be converted to searchable PDFs with being PDF/A-compliant.
        • Confirm that output filename is the same as the input filename.

      • Large-Scale Test
        • Prepare at least 1M image files for this test. Include single-page and multiple-page images. Include TIF and PDF images.
        • Prepare an FPS file to queue and process these images:
          • Use the Core: Convert to searchable PDF task.
          • Include a listening file supplier in order to confirm that added files can be converted to searchable PDF/A format.
        • Process the images.
        • Spot check a random subset of the output files and confirm that they are PDF/A-compliant.
        • Check at least 1,000 files if the check must be done manually.
        • Check at least 50,000 files if the check can be done automatically.
        • Make sure to check some of each of the following files:
          • Single-page TIF
          • Multiple-page TIF
          • Single-page PDF
          • Multiple-page PDF
