  6. Character Confidence Based Rule Objects

    • SCRs: FlexIDSCore #3936, #3937, #3938
    • Details: Create three new rule objects that allow rule writers to exploit character confidence provided by the OCR engine. An AFCondition object shall return true or false depending on evaluating the characters in the entire document. An Attribute Selector object will be created to support using a data score criterion. This also allows the Entity Name Data Scorer to be used by a selector. A Data Scorer object will allow selection of attributes (typically for removal) based on their character confidence.
