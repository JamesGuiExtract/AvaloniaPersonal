FLEX Index / ID Shield 8.0 Features Guide

  • Purpose

    • To test individual features as well as combinations of features.
    • To allow for individual experimentation during feature testing.
    • To document the tested aspects of various features.
    • Numbered items reference Arvind's Updated 8.0 feature list email from 01/26/2010.

  • Numbered Features

    1. Fuzzy Text Search - (#01)
      • Independent testing of fast method
      • Independent testing of better_fit method
      • Independent testing of error parameter
      • Independent testing of xtra_ws parameter
      • General independent testing

      • Review automated test results for Regular Expression Value Finder since comprehensive fuzzy tests are included.
      • Review automated test results for other rule objects that support regular expressions since basic fuzzy tests should be included.

    2. FAM Service - (#02)
      • Exercise configuration of SQL Server 2005 for use with FAM Service - see separate Instructions checklist for details.
      • Exercise configuration of SQL Server 2008 for use with FAM Service - see separate Instructions checklist for details.
      • Exercise the FAM Service checklist.
      • Exercise the FAM service in combination with one or more of the following features. Confirm either that the service works properly or that a meaningful exception is logged.
        • Scheduled processing
        • User authentication
        • File processing priority
        • PDF/A Conversion
        • Other, tester defined feature or combination

    3. Tagging Files - (#03)
      • Exercise Manage Tags from within DB Admin.
      • Exercise application of tags from within verification.
      • Confirm that new tags can be created during verification based on AllowDynamicTagCreation.
      • Exercise the following reports that provide tag information
        • Detailed report for specific document
        • ID Shield detailed report for specific document
        • List of all documents with specific tag
        • List of all tagged documents in the project (by tag)
        • List of all tags in the project
      • Document the details of additional independent testing of this feature.

    4. Skipping Files - (#04)
      • Exercise skipping of files from within verification of sensitive data.
        • Are files properly skipped?
        • Is verification history updated as expected?
        • Is the FAM status bar updated as expected?
        • Is the DB Admin Summary tab status bar updated as expected?
      • Exercise skipping of files from within verification of extracted data.
        • Are files properly skipped?
        • Is the FAM status bar updated as expected?
        • Is the DB Admin Summary tab status bar updated as expected?
      • Confirm that processing of skipped files from the FAM works as expected for files skipped by the user and by any user.
      • Confirm that the Select Files dialog within DB Admin works as expected to operate on files currently in Skipped status.
      • Confirm that the Select Files dialog within DB Admin works as expected to set files to Skipped status.
      • Document the details of additional independent testing of this feature.

    5. Mathematical FAM Condition - (#05)
      • Confirm that the object's property page works as expected.
      • Confirm that Match based upon mathematical condition works properly for the following situations
        • met versus unmet
        • randomly as a percentage
        • once every N times
        • based on File ID modulus for files already in the database
      • Document the details of additional independent testing of this feature.

      • Test use of forms-based processing where a previously created VOA is used to redact a fixed location on multiple documents.
      • Document the details of additional independent testing of this feature.
