NOTE: Items prefaced with an asterisk indicate functionality that is not yet complete.

  1. Install Options

    • Confirm that the user cannot continue with the installation if they do not agree with the terms of the license.
    • Confirm that a FLEX Index Custom SDK installation allows installation of the SDK Redistribution Components to the \FlexIndex\Redist folder.
      • DataEntry.msm
      • ExtractCommonMM.msm
      • ExtractFlexCommonMM.msm
      • UCLIDFlexIndex.msm
    • Confirm that a FLEX Index Compact SDK installation does not install Tutorials or Help.

    • Confirm that a newer version of FLEX Index cannot be installed on a test machine with an older version of any Extract Systems software.
    • Confirm that a newer version of ID Shield cannot be installed on a test machine with an older version of any Extract Systems software.
    • Confirm that a newer version of Extract Systems LM cannot be installed on a test machine with an older version of any Extract Systems software.

  5. Individually-Licensed Items

    • Create Searchable PDF
      • Use the COM License Generator utility to create a license which includes the Create Searchable PDF package. Copy this file to the LicenseFiles folder.
      • Confirm that the Core: Convert to searchable PDF task is available in the File Action Manager.
      • Use this task to convert some redacted TIF images, and confirm that the redacted output image is also available in PDF format.

    • Disable USB Key Serial Numbers
      • Use the COM License Generator utility to create a license which includes the Disable USB Key Serial Numbers package. Copy this file to the LicenseFiles folder.
      • Confirm that a customer rules set that is locked to a specific serial number can be run with a different USB Key.

    • Disable USB Counters (locked to PC) DEVELOPER MUST PERFORM THIS TEST!!!
      • Use the COM License Generator utility to create a license which includes the Disable USB Counters (locked to PC) package. Copy this file to the LicenseFiles folder.
      • Confirm that a customer rules set that is locked to a specific serial number can be run without a USB key.

    • FTP/SFTP File Transfer
      • Use the COM License Generator utility to create a license which includes the FTP/SFTP file transfer package. Copy this file to the LicenseFiles folder.
      • Confirm that the Core: Transfer file via FTP/SFTP task is available in the File Action Manager.
      • Confirm that files can be successfully transferred.

    • Handwriting Recognition
      • Use the COM License Generator utility to create a license which includes the Handwriting Recognition package. Copy this file to the LicenseFiles folder.
      • Confirm that the Find SSNs within image area value modifier is available in the Rule Set Editor.
      • Exercise the associated automated tests and confirm that the tests are successful.

    • Image Cleanup Engine
      • Use the COM License Generator utility to create a license which includes the Image Cleanup Engine package. Copy this file to the LicenseFiles folder.
      • Confirm that the Core: Cleanup image task is available in the File Action Manager.
      • Use this task to clean some images ( see the image files associated with the automated test ), and confirm that the output images are cleaner.

    • MICR Finding Engine
      • Use the COM License Generator utility to create a license which includes the MICR Finding Engine package. Copy this file to the LicenseFiles folder.
      • Also license ID Shield Server but not RDT.
      • Exercise the St Johns rules or another rule set that uses the MICR Finder.
      • Confirm that rules run and that appropriate redacted images are created.
      • Confirm that without a MICR Finding Engine license, the rules do not run and that appropriate redacted images are not created.

    • Modify PDF (Pegasus PDF Annotation Removal)
      • Use the COM License Generator utility to create a license which includes the Modify PDF (Pegasus PDF Annotation Removal) package. Copy this file to the LicenseFiles folder.
      • Confirm that the Core: Modify PDF file task is licensed.
      • Confirm that annotations can be removed from PDF documents.

    • PDF Read / Write
      • Use the COM License Generator utility to create a license which includes the PDF Read + Write package. Copy this file to the LicenseFiles folder.
      • Confirm that PDF documents can be opened in the Spot Recognition Window.
      • Confirm that PDF documents can be redacted in ID Shield, and that the redacted output image is also in PDF format.
