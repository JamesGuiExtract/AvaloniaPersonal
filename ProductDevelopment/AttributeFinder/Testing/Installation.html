General Installation Testing Checklist

NOTE: Items prefaced with an asterisk indicate functionality that is not yet complete.

  1. Install Options

    • Confirm that not agreeing with the terms of the license agreement halts the installation with no files copied to the system.
    • Confirm that a Custom SDK installation allows installation of the SDK Redistribution Components.
      • Confirm that Merge Modules for FlexIndexSDK and InputFunnel are copied to the \FlexIndex\Redist folder.
    • Confirm that a Compact SDK installation does not install Tutorials or Help.
    • [FlexIDSCore #2892] Confirm that installing FLEX Index provides a meaningful splash screen.

    • [FlexIDSCore #2521] Confirm that installing FLEX Index on a freshly ghosted test machine does not first require installation of .NET.
    • [FlexIDSCore #2522] Confirm that installing ID Shield on a freshly ghosted test machine does not first require installation of .NET.

    • [FlexIDSCore #2856] Confirm that a newer version of FLEX Index cannot be installed on a test machine with an older version of Extract Systems LM.
    • [FlexIDSCore #2856] Confirm that a newer version of ID Shield cannot be installed on a test machine with an older version of Extract Systems LM.
    • [LegacyRC #4663] Confirm that a newer version of FLEX Index cannot be installed on a test machine with an older version of ID Shield.
    • [LegacyRC #4663] Confirm that a newer version of Extract Systems LM cannot be installed on a test machine with an older version of ID Shield.
    • [LegacyRC #4663] Confirm that a newer version of ID Shield cannot be installed on a test machine with an older version of FLEX Index.
    • [LegacyRC #4663] Confirm that a newer version of Extract Systems LM cannot be installed on a test machine with an older version of FLEX Index.

    • *Confirm that an upgrade installation of FLEX Index is available and effective after installation of the previous version.
    • *Confirm that an upgrade installation of ID Shield is available and effective after installation of the previous version.

    • [FlexIDSCore #3151] Cannot Install Software If Not Administrator
      • Confirm that the installation process exits gracefully if the logged in user is not a member of the Administrators group.

    • [FlexIDSCore #3573] Windows version >= XP
      • Prepare a test machine with an unsupported operating system such as Windows 2000.
      • Confirm that FLEX Index cannot be installed.
      • Confirm that ID Shield cannot be installed.
      • Confirm that Extract Systems LM cannot be installed.

  5. Individually-Licensed Items

    • Create Searchable PDF
      • Use the COM License Generator utility to create a license which includes the Create Searchable PDF package. Copy this file to the CommonComponents folder.
      • Confirm that the Convert to searchable PDF task is available in the File Action Manager.
      • Use this task to convert some redacted TIF images, and confirm that the redacted output image is also available in PDF format.

    • Disable USB Key Serial Numbers
      • Use the COM License Generator utility to create a license which includes the Disable USB Key Serial Numbers package. Copy this file to the CommonComponents folder.
      • Confirm that a customer rules set that is locked to a specific serial number can be run with a different USB Key.

    • Disable USB Counters (locked to PC) WAYNE MUST PERFORM THIS TEST!!!
      • Use the COM License Generator utility to create a license which includes the Disable USB Counters (locked to PC) package. Copy this file to the CommonComponents folder.
      • Confirm that a customer rules set that is locked to a specific serial number can be run without a USB key.

    • Handwriting Recognition
      • Use the COM License Generator utility to create a license which includes the Handwriting Recognition package. Copy this file to the CommonComponents folder.
      • Confirm that the Find SSNs within image area value modifier is available in the Rule Set Editor.
      • Exercise the associated automated tests and confirm that the tests are successful.

    • Image Cleanup Engine
      • Use the COM License Generator utility to create a license which includes the Image Cleanup Engine package. Copy this file to the CommonComponents folder.
      • Confirm that the Cleanup image task is available in the File Action Manager.
      • Use this task to clean some images ( see the image files associated with the automated test ), and confirm that the output images are cleaner.

    • MICR Finding Engine
      • Use the COM License Generator utility to create a license which includes the MICR Finding Engine package. Copy this file to the CommonComponents folder.
      • Also license ID Shield Server but not RDT.
      • Exercise the St Johns rules or another rule set that uses the MICR Finder.
      • Confirm that rules run and that appropriate redacted images are created.
      • Confirm that without a MICR Finding Engine license, the rules do not run and that appropriate redacted images are not created.

    • PDF Read / Write
      • Use the COM License Generator utility to create a license which includes the PDF Read + Write package. Copy this file to the CommonComponents folder.
      • Confirm that PDF documents can be opened in the Spot Recognition Window.
      • Confirm that PDF documents can be redacted in ID Shield, and that the redacted output image is also in PDF format.
