Remote Desktop Services Testing

Test Machine Setup

  • Confirm that the test server has a server operating system installed - usually Server 2008 R2.
  • Open Server Manager.
  • Right-click on the Roles item and select Add Role.
  • Select Remote Desktop Services with the following options.
    • Remote Desktop Session Host
    • Remote Desktop Licensing
    • Network Level Authentication = No

    • Note that 1 error for "The computer account for the Remote Desktop license server cannot be added to the Terminal Server License Servers group in Active Directory Domain Services because access is denied." is expected.
  • Install and license ID Shield and SQL Server.
  • Configure the RemoteApp Manager. Click on the Add RemoteApp Programs item in the Actions window.
  • Use the RemoteApp Wizard to select the following Extract Systems software applications to support client-side verification.
    • DB Administration
    • File Action Manager
    • Rule Tester
    • View Exception Log
  • Copy a suitable ID Shield folder and prepare files for verification.
