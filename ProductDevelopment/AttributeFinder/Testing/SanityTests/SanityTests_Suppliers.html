FAM Sanity Test - File Suppliers

*FTP file supplying must be specially licensed for this test.

Goals

  • Exercise each defined file supplier.
  • Confirm that various configurations work as expected.
  • Confirm that no unwanted exceptions are logged.
  • Confirm that each file supplier works properly on various operating systems.

File Suppliers

  1. Files from context menu
    • Confirm that the configuration dialog can be displayed and that settings are persistent.
    • Confirm that files of a specified type can be supplied via context menu.
    • Confirm that files of a specified type can be supplied via context menu only from a specified path.

  2. Files from dynamic list
    • Confirm that the configuration dialog can be displayed and that settings are persistent.
    • Confirm that files can be supplied via a list specified with an absolute path.
    • Confirm that files can be supplied via a list specified using expandable tags.

  3. Files from folder
    • Confirm that the configuration dialog can be displayed and that settings are persistent.
    • Confirm that files can be supplied recursively from a specified folder.
    • Confirm that files that are added to a folder can be supplied.

  4. Files from FTP site
    • Confirm that the configuration dialog can be displayed and that settings are persistent.
    • Confirm that files can be supplied from the ExtractTest FTP account.
      • Exercise recursive supplying.
      • Exercise periodic polling.
      • Exercise polling at specified times.

  5. Files from static list
    • Confirm that the configuration dialog can be displayed and that settings are persistent.
    • Confirm that individual files from different folders can be selected and are properly queued.
    • Confirm that the Open dialog allows multiple selection.
    • Confirm that selected items can be removed.
    • Confirm that the list can be cleared.
    • Confirm that the list can be populated via Load List.
