FAM Sanity Test - Internal Use

* ID Shield, LabDE and FLEX Index should be installed and licensed for this test.
* RDT should be installed and separately licensed for this test.

FLEX Index - Demo

  • Copy the Demo_FlexIndex folder to the test machine.
  • Run Clean All.bat as local administrator.
  • Use SQL Management Studio to grant db_datareader and db_datawriter privileges to Domain Users for the Demo_FLEXIndex database.
  • Exercise ExtractData.fps.
    • Confirm that all images are processed.
    • Confirm that no exceptions are thrown during rule execution.
  • Exercise VerifyData.fps.
    • Confirm that appropriate data are found for each image.
    • Confirm that each field type can be swiped using the angular redaction tool.
    • Confirm that each field type can be swiped using the rectangular redaction tool.
    • Confirm that each field type can be swiped using the word redaction tool.
  • Confirm that verification and swiping require only a FLEX Index Client license.
