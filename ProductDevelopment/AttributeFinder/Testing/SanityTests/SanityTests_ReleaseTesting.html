FAM Sanity Test - Release Testing

Goals

  • Exercise fundamental software behavior before a version release.
  • Confirm that software installation works as expected.
  • Confirm that for each Start Menu item, the expected application starts.
  • Confirm that queuing and processing work correctly for each demo.
  • Confirm that verification works correctly for each demo.

Installation

  • Confirm that Extract Systems software installs correctly on a clean VM or test machine.
  • Confirm that Extract Systems software installs correctly as a version upgrade.
  • Confirm that software installs correctly via Silent Install.
  • Confirm that software installs correctly via Launch.exe.
  • Confirm that SQL Server Express and Extract Systems LM each install correctly.

Start Menu

  • Confirm that each application and utility launch without error from the FLEX Index Start Menu.
  • Confirm that each application and utility launch without error from the ID Shield Start Menu.
  • Confirm that each application and utility launch without error from the LabDE Start Menu.

Demos - Processing

  • Copy Demo_FLEXIndex, Demo_IDShield, and Demo_LabDE folders to the test machine.
  • Install each demo by exercising the associated CleanAll batch files.
  • Use SQL Server Management Server to exercise I:\Common\Testing \Product Testing\IDShield_Sanity\CreateDomainLogin_AddForDemoDBs.sql to allow Domain Users to access the demo databases.
  • Exercise queuing and processing for each demo.
  • Confirm that no errors were logged.

Demos - Verification

  • Create separate Client licenses for each product. Additional separate licenses for utilities and ignoring the USB key are also allowed.
  • Log in to the test machine as a domain user.
  • Exercise verification for Demo_FLEXIndex
    • Confirm good results found for each image.
    • Confirm swiping success for each field with each swiping tool.
    • Confirm a successful save and commit.
    • Confirm successful exercise of each toolbar button.
  • Exercise verification for Demo_IDShield
    • Confirm good results found for each image.
    • Confirm swiping success for each field with each swiping tool.
    • Confirm a successful save and commit.
    • Confirm successful exercise of each toolbar button.
  • Exercise verification for Demo_LabDE
    • Confirm good results found for each image.
    • Confirm swiping success for each field with each swiping tool.
    • Confirm a successful save and commit.
    • Confirm successful exercise of each toolbar button.
