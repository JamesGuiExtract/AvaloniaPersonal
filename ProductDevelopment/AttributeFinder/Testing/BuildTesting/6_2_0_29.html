  1. Descriptions of Changes

    • [FlexIDSCore #3097, #3189] Loop Value Finder
      • Confirm that the new value finder is available for use.
      • Exercise the Loop Finder checklist.
      • Do independent testing of Loop Finder functionality. Confirm that results are accumulated through loop iterations.

    • [FlexIDSCore #3098] Loop Preprocessor
      • Confirm that the new preprocessor is available for use.
      • Exercise the Loop Preprocessor checklist.
      • Do independent testing of Loop Preprocessor functionality.

    • [FlexIDSCore #3181] Extract Lines Returning Empty Attribute
      • Create an RSD file using a Conditional Value Finder where Extract Lines is used as the finder.
      • Configure Extract Lines to return a specified line that will not be found in the input. An example is line 25 from a 10-item list.
      • Confirm that no attribute is found and that the conditional finder returns false.

    • [FlexIDSCore #3188] Thread Exits Without Exception
      • Re-run one of the memory leak tests that exhibited the problem.
      • Continue running other memory leak tests and confirm that this error is no longer seen.

    • [LegacyRC #4908, #4922] Statistics tab in Vista and 2008 Server
      • Configure an FPS file to display the Statistics tab.
      • Confirm that each list is completely displayed without scrollbars.

    • [LegacyRC #4921] DB Admin Combo Box in Vista
      • Start the FAM DB Admin utility on a Vista test machine.
      • Select a database server that has many databases defined (i.e. Wood).
      • Select the Browse... item for database name.
      • Confirm that more than two items are displayed in the dropdown list.
