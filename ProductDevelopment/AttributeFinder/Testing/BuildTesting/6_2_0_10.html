FLEX Index / ID Shield 6.2.0.10 Testing

  1. Descriptions of Changes

    • [FlexIDSCore #3136] Copy Move Delete file processor
      • Confirm that read-only files can now be optionally moved.
      • Confirm that read-only files can now be optionally deleted.

    • [FlexIDSCore #3132] IDShield.ini supports Black or White redactions
      • Add RedactionColorInOutputFile=White to the General section.
      • Confirm that output redactions are white.
      • Modify the INI file to RedactionColorInOutputFile=Black.
      • Confirm that output redactions are black.
      • Remove the INI file entry for RedactionColorInOutputFile=Black.
      • Confirm that output redactions are black.

      • Add RedactionColorInOutputFile=White to the General section.
      • Create XML output files containing some redactions using the verification process.
      • Exercise RedactFromXml.exe and confirm that output redactions are white.
      • Run redaction rules via auto-redaction that save redactions as annotations.
      • Confirm that output redactions (annotations) are white.

    • [FlexIDSCore #3119] The Highlight tool no longer toggles redaction state in the verification UI
      • Select a highlight in the verification UI.
      • Confirm that the redaction state is not toggled.

    • [FlexIDSCore #3115, 3118, 3120, 3121, 3128, 3129] Highlights are moveable and resizable in the verification UI using the Select Highlights tool.
      • Exercise the Redaction Demo images and rules.
      • Test the ability to move and resize highlights found by rules.
      • Confirm that modified highlights persist when re-reviewing a saved image.
      • [LegacyRC #5042] Confirm that the Select Highlights icon is an arrow.

      • Add both rectangular and angular manual redactions.
      • Test the ability to move and resize the manual highlights.
      • Confirm that modified highlights persist when re-reviewing a saved image.

    • [FlexIDSCore #3044] PadValue VM accepts only digits
      • Confirm that the Required edit box accepts only digit characters.

    • [FlexIDSCore #3003] Conditional Task property page
      • Confirm that the Conditional Task property page inserts new True tasks after the current selection.
      • Confirm that the Conditional Task property page inserts new False tasks after the current selection.

    • [LegacyRC #5052] Exceptions logged with FAMDBAdmin database changes
      • Confirm that Application trace exceptions are logged for the following actions.
        • Database - Clear
        • Database - Reset lock
        • Database - Change password
        • Action - Add
        • Action - Remove
        • Action - Rename
        • Action - Manually set action status

    • [LegacyRC #5048] Lead Tools version 16
      • Check the file properties for Ltkrnu.dll.
      • Confirm that the version number is 16.0.0.4.

    • [LegacyRC #5044, 5045, 5046, 5047] Crystal Reports reporting functionality
      • Confirm that RunReport.exe exists in the CommonComponents folder.
      • Confirm that one or more reports can be run from FAMDBAdmin.exe.

    • [LegacyRC #4926] Auto-encryption is on by default in RDT builds
      • Install and license the Rule Development Toolkit.
      • Delete the registry key (if present) at HKCU/Software/Extract Systems/AttributeFinder/Settings/AutoEncrypt.
      • Start RDTConfig.exe.
      • Confirm that Automatically re-encrypt out-of-date ETF files is checked.

    • Miscellaneous
      • From the verification UI, confirm that the image cannot be scrolled during creation, movement, or resize of a manual redaction.

      • Confirm that ImageFormatConverter.exe supports a /retain option that retains redaction annotations.
      • Confirm that the auto-redaction property page mentions that saving redactions as annotations only applies to TIF images.
      • Confirm that the verification property page mentions that saving redactions as annotations only applies to TIF images.
      • Confirm that an Application trace exception is logged when retaining annotations and the output image is a PDF.
      • Confirm that an Application trace exception is logged when saving redactions as annotations and the output image is a PDF.

      • (?) Confirm that UCRSingleDocument.exe is installed with the Rule Development Toolkit.
