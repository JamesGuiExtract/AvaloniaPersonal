  1. Recently fixed SCRs

    • [LegacyRC #5639, #5671, #5829, FlexIDSCore #4281] VOA File Viewer
      • In the VOA File Viewer, create a new .eav file.
      • Create an attribute with a second-level subattribute (do not include a first-level subattribute). The format should be:

          Name|Macadamia
          ..InvalidSubattribute|Gazebo

      • Confirm that you cannot save the .eav file.

      • Open a .voa file TestImage001.tif.voa that exists in the same directory as TestImage001.tif
      • Click the "Save As" button.
      • Note that the default file name is TestImage001.tif and the default save as type is EAV files (*.eav).
      • Press the save button. Confirm that no confirmation dialog is displayed.

      • Open the .voa file at I:\Common\Testing\PVCS_Testing\FlexIDSCore\5829.
      • Confirm that it contains three top-level attributes, and that the first and third have subattributes.
      • Select the first and third top-level attributes and press the Merge Attributes button.
      • Confirm that the merged attribute has all of the subattributes from both of the original attributes.

      • Confirm that the file at I:\Common\Testing\PVCS_Testing\FlexIDSCore\4281 can be opened by the VOA File Viewer.

    • [LegacyRC #5835] FAM pause button
      • Set up an .fps file with at least one processing task and a number of files already queued.
      • Press the play button and then (as quickly as possible) hit the pause button.
      • Confirm that processing is correctly paused.

    • [FlexIDSCore #2256] RDTConfig crash
      • Open RDT Config and make a change.
      • Close RDT Config.
      • Confirm that no crash is seen.

    • [FlexIDSCore #3099] Legal description splitter
      • Run the legal description splitter on the following block of text:

        LOT 10, BLOCK 6, UNIVERSITY HILLS NO. 3, FILING NO. 1, CITY AND COUNTY OF DENVER, STATE OF COLORADO.

      • Confirm that "UNIVERSITY" is not changed to "UNITERSITY".

    • [FlexIDSCore #3564] Idle processing threads with busy database
      • During an endurance test, confirm that processing threads do not take an inordinate amount of time to grab new files once the currently processing files reach 100%.

    • [FlexIDSCore #4278] Copying an attributes value to another attribute

        Note: This test requires that the RDT be installed and licensed.

        • Use the ruletester to run Test.rsd on 1848657.tif.
