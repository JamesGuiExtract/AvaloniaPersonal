      • Confirm that a File - Start new log file... menu item exists.
      • Confirm that an Add Rename Comment dialog is presented.
      • Confirm that an added comment is stored as debug data.
      • Confirm that an added comment is not required.

      • Confirm that the default file extension for exporting debug data is *.txt.
      • Confirm that the Export Debug Data dialog includes an Only export unique values checkbox.
      • Confirm that the checkbox is unchecked by default.

      • Confirm that with the checkbox checked, duplicate entries are not seen in the exported debug data.

    • [LegacyRC #5564, #5565, #5566] Text function expander
      • Confirm that $Left() is available and functions properly.
        • The first parameter is the input string.
        • The second parameter is the number of characters to extract from the left side of the input string.
      • Confirm that $Right() is available and functions properly.
        • The first parameter is the input string.
        • The second parameter is the number of characters to extract from the right side of the input string.
      • Confirm that $Mid() is available and functions properly.
        • The first parameter is the input string.
        • The second parameter is the first (1-relative) character to take from the left side of the input string.
        • The third parameter is the number of characters to take from the input string.
        • Confirm that -1 for the third parameter extracts the remainder of the string.
