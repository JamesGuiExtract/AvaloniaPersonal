    • [FlexIDSCore #3211 - #3218, #3223] Verification and Auto-redact Property Pages
      • Confirm that the property pages have changed in the following ways.
        • Metadata output settings are configured in a separate dialog.
        • The Metadata output settings include controls for the existing settings.
          • A pre-populated combo box for Output version.
          • An edit box for Name of metadata output file.
          • An associated tag selector.
          • An associated ... (Browse) button.
        • Feedback collection settings are configured in a separate dialog.
        • Image output settings for verification are configured in a separate dialog.
        • General settings on the verification property page include a check box for Require users to specify exemption codes for redactions.
        • The Verification property page includes a groupbox and Customize... button for ID Shield Data File Location.
        • The Verification property page includes a button to configure Redaction text and color settings.
        • The Auto redaction property page includes a button to configure Redaction text and color settings.
        • The Redaction text and color settings dialog includes the following elements.
          • A group box for Text to place in redaction zones.
          • A pre-populated combo box for Redaction text.
          • An associated tag selector.
          • An associated read-only edit box for Sample text that uses the above pattern.
          • A group box for Redaction colors.
          • A pre-populated combo box for Border color.
          • A pre-populated combo box for Fill color.
          • Color support for Black and White.
          • A group box for Preferred text size and font.
          • An associated read-only edit box that displays the font name and size.
          • An associated ... (Browse) button that displays a Font selection dialog.
          • An associated label with If text is too large to fit inside redaction zone, it will automatically be scaled to fit.
        • Check boxes for Data categories to redact in the Auto redaction property page have been rearranged to save space.
        • The Auto redaction property page includes a group box for Redactions in output file.
        • Redactions in output file settings includes radio buttons for Retain any previously existing redactions in the output file and Output only redactions from this session (the default).
      • Confirm that the various changes are stable and persistent.
      • NOTE: Complete functionality may not be implemented yet.
