FLEX Index / ID Shield 7.0.0.2 Testing

  1. Descriptions of Changes

    • [FlexIDSCore #3021] OCR of blank region
      • Exercise the rules file attached to SCR #3021. An image link is provided.
      • Confirm that an exception is not thrown during processing of the associated image.

    • [FlexIDSCore #3186] Internal Error with VA - York - Unity rules
      • Make a local copy of the recent rules delivery for VA - York - Unity.
      • Make a local copy of the associated Day Forward images.
      • Confirm that the HKCU/Software/Extract Systems/AttributeFinder/Settings/CacheRSD registry key is set to 1.
      • Queue and process the images for a few minutes, confirming that no unusual errors are seen.
      • Modify the HKCU/Software/Extract Systems/AttributeFinder/Settings/CacheRSD registry key to 0.
      • Queue and process the images again for a few minutes, confirming that no unusual errors are seen.

    • [FlexIDSCore #3194] Loop Finder Validation
      • Create and configure a Loop Finder rule.
      • Modify the rule settings to change the number of iterations to a non-numeric value.
      • Confirm that the error is handled and displayed in an appropriate manner.

    • [FlexIDSCore #3198, #3199] Files Installed to CommonComponents
      • Check the CommonComponents folder.
        • Should contain all EXEs and DLLs
        • There should not be any other Bin subfolders
        • The ID Shield report files should be found in the CommonComponents/FileProcessingComponents/Reports/Standard folder.

    • [FlexIDSCore #3200] RSD Files with Internal Use Only flag
      • Confirm that the Sanity test runs correctly for the Rule Tester utility application.

    • [FlexIDSCore #3201, #3202] MICR Finder and Check Finder
      • Confirm that these two new Value Finders are available in the Rule Set Editor. It may be necessary to run Check For New Components.
      • Test these rule objects using the St Johns Set001 images.

    • [FlexIDSCore #3206] Default location of Component Data folder
      • Confirm that the ID Shield Testing section of the Sanity test runs correctly.

    • [LegacyRC #4861, #4830] Manual Zones
      • Make a local copy of the LA - Lafourche images from K:\Common\Engineering\Sample Files\Cott\LA - Lafourche\Random Subset Set001 Set002\Images.
      • Confirm that OCR works properly for most or all of the images.

    • [LegacyRC #5115] Image Format Converter for JPG Files
      • Exercise the Image Format Converter portion of the Miscellaneous Utilities testing checklist.
