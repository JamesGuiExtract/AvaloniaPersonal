    • [FlexIDSCore #3100] Legal Description Splitter

      • Confirm that the utility correctly creates a redacted image for each of the metadata formats.

    • [FlexIDSCore #3224, #3342] Support for multiple Verification sessions in metadata

      • Re-verify the image

      • Confirm that metadata includes a <RedactionTextAndColorSettings> node.
      • Confirm that Exemption Code information is included in the metadata.

    • [FlexIDSCore #3226, #3228, #3229, #3232] Exemption codes
      • Confirm that Apply last exemption code works for verification.
      • Confirm that Apply exemption code to all redactions works for verification.
      • Confirm that a confirmation message is displayed if Apply exemption code to all redactions is asked to overwrite one or more existing exemption codes.

      • Confirm that E works as a shortcut for Apply exemption code.
      • Confirm that Ctrl+E works as a shortcut for Apply exemption code to all redactions.
      • Confirm that L works as a shortcut for Apply last exemption code.

      • Review the XLS file attached to #3228. Confirm that the Code, Summary, and Description entries are provided in the user interface.
      • Review the XLS file attached to #3229. Confirm that the Code, Summary, and Description entries are provided in the user interface.

    • [FlexIDSCore #3325] Copyright dates in About boxes

      • Confirm that the copyright text in other About boxes is correct.

    • [FlexIDSCore #3326] MICR Finder and rotated checks

      • Confirm that the Check Finder is available as a rule object.

    • [FlexIDSCore #3330] Configure email settings

      • Confirm that Configure email settings displays the appropriate settings dialog and that the settings are persistent.

      • Confirm that an Invalid Identifier in Attribute Name/Type! error is not seen.

    • [FlexIDSCore #3341] Loop Finder error

    • [FlexIDSCore #3342] Merging attributes in VOA Viewer

      • Confirm that the button is disabled unless multiple attributes are selected.
      • Confirm that with multiple attributes selected, the only enabled buttons are: New Attribute, Delete Attribute, Clear Attributes and Merge Attributes. Also confirm that the Name, Value and Type edit boxes are cleared.

      • Exercise TestImage001.tif from the Redaction Demo. Merge two of the attributes, a clue and the associated social security number.

        • Confirm that the merged Value is the concatenation of the two text strings.
        • Confirm that the merged Type remains SSN.
        • Select the merged attribute and confirm that the entire text is highlighted.
