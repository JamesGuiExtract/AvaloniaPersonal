  1. Descriptions of Changes

    • [FlexIDSCore #3151] Cannot Install Software If Not Administrator
      • Confirm that the installation process exits gracefully if the logged in user is not a member of the Administrators group.

    • [FlexIDSCore #3162] Black and White Redactions Via Verification
      • Add RedactionColorInOutputFile=White to the General section of IDShield.ini.
      • Confirm that output redactions are white.
      • Modify the INI file to RedactionColorInOutputFile=Black.
      • Confirm that output redactions are black.
      • Remove the INI file entry for RedactionColorInOutputFile=Black.
      • Confirm that output redactions are black.

      • Add RedactionColorInOutputFile=White to the General section.
      • Verify the images and confirm that metadata XML files are created.
      • Exercise RedactFromXml.exe and confirm that output redactions are white.

    • [LegacyRC #5048] Lead Tools version 16
      • Confirm that the opening an image in the Spot Recognition Window does not display the NAG DLL.
      • Confirm that rotated view perspectives display without errors.
      • Confirm that PDF images display without errors.

    • [LegacyRC #5063, #5064, #5067] Various Reporting Changes
      • Exercise the following items for each installed report as noted below.
        • Confirm that the report conforms to the Report Design Standards section of the 7.0 Reporting Features Specification.
        • Confirm that an XML file associated with the report is present in the reports folder. Note that DateRange values should be the enumeration such as LastWeek.
        • Confirm that a thumbnail image of the selected report is provided.
        • Confirm that a PDF output file can be auto-generated using command-line parameters.
        • Confirm that the /ow command-line parameter allows the PDF output file to be overwritten.
        • Confirm that not using the /ow command-line parameter raises an exception if the output PDF file already exists.
        • Confirm that the /prompt command-line parameter forces display of a parameters prompt if the report requires parameters.

      • Installed reports
        • File Action Manager Action List Report
        • File Action Manager Computer List Report
        • File Action Manager File Summary Report
        • File Action Manager Machines Used For Processing Report
        • File Action Manager Page Summary Report
        • File Action Manager User List Report

      • Confirm that the caption on the main UI has Report Viewer.

    • [LegacyRC #5062] Block Fitting
      • Exercise the Redaction Demo rules and images and begin verification.
      • Use the rectangle tool to highlight a line of text while holding down the Shift key.
      • Confirm that one attribute is created that contains the highlighted text.
      • Confirm that the highlight automatically shrunk to fit the pixels within the rectangular area.
      • Confirm that the highlighted region retains its rectangular shape.

      • Use the angle tool to highlight a line of text while holding down the Shift key.
      • Confirm that one attribute is created that contains the highlighted text.
      • Confirm that the highlight automatically shrunk to fit the pixels within the rectangular area.
      • Confirm that the highlighted region retains its angle.

      • Confirm that Block Fitting also works within the Spot Recognition Window.

    • [LegacyRC #5065, #5066, #5073] Line Fitting
      • Exercise the Redaction Demo rules and images and begin verification.
      • Use the rectangle tool to highlight multiple lines of text while holding down the Ctrl key.
      • Confirm that one attribute is created that contains individual lines of highlighted text.
      • Confirm that each highlighted line automatically shrunk to fit the pixels within the rectangular area.
      • Confirm that the highlighted region retains its rectangular shape.

      • Confirm that the same functionality is seen with a multi-line angled highlight.

      • Confirm that line fitting within the Spot Recognition Window produces only one dialog with OCR results.

    • [FlexIDSCore #3148] Grouping Attributes From Line Fitting in Verification UI
      • Exercise the Redaction Demo rules and images and begin verification.
      • Highlight multiple lines of text while holding down the Ctrl key.
      • Confirm that one attribute is created that contains the collected highlights.
      • Confirm that clicking on one highlight will select the entire group.

    • [LegacyRC #5071] Disable Auto-Fitting for VOA Viewer and Flex Data Entry
      • Confirm that highlighting multi-line text while holding the Ctrl key does not create a multi-zone attribute in the VOA Viewer application.
      • Confirm that highlighting multi-line text while holding the Ctrl key does not create a multi-zone attribute in the Flex Data Entry application.

    • [LegacyRC #5051] Automatic Reconnection To Database After Network Problem
      • Exercise the Database Connectivity checklist.
