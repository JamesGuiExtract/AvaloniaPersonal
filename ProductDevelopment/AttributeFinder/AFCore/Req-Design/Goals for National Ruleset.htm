Goals and Requirements for a Developing and Managing a National Ruleset

 

Terms:

  1. A Document Template represents the layout and sequence of words, paragraphs, graphics and empty space in a document.  Some of the empty space in a document template may be used for filling in document data.
  2. A Document Template Set (DTS) is a collection of unique document templates.
  3. Document Data represents the unique information in the document without any regard to where it may appear in the document.  Examples of document data are party names, dates, amounts, legal descriptions, roles, barcodes, notary seals, signatures, etc.
  4. A Document Type is a specific type of document in a specific industry.  Examples are "Mortgage documents" in the "County documents" industry, or "Invoices" in the "Accounting" industry.
  5. A Completed Document is the combination of a document template and appropriate document data.
  6. A National Field is a particular type of data that is of interest to and is captured by customers or prospects all across the U.S.A.
  7. A National RuleSet is a set of FlexIndex rules that is capable of finding values at a "reasonable" success rate for a particular national field on documents received from anywhere in the country.
  8. The National Document & Data (NDD) is defined as the massive collection of images and their corresponding index data for the national fields that UCLID has received from all customers or prospects that it has dealt with at any given time.

Derived Statements:

  1. Two completed documents are considered as using the same document template if they look exactly the same when all document data is removed from them.
  2. Completed documents of a particular document type may be using different document templates.
  3. A document template is an abstract concept and therefore is not an independent entity.  It is always represented only with an example of a completed document.

High Level Goals:

  1. Grantor, Grantee, Legal Description, and TaxID need to be treated as national fields and national rulesets need to be developed for each of these fields.  More fields may be elevated to the "national field" level in the future.
  2. We need to develop a set of procedures to collect, maintain, and continuously improve the set of document templates and rules to capture data for each national field.

Derived Requirements:

  1. As we receive new sets of completed documents from customers or prospects, we should be able to discover document templates therein which we don't have in the DTS for a particular national field.
  2. For each newly discovered unique document template for a particular national field, we should:
    1. Add the document template to the DTS associated with the national field.
    2. Enhance and test rules for the national field to allow capturing of data on the new document template
    3. Add a new automated test entry with an instance of a completed document containing the new document template.
    4. Run the complete automated test for the national field to ensure that the enhanced rules capture data successfully on documents all document templates, including the newly added one.
  3. At any given time, we should be able to determine the following within a short amount of time (less than half an hour):
    1. The geographical distribution of document templates associated with a particular national field.
    2. The geographical distribution of document templates associated with any national field.
    3. The document type distribution of document templates associated with a particular national field.
    4. The document type distribution of document templates associated with any national field.
    5. The customer from whom we received a particular document template associated with a particular national field.
    6. The number of document templates we have successfully written rules for with respect to a particular national field.
    7. The set of document templates associated with a particular rule (or portion of a rule) of a particular national field.
  4. Few times a year (or more often if necessary), the national ruleset for the various national fields shall be run against the NDD to double-check that the capture rate percentages are generally following an upward trend.
