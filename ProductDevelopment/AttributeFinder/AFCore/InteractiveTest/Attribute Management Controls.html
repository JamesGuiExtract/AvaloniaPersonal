Rule Set Editor - Attribute Management Controls

  1. Attributes - Combo Box

    • Add button displays Add Attribute dialog.
      • Define Attribute name edit box is editable.
      • OK button is inactive until a name is entered.
      • OK button saves changes and exits dialog.
      • OK button throws exception when an invalid attribute name is entered (i.e. a-b).
      • Cancel button ignores changes and exits dialog.
      • X button ignores changes and exits dialog.
    • Delete button displays Confirm Delete dialog.
      • YES button deletes attribute.
      • NO button cancels dialog and does not delete attribute.
    • Rename button displays Rename Attribute dialog.
      • Existing name is pre-populated in the edit box.
      • OK button becomes inactive if the name is cleared.
      • OK button saves changes and exits dialog.
      • OK button throws exception when an invalid attribute name is entered (i.e. a-b).
      • Cancel button ignores changes and exits dialog.
      • X button ignores changes and exits dialog.
      • New name is displayed on RuleSet Editor dialog.

  2. Attribute Rules - List Box

    • Insert button displays Add Attribute Finding Rule dialog.
    • Highlighting an existing clue in the list box entry activates Delete and Modify buttons.
    • CTRL + Left Click allows for multiple selection of list box items.
    • SHIFT + Left Click allows for multiple selection of list box items.
    • Right Click on list box displays context menu

      Context Menu Options

      • Cut exists
        • Items can be successfully cut to the clipboard.
        • Click on Cut displays confirm delete dialog.
        • Successfully cuts multiple selected items.
      • Copy exists
        • Items can be successfully copied to the clipboard.
        • Successfully copies multiple selected items.
      • Paste exists
        • Items can be successfully pasted from clipboard.
        • Successfully pastes multiple cut or copied items.
      • Delete exists
        • Item is deleted successfully.
        • Click on Delete displays confirm delete dialog.
        • Successfully deletes multiple selected items.
      • Cancel exists
        • Exits context menu successfully.

    • Delete button displays Confirm Delete dialog

    • Confirm Delete Dialog

      • YES button deletes the finding rule and exits the dialog.
      • NO button ignores delete request and exits dialog.

    • Modify button displays a pre-populated Add Attribute Finding Rule dialog.
      • Changes made are persistent.
    • Graphical UP arrow button moves the selected item up in the list box.
    • Graphical DOWN arrow button moves the selected item down in the list box.
    • Graphical UP and DOWN arrow buttons become inactive when reaching top or bottom of the list.
    • Stop searching after first value found check box is checkable.
