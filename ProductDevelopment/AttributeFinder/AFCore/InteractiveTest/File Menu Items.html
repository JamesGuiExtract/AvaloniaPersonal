Ruleset Editor - File Menu Items

Window Form:

  • Ruleset Editor text appears in title bar.
  • RSD filename appears in title bar when loaded.
  • _ - Minimizes window and child windows (including Ruleset Tester).
  • X - Closes window.
  • Click on X displays Save Changes dialog when changes have not been saved.
  • ALT+F4 closes window and child windows (including Ruleset Tester)
  • ALT+F4 displays Save Changes dialog when changes have not been saved.
  • [P16 #2505, #2506] Confirm that neither a memory error nor a Windows error are seen at application exit.

File Menu Items:

  1. File - Drop Down

    New

    • Click on New clears current rules.
    • Click displays Save Changes dialog when changes have not been saved.
    • CTRL+N clears current rules.

    Open

    • Click on Open displays File - Open dialog.
    • Click displays Save Changes dialog when changes have not been saved.
    • CTRL + O displays dialog.
    • Opens RSD files.
    • Opens RSD.ETF files.
    • When opened, RSD.ETF files are not editable or viewable.

    Import

    • Click on Import displays dialog.
    • CTRL + I displays dialog.
    • RSD files can be imported successfully.
    • Files imported with attributes of the same name are imported with _# appended to the name.

    Export

    • Click displays Export Attribute Rules dialog.
    • CTRL + E displays dialog.
    • RSD files can be exported successfully.

      Export Attribute Rules Dialog

      • OK - Exports attributes successfully.
      • Cancel - Does not export attributes and closes dialog.
      • Only Checked attributes are exported.
      • Changing the name of the exported attribute changes the name in the exported file.

    Save

    • Click on Save saves file if has previously been saved.
    • Click on Save displays Save As dialog, if file has never been saved.
    • CTRL+S saves changes to the file.
    • CTRL+S opens Save As dialog if file has never been saved.

    Save As

    • Click on Save As displays Save As dialog.

    Recent Files

    • Selecting Recent Files displays submenu file list of recently opened files.
    • Click on Recent Files -> Submenu Item opens the file.
    • Click on Recent Files -> Submenu Item displays Save Changes dialog when changes have not been saved.
    • Newly opened files get added to this list

    Properties

    • Selecting Properties opens Ruleset Properties dialog.
      • Ruleset Properties Dialog
    • FlexIndex - Indexing check box is checkable.
      • When checked, Indexing appears on status bar.
    • FlexIndex - Pagination check box is checkable.
      • When checked, Pagination appears on status bar.
    • IDShield - Redaction (By Page) check box is checkable.
      • When checked, Redaction (pages) appears on status bar.
      • When checked using the mouse, Redaction (By Document) is automatically unchecked.
      • [P16 #2973] When checked using the space bar, Redaction (By Document) is automatically unchecked.
    • IDShield - Redaction (By Document) check box is checkable.
      • When checked, Redaction (docs) appears on status bar.
      • When checked using the mouse, Redaction (By Page) is automatically unchecked.
      • [P16 #2973] When checked using the space bar, Redaction (By Page) is automatically unchecked.
    • [P16 #2973] Only one of the check boxes: IDShield - Redaction (By Page) or IDShield - Redaction (By Document) may be checked at one time.
    • Key Serial #s: edit box is editable.
      • When entered, the value entered is displayed on status bar.
    • Ruleset can be executed only internally check box is checkable.
      • When checked, InternalUseOnly = Yes is displayed on status bar.

    • OK - Save changes and exits dialog.
    • Cancel - Cancels changes and exits
    • X button ignores changes and exits dialog

    Exit

    • [P16 #2452] Click on Exit - exits program successfully ( without an exception or error message ).
    • Click displays Save Changes dialog when changes have not been saved.

      Save Changes Dialog

      • Yes - Saves changes and exits program.
      • No - Does not save changes and exits program.
      • Cancel - Does not exit program, but does exits menu and does not save changes.
      • Clicking X - Does not exit program, but does exits menu and does not save changes.

  2. Tools - Drop Down

    • Click on Open Rule Tester opens Rule Tester dialog.
      • Works on RSD files.
      • Works on RSD.ETF files.
    • CTRL + T opens Rule Tester dialog.
    • Click on Execute Test Harness starts an instance of the Test Harness.
    • Click on Check for New Components works.
      • A list of objects found is provided.
      • [P16 #2710] The number of Input Receivers is included in the list.

  3. Help - Drop Down

    • Click on Attribute Finder Help opens Flex Index RDT help file.
    • Click on About Attribute Finder brings up about About Attribute Finder dialog box.
      • Dialog displays the correct version number - including Patch if appropriate.
      • Dialog displays the current FKB Update version number.
