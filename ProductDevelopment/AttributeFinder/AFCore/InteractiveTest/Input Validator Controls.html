Ruleset Editor - Input Validator Controls

Select Input Validator

  • Displays description and type of input validator correctly
  • Checkbox is present
    • Checkbox is disabled when no input validator is selected
    • Checkbox is checkable when an input validator is selected
    • Checkbox setting is persistent
  • [P16 #966] Double-click on defined input validator displays configuration dialog.
  • [P16 #966] Ctrl + Double-click on defined input validator displays the Select Input Validator dialog.
  • [P16 #966] Alt + Double-click on defined input validator displays the Configure description dialog.
  • Right click displays context menu

    Context Menu Options

    1. "Cut" exists
      1. Items can be successfully cut to the clipboard
      2. Click on "Cut" loads confirm delete dialog.
      3. Checkbox disables when cut takes place
    2. "Copy" exists
      1. Items can be successfully copied to the clipboard
    3. "Paste" exists
      1. Items can be successfully pasted from clipboard
      2. Paste enables checkbox
      3. Checkbox retains state that the copied/cut input validator had
    4. "Delete" exists
      1. Item is deleted successfully.
      2. Click on "Delete" loads confirm delete dialog.
      3. Checkbox disables when input validator is deleted
    5. "Cancel" exists
      1. Exits context menu successfully.
  • "Select" button displays "Select Input Validator Dialog"

Select Input Validator Dialog

  • "Input Validator Description" edit box is editable
  • "Select Input Validator" combo box is selectable
    • "<None>" is listed
    • "Date" is listed
    • "Double" is listed
    • "Float" is listed
    • "Long Integer" is listed
    • "Regular Expression" is listed
    • "Short Integer" is listed
    • "Value From List" is listed

  • "<None>" clears the existing validator and description
  • "Configure" button displays appropriate Input Validator dialog
  • "Configure" button is only active when Input Validator has a dialog
  • "OK" button saves changes and exits dialog
  • "Cancel" button ignores changes and exits dialog
  • "X" button ignores changes and exits dialog
  • "Use the following Input Validator" edit box is not editable
  • "Use the following Input Validator" edit box displays selected Input Validator and corresponding description
