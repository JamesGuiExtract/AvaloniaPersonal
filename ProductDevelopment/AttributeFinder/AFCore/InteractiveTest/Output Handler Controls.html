Ruleset Editor - Output Handler Controls

Select Output Handler

  • Displays description and type of output handler correctly.
  • Checkbox is present
    • Checkbox is disabled when no output handler is selected.
    • Checkbox is checkable when an output handler is selected.
    • Checkbox setting is persistent.
  • [FlexIDSCore #966] Double-click on defined output handler displays configuration dialog.
  • [FlexIDSCore #966] Ctrl + Double-click on defined output handler displays the Select Output Handler dialog.
  • [FlexIDSCore #966] Alt + Double-click on defined output handler displays the Configure description dialog.
  • Right click displays context menu.

    Context Menu Options

    • Cut exists.
      • Items can be successfully cut to the clipboard.
      • Click on Cut loads confirm delete dialog.
      • Checkbox disables when cut takes place.
    • Copy exists.
      • Items can be successfully copied to the clipboard.
    • Paste exists.
      • Items can be successfully pasted from the clipboard.
      • Paste enables checkbox.
      • Checkbox retains state of copied/cut output handler.
    • Delete exists.
      • Item is deleted successfully.
      • Click on Delete displays Confirm delete dialog.
      • Checkbox disables when output handler is deleted.
    • Cancel exists.
      • Exits context menu successfully.

Select Output Handler Dialog

  • Select button displays Select Output handler dialog.
  • Output Handler Description edit box is editable.
  • Select Output Handler combo box is selectable.
    • <None> is listed.
    • Change case is listed.
    • Conditionally handle output is listed.
    • Eliminate duplicates is listed.
    • Merge attribute trees is listed.
    • Merge attributes is listed.
    • Modify attributes is listed.
    • Move and modify attributes is listed.
    • Output data to VOA file is listed.
    • Output data to XML file is listed.
    • Reformat person names is listed.
    • Remove attributes is listed.
    • Remove characters is listed.
    • Remove entries from list is listed.
    • Remove invalid entries is listed.
    • Remove spatial information is listed.
    • Replace strings is listed.
    • Run object on query is listed.
    • Select multiple output handlers is listed.
    • Select only unique attributes is listed.
    • Select using majority is listed.
    • Select with UI is listed.
    • Translate values or types is listed.

  • <None> clears the existing output handler and description.
  • Configure button displays appropriate output handler dialog.
  • Configure button is only active when output handler has a property page.
  • OK button saves changes and exits the dialog.
  • Cancel button ignores changes and exits the dialog.
  • X button ignores changes and exits the dialog.
