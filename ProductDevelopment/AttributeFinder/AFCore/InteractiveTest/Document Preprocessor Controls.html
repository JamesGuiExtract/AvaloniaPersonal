Ruleset Editor - Document Preprocessor Controls

  1. Display Box

    • Displays description and type of preprocessor correctly.
    • Checkbox is present
      • Checkbox is disabled when no preprocessor is selected.
      • Checkbox is checkable when a preprocessor is selected.
      • Checkbox setting is persistent.
    • [FlexIDSCore #966] Double-click on defined preprocessor displays configuration dialog.
    • [FlexIDSCore #966] Ctrl + Double-click on defined preprocessor displays the Select Document Preprocessor dialog.
    • [FlexIDSCore #966] Alt + Double-click on defined preprocessor displays the Configure description dialog.
    • Right click displays context menu

      Context Menu Options

      1. Cut exists
        • Items can be successfully cut to the clipboard.
        • Click on Cut loads confirm delete dialog.
        • Checkbox disables when cut takes place.
      2. Copy exists
        • Items can be successfully copied to the clipboard.
      3. Paste exists
        • Items can be successfully pasted from clipboard
        • Paste enables checkbox
        • Checkbox retains state from the copied/cut preprocessor.
      4. Delete exists
        • Item is deleted successfully.
        • Click on Delete loads confirm delete dialog.
        • Checkbox disables when preprocessor is deleted.
      5. Cancel exists
        • Exits context menu successfully.

  2. Select Button

    • Brings up Select Document Preprocessor dialog.
    • Document Preprocessor Description edit box is working.
    • Select Document Preprocessor combo box is working.
      • <None> is listed.
      • Selecting <None> clears and grays out the Description edit box and Configure button.

      • Advanced Replace String is listed.
      • Conditionally Preprocess is listed.
      • Document Classifier is listed.
      • Locate Image Region is listed.
      • Remove Spatial Info is listed.
      • Replace Strings is listed.
      • Select Multiple Document Preprocessors is listed.
      • Select Page Region is listed.

    • The Document Preprocessor must be configured text appears when a preprocessor is selected.
    • The Document Preprocessor must be configured text disappears when a preprocessor is configured.
    • Configure button brings up corresponding document preprocessor dialog.
    • OK - Preprocessor changes to selected type.
      • OK - Brings up Configuration message box if the preprocessor is not configured.
      • OK on Configuration message box brings up configuration dialog for the selected component.
      • X button on Configuration message box brings up configuration dialog for the selected component.
    • Cancel - Exits dialog and does not change the preprocessor.
    • Changes made are persistent.
