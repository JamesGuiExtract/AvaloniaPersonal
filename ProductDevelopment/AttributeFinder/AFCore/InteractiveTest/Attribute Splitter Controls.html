Ruleset Editor - Attribute Splitter Controls

Select Attribute Splitter

  • Displays description and type of attribute splitter correctly
  • Checkbox is present
    • Checkbox is disabled when no attribute splitter is selected
    • Checkbox is checkable when an attribute splitter is selected
    • Checkbox setting is persistent
  • [P16 #966] Double-click on defined splitter displays configuration dialog.
  • [P16 #966] Ctrl + Double-click on defined splitter displays the Select Attribute Splitter dialog.
  • [P16 #966] Alt + Double-click on defined splitter displays the Configure description dialog.
  • Right click displays context menu

    Context Menu Options

    1. "Cut" exists
      1. Items can be successfully cut to the clipboard
      2. Click on "Cut" loads confirm delete dialog.
      3. Checkbox disables when cut takes place
    2. "Copy" exists
      1. Items can be successfully copied to the clipboard
    3. "Paste" exists
      1. Items can be successfully pasted from clipboard
      2. Paste enables checkbox
      3. Checkbox retains state that the copied/cut attribute splitter had
    4. "Delete" exists
      1. Item is deleted successfully.
      2. Click on "Delete" loads confirm delete dialog.
      3. Checkbox disables when attribute splitter is deleted
    5. "Cancel" exists
      1. Exits context menu successfully.
  • "Select" button displays "Select Attribute Splitter Dialog"

Select Attribute Splitter Dialog

  • "Attribute Splitter Description" edit box is editable
  • "Select Attribute Splitter" combo box is selectable
    • "<None>" is listed
    • "Split a Legal Description" is listed
    • "Split an Address" is listed
    • "Split attributes using RSD file" is listed
    • Clicking the "Tag Selector" button to the right of the edit box brings up the "Tag Selector" Drop Down List
      • See Tag Selector
      • Selected item populates the edit box at the cursor position
      • Selected item replaces highlighted text in the edit box
    • "Split attributes using tokens" is listed
    • "Split Date and/or Time" is listed
    • "Split the name of Company or Person" is listed
    • "Split the name of a Person" is listed

  • "<None>" clears the existing splitter and description
  • "Configure" button displays appropriate Attribute Splitter dialog
  • "Configure" button is only active when Attribute Splitter has a dialog
  • "OK" button saves changes and exits dialog
  • "Cancel" button ignores changes and exits dialog
  • "X" button ignores changes and exits dialog
  • "Use the following Attribute Splitter" edit box is not editable.
  • "Use the following Attribute Splitter" edit box displays selected Attribute Splitter and corresponding description
