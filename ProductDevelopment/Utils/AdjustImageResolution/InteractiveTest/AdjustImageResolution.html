Adjust Image Resolution Utility (AIR) - Testing Checklist

  1. Basic Operation and Command-Line Arguments

    • Get a local copy of TooCoarse.tif and ThreeCoarsePages.tif test images from ProductDevelopment/Utils/AdjustImageResolution/InteractiveTest.
    • Confirm that AIR requires a license file.
    • Confirm that an error message is displayed if AIR executes with only two arguments.
    • Confirm that an error message is displayed if AIR executes with four arguments.
    • Confirm that an error message is displayed if AIR executes with six arguments.
    • Create and confirm existence of a copy of the original TooCoarse.tif and ThreeCoarsePages.tif test images.
    • Confirm that AIR supports infile, XResolution, and YResolution, arguments.
    • Confirm that AIR supports the above arguments, plus maximum height and maximum width arguments in inches where format is: -Hn -Wn.

  2. Testing Conditions and Success Cases

    • Basic command-line is: <filename> 300 300.
    • Advanced command-line is: <filename> 300 300 -H12 -W9.
    • Confirm that OCR of filename before AIR execution yields an empty USS file.
    • Confirm that OCR of filename after AIR execution yields a non-empty USS file.
    • Confirm proper behavior for TooCoarse.tif.
    • Confirm proper behavior for ThreeCoarsePages.tif.
    • Confirm proper behavior for TooCoarse.tif with -H12 and -W0 parameters.
    • Confirm proper behavior for TooCoarse.tif with -H0 and -W9 parameters.
    • Confirm proper behavior for TooCoarse.tif without -H12 and -W9 parameters.

  3. Error Cases

    • Confirm usage dialog displayed for TooCoarse.tif with -H0 and -W0 parameters.
    • Confirm empty USS file for TooCoarse.tif with only 100 and 100 parameters.
    • Confirm empty USS file for TooCoarse.tif 100 100 -H12 -W9.
