USS File Viewer - Testing Checklist

  1. Toolbar

    • Confirm that a Page navigation toolbar is visible under the menu bar.
    • Confirm that proper enable / disable behavior of navigation buttons depending on the number of pages in the image.
    • Confirm that proper enable / disable behavior of navigation buttons depending on the active page.
    • Confirm that entering a page number too big for the open file automatically reverts the edit box to the previous value.
    • Confirm that entering a 0 page number automatically reverts the edit box to the previous value.
    • Confirm that the edit box is wide enough for at least a 1000-page file.
    • [JIRA #11994] Confirm that entering a negative page number automatically reverts the edit box to the previous value.

  2. Page Bar

    • Confirm that the left side of the application window shows the page number of the character after the cursor.
    • Confirm that a horizontal line is the delimiter between pages.
    • Confirm that the page bar is wide enough for at least a 1000-page file.

  3. Drag and Drop

    • Confirm that dropping a .uss file on the main dialog opens the .uss file.
    • Confirm that dropping a .txt file on the main dialog opens the .txt file.
    • Confirm that dropping an image file on the main dialog opens the associated .uss file.
    • Confirm that dropping a three-digit extension image file (such as 123.001) on the main dialog opens the associated .uss file.
    • Confirm that dropping a double extension file on the main dialog opens the associated .uss file. An example is dropping 123.tif.voa and expecting 123.tif.uss to open.
    • Confirm that dropping a double extension file (such as 123.001.voa) on the main dialog opens the associated .uss file.

    • Confirm that dropping an image file that does not have a corresponding .uss file displays a meaningful error message.
    • Confirm that dropping a double extension file that does not have a corresponding .uss file displays a meaningful error message.

  4. File Menu

    • Open
      • Confirm that the Open menu item displays standard Windows Open dialog.
      • Confirm that the default file extension in .uss.
      • Confirm that contents of the opened file are displayed in a read-only edit box.
      • Confirm that the window caption shows Filename - USS File Viewer.

    • Close
      • Confirm that the Close menu item clears the read-only edit box.
      • Confirm that after Close, the window caption shows USS File Viewer.

    • Properties
      • Confirm that the Properties menu item displays a Properties dialog.
      • Confirm presence of the selectable path to the Source Document:.
      • Confirm presence of the selectable path to the Original Source Document:.
      • Confirm selectable presence of the OCR Engine Version.
      • Confirm that the OCR Engine version for a new USS file is Nuance 19.11.

      • Confirm presence of a list box for Page Information:.
      • Confirm presence of columns for Page #, Height, Width, Deskew and Orientation.

      • [ISSUE-12609] Confirm that USS files created via parallel OCR include OCR Engine Version.

    • Exit
      • Confirm that the Exit menu item closes the application.

  5. Tools Menu

    • Find
      • Confirm that the Find... menu item displays the Find dialog.
      • Confirm that Ctrl + F also displays the Find dialog.
      • Confirm that the Find what: edit box is editable.
      • Confirm that the Find button searches the main read-only edit box for the text contained in the local edit box.
      • Confirm that a No matches found message box is displayed when no match is found.
      • If a match is found
        • Confirm that the first match after the current cursor position is highlighted in the main read-only USS File Viewer window.
        • Confirm that the Find button becomes a Next button.
        • Confirm that the Previous button is enabled.
        • Confirm that the Next/Previous buttons allow you to step forward or backword through the matches, wrapping at the end of the document.
        • Confirm that the only enabled controls in the Find dialog are the Next, Previous, and Reset buttons.
        • Confirm that the Reset button re-enables the controls and moves focus to the Find what edit box.
        • Confirm that F3 and Shift+F3 function as shortcuts for Find Next / Find Previous.
      • Confirm that the Regular Expression check box is checkable.
      • Confirm that search strings are treated as regular expressions when checked.
      • Confirm that the Case Sensitive check box is checkable.
      • Confirm that searches are case sensitive when checked.

      • Confirm that the From % to % check box is selectable.
      • Confirm that selection enables the two corresponding edit boxes.
      • Confirm that input in the First edit box is restricted to numbers between 0-99.
      • Confirm that input in the Second edit box is restricted to numbers 1-100.
      • Confirm that only the restricted area of the document is searched.

      • Confirm that the Find in font sizes check box is selectable.
      • Confirm that selection activates the corresponding combo box and the two corresponding edit boxes.
      • Confirm that the From: value must be less than or equal to the to value.
      • Confirm that only the specified font sizes are searched.

      • Confirm that the X button exits the dialog.

    • Character Info
      • Confirm that the Character Info menu item displays the Character Information dialog.
      • Confirm that Ctrl + I also displays the Character Information dialog.
      • Confirm that the Guess 1, Guess 2 and Guess 3 edit boxes display the ASCII character following the cursor position.
      • Place the cursor immediately before the last character in a paragraph. Confirm that the End Of Zone and End Of Paragraph checkboxes are checked.
      • Place the cursor immediately before a non-whitespace character.
        • Confirm that the Page # edit box displays the current page number.
        • Confirm that the Font Size edit box displays the font size of the character.
        • Confirm that the Confidence edit box displays a number between 10 and 100.
        • Confirm that the Top, Left, Bottom and Right edit boxes are populated with pixel positions.
        • Confirm that one or more of the various Font checkboxes are checked.

      • Place the cursor immediately before a whitespace character.
        • Confirm that the End Of Zone and End Of Paragraph checkboxes are cleared.
        • Confirm that the Page # edit box is cleared.
        • Confirm that the Font Size edit box contains 0 pt.
        • Confirm that the Confidence edit box contains 100.
        • Confirm that the Top, Left, Bottom and Right edit boxes are set to -1.
        • Confirm that the various Font checkboxes are cleared.

      • Modify text selection in the main dialog to include multiple characters.
        • Confirm that the Guess 1, Guess 2 and Guess 3 edit boxes are cleared.
        • Confirm that the End Of Zone and End Of Paragraph checkboxes are cleared.
        • Confirm that the Page # edit box is populated.
        • Confirm that the Font Size edit box is renamed to Common Fnt Sz and is populated.
        • Confirm that the Confidence edit box is renamed to Avg Confidence and is populated with a value between 10 and 100.
        • Confirm that one or more of the various Font checkboxes are checked.

    • Font Size Distribution
      • Confirm that the Font Size Distribution menu item displays the Font Sizes dialog.
      • Confirm that the total number of non-whitespace characters is displayed at the top of the dialog.
      • Confirm that the Font Size column displays a list of font sizes in sorted order.
      • Confirm that the Num Chars column displays the number of characters of a particular Font Size.
      • Confirm that the Percentage column displays the percentage of the characters of a particular Font Size in the document.

    • Word Length Distribution
      • Open a USS file and select some text.
      • Confirm that the Word Length Distribution menu item displays the Word Lengths dialog.
      • Confirm that the Total Number of Words is displayed at the top of the dialog.
      • Confirm that a three-column list is displayed with Word Length, Count and Percentage columns.
      • Confirm that a vertical scrollbar is provided if necessary.
      • Confirm that the OK button closes the dialog.

  6. Status Bar

    • Confirm that the Page: item displays the page number at the current cursor position.
    • Confirm that the page number is -1 at the end of the document.

    • Confirm that the Page Con: item displays the average character confidence on the active page.

    • Confirm that the Start Pos: item displays the index of the character at the beginning of the selected text.
    • Confirm that if no text is selected it displays the zero-based index of the character following the cursor.

    • Confirm that the End Pos: item displays the index of the character at the end of the selected text.
    • Confirm that if no text is selected it displays the index of the character following the cursor.

    • Confirm that the Con: item displays the character confidence of the currently selected text.
    • Confirm that if no text is selected, it displays the character confidence of the character following the cursor.

    • Confirm that the % item displays the percentage of the document before the current cursor position.
    • Confirm that the value is 0% at the beginning of the document.
    • Confirm that the value is 100% at the end of the document.

    • Confirm that the status bar is displayed correctly during window resize events.

    • [JIRA #11982] Confirm that the Status Bar is properly initialized after opening a file.
    • [JIRA #11992] Confirm that the status bar is properly initialized immediately after opening a USS file via double-click.

  7. Help Menu

    • Confirm that the F1 key does not display an error message.
    • Confirm that the About USS File Viewer menu item displays the About USS File Viewer dialog.
    • Confirm that the dialog displays the expected version number plus copyright information.
    • Confirm that the same dialog is displayed from the About USS File Viewer menu item located on the system menu.

  8. Command-Line Options

    • Confirm that if a USS file name is provided as an argument, e.g. USSFileViewer.exe filename.uss, the specified file is opened.
    • Confirm that if two or more arguments are provided, a usage message is displayed and the application exits smoothly.
