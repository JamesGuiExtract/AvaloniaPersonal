CreateMultiPageImage Utility - Testing Checklist

  1. General

    • Confirm that CreateMultiPageImage requires a suitable license file.
    • [LegacyRCAndUtils #4345] Confirm that CreateMultiPageImage /? displays usage text.
    • Confirm that CreateMultiPageImage displays a usage message if other than 1, 2, 3, or 7 arguments are provided on the command-line.

  2. File Name Formats - using InteractiveTest folder as command-line argument

    The batch file test_CreateMultiPageImage.bat that is found in the InteractiveTest folder will run the following tests automatically. In order to use it, copy the three images and the batch file to a local folder and run it.

    • Name_Dot_DDD_Dot_Tif.
      • Delete any existing A.tif.
      • Copy test pages as A.001.tif, A.002.tif, A.003.tif. Run the application.
      • Confirm that a three-page A.tif file exists.

    • Name_Dot_DDD.
      • Delete any existing A.tif.
      • Copy test pages as A.001, A.002, A.003. Run the application.
      • Confirm that a three-page A.tif file exists.

    • Name_Hyphen_D_Dot_Tif.
      • Delete any existing A.tif.
      • Copy test pages as A-1.tif, A-2.tif, A-3.tif. Run the application.
      • Confirm that a three-page A.tif file exists.

    • Name_Underscore_P_D_Dot_Tif.
      • Delete any existing A.tif.
      • Copy test pages as A_P1.tif, A_P2.tif, A_P3.tif. Run the application.
      • Confirm that a three-page A.tif file exists.

    • Name_Underscore_DDDD_Dot_Tif.
      • Delete any existing A.tif.
      • Copy test pages as A_0001.tif, A_0002.tif, A_0003.tif. Run the application.
      • Confirm that a three-page A.tif file exists.

    • DDDDDDDD_Dot_Tif.
      • Delete any existing A.tif.
      • Copy test pages as 00000001.tif, 00000002.tif, 00000003.tif. Run the application.
      • Confirm that a three-page A.tif file exists.
