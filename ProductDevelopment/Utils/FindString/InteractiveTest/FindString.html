FindString Utility - Testing Checklist

  1. General

    • Confirm that FindString requires a suitable license file. (RDT License)
    • Confirm that FindString displays a usage message if other than 2-10 arguments are provided on the command-line.

  2. FindString - Interactive testing

    • Copy the contents of the folder at \ProductDevelopment\Utils\FindString\InteractiveTest to C:\test (case sensitive).
    • Follow the instructions that appear on the command line after running FindString_Test.bat.
    • For each compare case, confirm that there are no differences.
    • For each usage display confirm that there is a console message matching the expected message.
    • For each exception dialog, ensure the exception text matches the expected message.
    • For the exception log, (which is written to the same directory as the FindString_Test.bat file) confirm its existence with a current timestamp.
