SetOp Utility - Testing Checklist

  1. General

    • Confirm that SetOp requires a suitable license file.
    • Confirm that SetOp displays a usage message if other than 3, 4, 5, or 6 arguments are provided on the command-line.

  2. SetOp - Interactive testing

    • Run SetOp_Test.bat.
    • For each test case that is diff'ed, confirm that FC: no differences encountered is displayed in the command window.
    • For each exception dialog, ensure the exception text matches the expected message.
    • For the exception log, confirm its existence with a current timestamp.
