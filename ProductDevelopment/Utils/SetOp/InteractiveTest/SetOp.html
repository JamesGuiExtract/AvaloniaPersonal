SetOperations Utility - Testing Checklist

  1. General

    • Confirm that SetOperations requires a suitable license file.
    • Confirm that SetOperations displays a usage message if an invalid number of arguments are provided on the command line.

  2. SetOperations - Interactive testing

    • Run SetOp_Test.bat.
    • For each test case that is diff'ed, confirm that FC: no differences encountered is displayed in the command window.
    • Confirm that odd-numbered tests print the text List of duplicates: red.
    • For the exception log, confirm its existence with a current timestamp.
    • Confirm that the exception complains about the presence of duplicate items in one of the lists.

  3. Fixed SCRs

    • [LegacyRC #4990] Confirm that SetOperations.exe works correctly with large (~million line) files.
