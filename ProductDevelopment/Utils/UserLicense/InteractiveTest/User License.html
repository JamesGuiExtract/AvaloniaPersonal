UserLicense Utility - Testing Checklist

  • Confirm that the title bar says "User Key Generator"
  • Confirm that the following edit boxes exist and are editable
    • Name
    • Company Name
    • Phone Number
    • Email
    • Registration Key
  • Confirm that the "Generate license key" button enables the "Copy Info to Clipboard" button and inserts a license key into "User License Key" text box.
  • Confirm that the "User License Key" edit box is read only
  • Confirm that clicking the "Copy Info to Clipboard" button while any of each of the following editboxes are empty will throw an exception
    • Name
    • Company Name
    • Phone Number
    • Email
  • Confirm that clicking the "Copy Info to Clipboard" button while any of each of the following editboxes contain blank spaces will throw an exception
    • Name
    • Company Name
    • Phone Number
    • Email
  • Confirm that copying data allows it to be pasted into the COMLicense generator located at I:\Common\Engineering\Tools\SecureClients\COMLicense_InternalTesting via the "Paste User String" button.
  • Confirm that the copied data can be pasted into a notepad document and has a readable format and meaningful labels for everything except the license key.
