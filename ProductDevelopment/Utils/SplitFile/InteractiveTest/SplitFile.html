Split File Utility - Testing Checklist

  1. Basic Operation and Command-Line Arguments

    • Usage.
      • From a command prompt, type SplitFile.
      • Confirm that a usage statement is provided: SplitFile <File with file list> <# lines per file>

    • Test
      • Create a file with 10 lines named input.txt.
      • Open a command prompt in the input.txt directory.
      • Execute the following command: SplitFile input.txt 5
      • Confirm that two files are created: input.1 with lines 1-5 and input.2 with lines 6-10.
