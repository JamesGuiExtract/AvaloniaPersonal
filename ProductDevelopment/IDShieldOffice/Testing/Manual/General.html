ID Shield Office
General Features Testing Checklist

  • [IDShieldDesktop #4] Background OCR Support
    • Run ID Shield Office without opening an image. Confirm that no status information is presented about OCR progress.
    • Open an image.
      • Confirm that the status bar has a pane with OCR 00%.
      • Confirm that after a brief pause, OCR progress is provided in the status pane.
      • Confirm that when OCR is finished, the status bar text is OCR Complete! and that the font color of the text is green.
    • Open another image.
      • Confirm that the status bar resets to OCR 00% with the normal black font color.
      • Confirm that after a brief pause, OCR progress is provided in the status pane.
      • Before OCR finishes, open yet another image. Confirm that the OCR status briefly flashes OCR Canceled with a blue font color.
      • Confirm that OCR progress then resets and starts over.
    • Open an image that fails OCR.
      • Use I:\Common\Testing\Product Testing\IDShieldOffice\Images\BadOcr.tif.
      • Confirm that the status bar eventually displays OCR Error! with a red font color.
      • Confirm that double-clicking on the status bar pane provides an exception dialog.
      • Confirm that message showed in the dialog is not "Failure in OCR thread!
      • Confirm that a tooltip will be displayed when hovering over the OCR ERROR! pane on the status bar.
      • Confirm that double clicking the OCR Error! will display a meaningful message.
    • Open another image.
      • Confirm that the status bar resets to OCR 00% with the normal black font color.
      • Confirm that after a brief pause, OCR progress is provided in the status pane.
      • Before OCR finishes, exit ID Shield Office. Confirm that the OCR status briefly flashes OCR Canceled with a blue font color.
      • Confirm that exit from ID Shield Office occurs without significant delay.

  • Preferences - General
    • Open ID Shield Office and select the Edit - Preferences menu
    • Select the General tab.
    • Confirm that a slider is provided for specifying the OCR tradeoff between accuracy and speed.
    • Confirm that the default setting is in the middle for Balanced.
    • Confirm that a checkbox is provided for requiring all pages to have been visited before redacted document can be output.
    • Confirm that the checkbox is checked by default.
    • Confirm that the setting is persistent within the same session.
    • Confirm that the setting is persistent between sessions.

    • Confirm that a drop-down list is provided for default redaction fill color with the following entries:
      • Black
      • White
    • Confirm that the default fill color is Black.
    • Confirm that the default fill color is persistent within the same session.
    • Confirm that the default fill color is persistent between sessions.

  • Miscellaneous
    • Confirm that ID Shield Office executable file has a proper icon
    • Open ID Shield Office.
    • Confirm that splash screen appears and has a proper resolution.
    • Confirm that Help - ID Shield Office help displays ID Shield Office 1.0 Help including the following information
      • Getting started book
      • redacting book
      • Saving,Printing&Auditing book
      • Glossary book
    • Confirm that Help - Regular expression help displays a web page for Regular Expression Syntax (Scripting).
    • Confirm that Help - About ID Shield Office displays an About box including the following information
    • Confirm that the background of logo is transparent
      • ID Shield Office icon
      • ID Shield Office version number
      • Copyright date

    • Main window gets focus after non-model dialog closed
      • Launch IDSO without open an image file
      • Click Data type finder, or Bracketed text finder, or Wordlist finder to display a non-model dialog.
      • Switch to other application clicking taskbar icon
      • Switch back to IDSO non-model dialog clicking non-model dialog icon in the taskbar
      • Close non-model dialog
      • Confirm that main window gets focused
        # Currently it doesn't work with IE

    • Close IDSO while OCR not completing
      • [IDSO #31] Open 0350pages.tif (350 pages) image file.
      • Close ID Shield Office while OCR is not completed.
      • Confirm that ID Shield Office terminated normally.

    • Large memory usage and slow closing of IDSO
      • Open an image in IDSO
      • Create several redactions (at least 25)
      • Open Windows Task Manager.
      • Confirm that the memory usage is not exceptionally high (example: 102MB with 62 redactions)
      • Close IDSO
      • Confirm that IDShieldOffice.exe process disappears from task manager and no exceptional CPU usage spikes.

    • Confirm that IDSO window size is persistent across sessions.
