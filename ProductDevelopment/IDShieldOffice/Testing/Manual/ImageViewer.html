Image Viewer Testing Checklist

  • Tile Navigation
    • Set Fit-To-Page mode.
    • Open an image with at least 3 pages
    • Confirm that Previous Tile button and menu are disabled.
    • Click Next Tile.
    • Confirm that the Second page is displayed.
    • Confirm that both Previous Tile and Next Tile buttons are enabled.
    • Confirm that both Previous Tile and Next Tile menu are enabled.
    • Move directly to the last page and confirm that Next Tile button and menu are disabled.
    • Click Previous Tile.
    • Confirm that the page prior to the last is displayed.
    • Confirm that both Previous Tile and Next Tile buttons are enabled.
    • Confirm that both Previous Tile and Next Tile menu are enabled.

    • Confirm that both Previous object and Next object menu are enabled.

      • Press the next layer object button.

  • Zoom Navigation
    • Set Fit-To-Page mode.
    • Open a multi-page image.
    • Move directly to the last page and confirm that both the Zoom Previous and Zoom Next buttons and menu are disabled.
    • Zoom in one step.
    • Confirm that both the Zoom Previous button and menu are enabled.
    • Zoom previous and confirm that both the Zoom Next button and menu item are enabled.
    • Click Zoom in toolbar button.
    • Confirm that image is zoomed.
    • Click Zoom out toolbar button.
    • Confirm that image is zoomed.

  • Page rotation
    • Open an image file
    • Click Rotate counterclockwise toolbar button
    • Confirm that image is rotated Counterclockwise
    • Click Rotate clockwise toolbar button
    • Confirm that image is rotated clockwise

    • [IDSO #188, 190] Image distortion
      • Open FindTextTest.tif image file.
      • Redact one line and apply bates numbers at the first page.
      • Rotate clockwise or counterclockwise 90 degree.
      • Save the image file.
      • Rotate back the image file.
      • Confirm that Bates number and the redaction are in the right position.

      • Close and reopen saved image file
      • Rotate the first page 90 degrees.
      • Click Print Preview and then click Close on the Print Preview dialog.
      • Save the image.
      • Confirm that Bates number and the redaction are in the right position.

    • Open an image with at least 3 pages
    • Confirm that First Page and Previous Page buttons are disabled.
    • Confirm that First Page and Previous Page menu are disabled.
    • Click Next Page button to move to the second page.
    • Confirm that First Page and Previous Page buttons are now enabled.
    • Confirm that First Page and Previous Page menu are now enabled.
    • Click Last Page button to move to the last page.
    • Confirm that Last Page and Last Page buttons are disabled.
    • Confirm that Next Page and Next Page menu are disabled.
    • Click Previous Page button to move to the second page.
    • Confirm that Last Page and Last Page buttons are now enabled.
    • Confirm that Next Page and Next Page menu are now enabled.

    • Click View - Goto Page - Page number....
    • Confirm Goto Page dialog appears and then input a page number and click OK.
    • Confirm that specified page is now displayed.
    • Click View - Goto Page - Page number....
    • Confirm Goto Page dialog appears and then input a page number and click cancel.
    • Confirm that not navigated to the specified page.

    • Specify a page number in the toolbar textbox and then enter.
    • Confirm that specified page is now displayed.

      • pinned
      • closed by menu
      • closed by click x at the top right of individual pane

  • PVCS Testing
    • [.NetRCAndUtils #159] ImageViewer - highlights move on non-visible page
      • Open a multipage image in the image viewer
      • Draw a few redactions over text such that you will remember where they are located.
      • Go to the next page and draw a few more redactions.
      • Select the "Select redactions".
      • CTRL+A to select all redactions.
      • Go to the first page.
      • Draw a box around all the redactions on the page to select them.
      • Go to page 2.
      • Hit CTRL+A to select all redactions.
      • Move the redactions on page 2 to a valid location (ie. on the image).
      • Confirm the redactions on page 1 are still in their original positions.
      • Confirm the redactions on page 2 are still in their new positions.

    • [IDSD #338 - VISTA ONLY] Saved layer objects do not always appear on load
      • Open Image A with ID Shield and make sure all layers are turned on.
      • Make a manual redaction.
      • Save the document in IDSO format.
      • Open Image B.
      • Make a manual redaction.
      • Save the document in IDSO format.
      • Open Image A from the MRU list.
      • Repeat these steps 3 times, each time opening the other document from the MRU list without closing the other document first. Each time ensure that the redactions make in previous steps appear when the document is opened.
