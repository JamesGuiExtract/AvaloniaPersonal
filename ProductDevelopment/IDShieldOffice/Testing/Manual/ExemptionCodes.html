Exemption Code Testing Checklist

  • Configuration
    • Can require specification of at least one exemption code for each redaction
    • Can automatically associate last used exemption code(s) with newly added redactions
    • Can prompt user to specify exemption code(s) with newly added redactions
    • Can specify exemption code(s) to associate with redactions found using built-in rules
    • Can automatically associate exemption code(s) with search patterns

    • Can specify location of exemption code(s)
      • Painted within redaction area with clipped text
      • [Future] Painted in exemption code box with associated anchor to redaction area. Exemption code box is moveable
      • [Future] Exemption codes are painted at the bottom of the page and referenced from redaction areas like footnotes

  • Functionality
    • Associate one or more exemption codes with each redaction
    • Can select multiple redactions and assign exemption code(s) in append mode
    • Can select multiple redactions and assign exemption code(s) in replace mode
    • Can select multiple redactions and remove any associated exemption code(s)
    • Support for custom user-specified exemption code lists

    • Support for built-in exemption code lists
      • Privacy Act - USA
      • Freedom of Information Act - USA
      • Freedom of Information Act - UK
      • Freedom of Information Act - Scotland
      • Discovery Case
