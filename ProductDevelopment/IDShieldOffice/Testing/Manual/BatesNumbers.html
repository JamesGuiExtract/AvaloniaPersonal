ID Shield Office
Bates Number Testing Checklist

  • Preferences - Bates number
    • Open ID Shield Office and select the Edit - Preferences menu
    • Select the Bates number tab
    • Confirm that a checkbox is provided for requiring a Bates number before files can be saved
    • Confirm that the default setting is checked
    • Confirm that the setting is persistent within the same session
    • Confirm that the setting is persistent between sessions

    • Confirm that a read-only edit box is provided to display the Bates number format for the current page
    • Confirm that a Change... button is provided
    • Confirm that the Change... button displays a Change Bates number format dialog
    • Confirm that the setting is persistent within the same session
    • Confirm that Bates number format is persistent between sessions

    • Confirm that a Change... button is provided
    • Confirm that double click disabled Bates number format for current page editbox shows Change Bates number format dialog
    • Confirm that the Change default position and appearance... button displays a Change Bates number default position and appearance dialog
    • Confirm that the setting is persistent within the same session
    • Confirm that Bates number position is persisted between sessions

    • [IDSD #61] Confirm that user has an option to create a file as next number file

  • Preferences - Change Bates number format dialog
    • Open ID Shield Office and select the Edit - Preferences menu
    • Select the Bates number tab
    • Click Change... button to display Change Bates number format dialog dialog
    • Confirm that tab order is correct.
    • Confirm that dialog icon is the same as main window.
    • Confirm that next number defaults to As specified
    • Confirm that next number is formatted as Zero pad to 6 digits by default.
    • Confirm that Bates number format defaults to Use next number for each document and add separator with page number
    • Confirm that page number is formatted as Zero pad to 3 digits by default.
    • Confirm that default page number separator is /
    • Confirm that prefix and suffix is empty by default
    • Confirm that Sample Bates number is according to above setup
    • Add prefix and suffix, change default zero pad for next number and page number
    • Confirm that Sample Bates number is updated to the new format
    • Click OK
    • Confirm that the read-only Bates number editbox in the User preferences dialog is updated to above sample Bates number

  • Preferences - Change Bates number default position and appearance dialog
    • Open ID Shield Office and select the Edit - Preferences menu
    • Select the Bates number tab
    • Click Change Bates number default position and appearance ... button to display Change Bates number default position and appearance dialog
    • Confirm that tab order is correct.
    • Confirm that double click disabled font edit box shows Font dialog.
    • Confirm that dialog icon is the same as main window.
    • Confirm that font defaults to Arial 30pt;Bold
    • Confirm that the editbox to display font options is read only
    • Confirm that default anchor positions for Bates number are as following.
      0.50 inches from Right of the page
      0.50 inches from Top of the page
    • Confirm that default alignment for the anchor is right-top
    • Change all the default values
    • Click Cancel
    • Click Change Bates number default position and appearance ... button to display Change Bates number default position and appearance dialog
    • Confirm that all the options are still default values(not changed).

  • Configuration
    • Can specify alphanumeric prefix for the Bates number
    • Can specify alphanumeric suffix for the Bates number
    • Can specify special characters such as #,@,etc. as part of the Bates number prefix or suffix
    • Can specify a zero-padded document number up to a user-specified number of digits by direct input
    • Can specify a zero-padded document number up to a user-specified number of digits by a shared file
    • Can specify a zero-padded page number up to a user-specified number of digits
    • Can specify using next number on each page instead of a page number
    • Can specify an anchor position relative to the page
    • Can specify font size and style for Bates numbering
    • Can specify one of nine anchor points
