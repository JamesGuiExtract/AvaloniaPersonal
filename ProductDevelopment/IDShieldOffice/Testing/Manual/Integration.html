ID Shield Office
Printing and Integration Testing Checklist

  1. Printing

    • Print preview
      • Confirm that page is autofit to the initial preview window
      • The redacted document can be previewed with opaque redactions
      • Confirm that Bates# and redactions always previewed even if layer is off
      • Confirm that Clues are never previewed
      • Confirm that Black and White redaction color appear properly
      • Confirm that watermark can be previewed with evaluation license
      • Confirm that watermark can be printed with evaluation license
      • Confirm that initial zoom mode is set to Auto
      • Click Two pages button in the Print preview dialog
      • Confirm that preview window are auto zoomed to fit two pages
      • Confirm that Zoom works in Print preview dialog
      • Confirm that One page, two pages, Three pages, Four pages, Six pages work
      • Confirm that Page setup options applied to print preview

    • Print to physical printer
      • Confirm that can not print to ID Shield virtual printer.
      • [V2] Confirm that the original document can be printed without any redactions, document stamps, or Bates numbers.
      • Create an image file with Bates#, redactions and clues from IDSO.
      • Confirm that the redacted document can be printed with opaque redactions
      • Confirm that Bates# and redactions always printed even if layer is off
      • Confirm that Clues are never printed
      • Confirm that Black and White redaction color appear properly
      • Confirm that watermark printed with evaluation license
      • Confirm that Page setup options applied to print
      • Confirm that Bates# position are correctly printed
      • Confirm that Bates# are visible in the printed document

    • Printing progress dialog icon
      • Open an image at least 2 pages
      • Click File - Print preview menu
      • Confirm that icon for Generating previews dialog is correct

    • Printing margin
      • Open an image in IDSO
      • Draw some redactions on it
      • Open the page setup menu
      • Set margins to 1" in all edit boxes
      • Select Print preview
      • Confirm that redactions are rendered in right position

    • [IDSD:329] Print/Print preview - need intuitive message with no printer
      • Uninstall any currently installed printers other than ID Shield
      • Open an image and select File | Print.
      • Confirm an error message is displayed explaining that a printer other than ID Shield needs to be installed.
      • Select File | Print preview.
      • Confirm the same message is displayed.
      • Select File | Page setup.
      • Confirm the same message is displayed.

    • [.NetRCAndUtils:241] Print and PrintPreview - Need to handle rotated images properly
      • Open an image in ID Shield Office.
      • Draw a couple redactions and press Apply bates numbers (and remember their exact locations).
      • Press the Rotate clockwise button.
      • Select File | Print Preview.
      • Ensure the print preview shows the redactions and bates number(s) in the correct locations.
      • Press Print.
      • Ensure the page is printed with the redactions and bates number(s) in the correct locations.
      • Press the Rotate clockwise button so that the page is now upside down.
      • Select File | Print Preview.
      • Ensure the print preview shows the redactions and bates number(s) in the correct locations.
      • Press Print.
      • Ensure the page is printed with the redactions and bates number(s) in the correct locations.

    • [.NetRCAndUtils:254] Print - doesn't print non-visible layer objects
      • Open an image in ID Shield Office.
      • Make a redaction.
      • Press Apply bates numbers.
      • Make the redaction and bates number layers non-visible.
      • Press Print.
      • Confirm the printed document contains both the redaction and the layer object.

    • [.NetRCAndUtils:258] Print preview - Exception when viewing second lesser-page document
      • Open a multi-page document.
      • View the print preview.
      • Open another document that has fewer pages than the first document.
      • View the print preview.
      • Ensure no errors are displayed.

  2. Command line

    • Command line help/reset settings
      • Open command line window
      • Change current directory to C:\Program Files\Extract Systems\IDShieldOffice\Bin
      • Execute IDShieldOffice.exe /? in the command line
      • Confirm that a Usage messagebox appears to show the usage.
      • Click OK
      • Confirm that ID Shield Office is not launched.
      • Confirm that IDShieldOffice.exe process doesn't exist in the task manager

      • Open IDSO from desktop shortcut.
      • Relocate toolstrip and form locations and close IDSO.
      • Execute IDShieldOffice.exe /reset in the command line
      • Confirm that toolstrip and form locations are reset to default.

    • Command line license switch
      • Open command line window
      • Change current directory to C:\Program Files\Extract Systems\IDShieldOffice\Bin
      • Execute IDShieldOffice.exe /license in the command line
      • Confirm that UserLicense.exe is launched.
      • Confirm that the corresponding help page in the documentation is displayed.
      • Confirm that IDSO instance is not launched.

    • [IDSD #109] Open supported image from command line
      • Open command line window
      • Change current directory to C:\Program Files\Extract Systems\IDShieldOffice\Bin
      • Execute IDShieldOffice.exe C:\Images\FindTextTest.tif on the command line. Note that the path may need to be modified depending on the test file location.
      • Confirm that ID Shield Office is launched and the image file is opened.

      • Copy FindTextTest.tif to FindTextTest_copy.tif
      • Execute IDShieldOffice.exe C:\Images\FindTextTest.tif /d on the command line. Note that the path may need to be modified depending on the test file location.
      • Close the launched instance of ID Shield Office.
      • Confirm that FindTextTest_copy.tif is deleted.

      • Execute IDShieldOffice.exe C:\Images\FindTextTest.tif on the command line. Note that the path may need to be modified depending on the previous location of the test file.
      • Confirm that an Error message appears.
      • Click OK in the messagebox.
      • Confirm that ID Shield Office is launched.

    • Open large PDF from command line
      • Open command line window
      • Change current directory to C:\Program Files\Extract Systems\IDShieldOffice\Bin
      • Execute IDShieldOffice.exe C:\Images\0050pages.pdf on the command line. Note that the path may need to be modified depending on the previous location of the test file.
      • Confirm that ID Shield Office is launched first.
      • Confirm that Load document,please waiting... displayed on the status bar.
      • Confirm that the image file can be opened.
