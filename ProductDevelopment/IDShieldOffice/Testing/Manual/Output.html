  • Configuration
    • Open ID Shield Office and select the Edit - Preferences menu.
    • Select the Output files tab.

    • Confirm that the default output path for PDF is $DirOf(<SourceDocName>)\$FileNoExtOf(<SourceDocName>).redacted.pdf.
    • Confirm that the default output path for IDSO is <SourceDocName>.idso and read-only.
    • Confirm that tags button is disabled when saving IDSO files by default.
    • Confirm that an associated Tags > button is provided.
    • Confirm that the Tags > button provides a drop-down list of functions including ( among others ):
      • <SourceDocName>
      • $DirNoDriveof()
      • $DirOf()
      • $DriveOf()
      • $ExtOf()
      • $FileNoExtOf()
      • $FileOf()
      • $InsertBefortExt()
      • $Offset()
      • $PadValue()
      • $Replace()
      • $TrimAndConsolidateWS()
    • Confirm that the selected tag replaces text selected in the edit box.
    • Confirm that the selected tag is inserted at the cursor position in the edit box.
    • Confirm that the default output path is persistent within the same session.
    • Confirm that the default output path is persistent between sessions.

    • Confirm that read-only text is provided for Sample original input file name when the checkbox is checked.

    • Confirm that the Sample redacted output file name changes with changes in the Default save format.
    • Confirm that the Sample redacted output file name changes with changes in the Default output path.
    • Confirm that read-only text is not provided when the checkbox is unchecked.
