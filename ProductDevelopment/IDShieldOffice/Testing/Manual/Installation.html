ID Shield Office
Installation and Licensing Testing Checklist

  • Installation
      [IDSO #18] Confirm that "IDShieldOffice" should be replaced with "ID Shield Office" in all InstallShield Wizard Screens
    • Unprivileged user installation
      • Login as an unprivileged user.
      • Insert installation CD into CD-ROM.
      • Confirm that ID Shield Office splash screen appears.
      • Confirm that ID Shield Office - InstallShield Wizard messagebox that prompts installation needs administrator privileges appears.
      • Click OK in the messagebox.
      • Confirm that InstallShield Wizard Complete dialog appears.
      • Click Finish.
      • Confirm that ID Shield Office is not installed.

    • Initial Installation
      • Prepare a clean machine without ID Shield Office installed before.
      • Login in as privileged user.
      • Insert installation CD into CD-ROM.
      • Confirm that ID Shield Office splash screen appears.
      • Confirm that Welcome to the InstallShield Wizard for ID Shield Office dialog appears and click Next.
      • Select I accept the terms of the license agreement radio button on License Agreement dialog and click next.
      • Confirm that the Select Options dialog appears and click next with its default options.
      • Confirm that the Setup Status dialog appears.
      • Confirm that a dialog asking the user Would you like to set ID Shield as default printer for current user? This will make this software easier to use. does not appear.
      • Confirm that a status bar appears for installation of the Virtual printer.
      • Confirm that InstallShield Wizard dialogs appears and click finish.
      • Confirm that a Yes/No messagebox appears to request a license file.
      • Click Yes
      • Confirm that ID Shield Office install wizard is ended gracefully.
      • Confirm that the UserLicense.exe and the corresponding help pages are provided to help the user go through the licensing process.
      • Confirm that following rule files are installed in the C:\Program Files\Extract Systems\IDShieldOffice\Rules\DataTypes folder:
        • Account numbers - Credit and debit card numbers\Clues.dat.ese
        • Account numbers - Credit and debit card numbers\Data.dat.ese
        • Account numbers - Other account numbers\Clues.dat.ese
        • Account numbers - Other account numbers\Data.dat.ese
        • Account numbers - Savings, checking, and bank account numbers\Clues.dat.ese
        • Account numbers - Savings, checking, and bank account numbers\Data.dat.ese
        • Driver's license numbers\Clues.dat.ese
        • Driver's license numbers\Data.dat.ese
        • E-mail addresses\Clues.dat.ese
        • E-mail addresses\Data.dat.ese
        • Federal tax identification numbers\Clues.dat.ese
        • Federal tax identification numbers\Data.dat.ese
        • Social security numbers\Clues.dat.ese
        • Social security numbers\Data.dat.ese

      • Confirm that shortcuts on desktop and launch bar are provided
      • Confirm following items are included in Extract Systems group in the Start menu:
        • ID Shield Office
        • ID Shield Office Help
        • User License
        • View Exception Log

      • Confirm that CD drive icon is the ID Shield Office icon
      • Open CD drive and confirm that the following files are available:
        • An installation help file: IDShieldOffice_install.chm
        • autorun.inf
        • IDShield.ico
        • readme.txt
        • Setup.exe

      • Generate a license file and copy it to C:\Program Files\Extract Systems\CommonComponents
      • Launch IDSO and confirm that Toolbars are not stacked in 4 rows.
      • Confirm that Create angular redaction and Create rectangular redaction toolbar buttons are separated.
      • Launch View Exception Log and confirm that either
        • An Error 2, file not found message is presented if the exception log is not available, OR
        • UCLIDException.uex is opened by default.

    • Installation - repair
      • Confirm that ID Shield Office has been installed.
      • Delete the ID Shield Office executable file.
      • Confirm that ID Shield Office does not start.
      • Insert ID Shield Office installation CD into CD-ROM.
      • Confirm that Welcome dialog appears.
      • Select Repair radio button and then click Next.
      • Confirm that the Setup Status dialog appears.
      • Confirm that the InstallShield Wizard Complete dialog appears and click Finish.
      • Confirm that ID Shield Office starts.

    • Downgrade Installation
      • Confirm that ID Shield Office has been installed.
      • Insert an installation CD for an earlier version of ID Shield Office. Alternatively, Setup.exe for an earlier version can be run from the network.
      • Confirm that Another version of 'ID Shield Office' is already installed on this machine. The setup can not continue. error dialog appears.
      • Click OK in the dialog.

    • Upgrade Installation
      • Confirm that ID Shield Office has been installed.
      • Insert an installation CD for a newer version of ID Shield Office. Alternatively, b>Setup.exe for a newer version can be run from the network.
      • Confirm that Another version of this product is already installed. To configure or remove the existing version of this product, use Add/Remove programs on the control panel. dialog appears and then close OK.
      • Confirm that Installation terminates.
      • Confirm that the Current version is not removed.

    • Uninstallation
      • Control Panel Uninstallation
        • Open Control Panel and click Add or Remove Programs.
        • Select ID Shield Office and click Remove.
        • Confirm that the screen is not frozen during uninstallation.
        • Confirm that Extract Systems folder in the Start menu is removed.
        • Confirm that ID Shield Office in the installation folder is removed.
        • Confirm that the ID Shield Printer is removed.

      • Setup Program Uninstallation
        • Click the ID Shield Office setup.exe file in the CD drive.
        • Select Remove radio button in the Welcome dialog and click next.
        • Confirm that Do you want completely remove the selected application and all of its features? dialog appears and click No.
        • Confirm that the Welcome dialog reappears and click Cancel.
        • Confirm that the Exit Setup dialog appears and click Yes.
        • Confirm that InstallShield Wizard Complete dialog appears and click Finish.
        • Confirm that ID Shield Office is not removed.
        • Confirm that the ID Shield Printer is not removed.

        • Insert ID Shield Office installation CD into CD-ROM.
        • Select Remove radio button in the Welcome dialog and click next.
        • Confirm that Do you want completely remove the selected application and all of its features? dialog appears and click Yes.
        • Confirm that the screen is not frozen during uninstallation.
        • Confirm that Uninstall Complete dialog appears and click Finish.
        • Confirm that Extract Systems folder in the Start menu is removed.
        • Confirm that ID Shield Office in the installation folder is removed.
        • Confirm that the ID Shield Printer is removed.

  • Licensing
    • No License
      • Remove or rename any existing license file to a different extension in the CommonComponents folder.
      • Start ID Shield Office from start menu.
      • Confirm that a Message box appears to request a license file.
      • Click OK
      • Confirm that UserLicense.exe is launched.
      • Confirm that the corresponding Help file page is displayed.
      • Confirm that ID Shield Office is not launched.

    • Evaluation License
      • Copy an evaluation license file into the CommonComponents folder.
      • Start ID Shield Office from Start menu.
      • Confirm that ID Shield Office starts.
      • Click Help - About ID Shield Office....
      • Confirm that the About ID Shield Office dialog appears.
      • Confirm that a license expiration date is provided in the license information region.

    • Full License
      • Copy a full license file into the CommonComponents folder.
      • Start ID Shield Office from Start menu.
      • Confirm that ID Shield Office starts.
      • Click Help - About ID Shield Office....
      • Confirm that the About ID Shield Office dialog appears.
      • Confirm that Permanently licensed is provided in the license information region.

    • Evaluation/Full License
      • Copy both license files into the CommonComponents folder.
      • Start ID Shield Office from Start menu.
      • Confirm that the ID Shield Office starts.
      • Click Help - About ID Shield Office....
      • Confirm that the About ID Shield Office dialog appears.
      • Confirm that the Permanently licensed is provided in the license information region.
