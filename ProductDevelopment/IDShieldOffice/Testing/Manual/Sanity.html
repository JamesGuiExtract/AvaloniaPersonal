  • Preparation

    • Copy I:\Common\Testing\Product Testing\IDShieldOffice\Images\ to local drive.
    • Copy I:\Common\Engineering\ProductTesting\ManyPage Images to local drive.
    • Generate Full/Evaluation license in local drive.

  • Full License
    • File Input/Output
      • Configuration
        • Launch IDSO and open user preference dialog.
        • Switch to Output files tab
        • Check Use the following expression as default output path for redacted files

      • Open/Save TIFF
        • Open FindTextTest.tif image file in local folder.
        • Click Create rectangular redaction button and draw a rectangular redaction.
        • Confirm that the redaction is filled with black.
        • Draw another rectangular redaction.
        • Click Select redactions and Objects and select the drawn redaction.
        • Change FillColor from Black to White in the properties windows.
        • Click Bates# to apply Bates number.
        • Click Data type finder to display the dialog
        • Check all the checkboxes and Click Find next to highlight all the clues.
        • Close the dialog.
        • Navigate to the second page.
        • Click Save toolbar button and accept default settings.
        • Confirm that the FindTextTest.redacted.tif and FindTextTest.tif.idso are output and located in the same local folder.
        • Verify that FindTextTest.redacted.tif includes the specified redactions and that the white redaction has a black border.
        • Reopen FindTextTest.tif image file in local folder.
        • Confirm that all the redactions is loaded.
        • Confirm that all the clues is loaded.
        • Confirm that Bates# is loaded.

      • Open/Save PDF
        • Open FindTextTest.pdf image file in local folder.
        • Click Create rectangular redaction button and draw a rectangular redaction filled with black.
        • Draw another rectangular redaction filled with white.
        • Click Bates# to apply Bates number.
        • Click Data type finder to display the dialog
        • Check all the checkboxes and Click Find next to highlight all the clues.
        • Close the dialog.
        • Click Save toolbar button and save the image as PDF format.
        • Confirm that the FindTextTest.redacted.pdf and FindTextTest.pdf.idso are output and located in the same local folder.
        • Verify that FindTextTest.redacted.pdf includes the specified redactions and that the white redaction has a black border.
        • Reopen FindTextTest.pdf image file in local folder.
        • Confirm that all the redactions is loaded.
        • Confirm that all the clues is loaded.
        • Confirm that Bates# is loaded.

      • Open TIFF and save as PDF
        • Open FindTextTest.tif.idso image file saved above.
        • Click File - Save as... menu.
        • Specify save as type as PDF files and file name in the Save Document dialog.
        • Confirm that the saved file is output to specified locations with specified name.
        • Verify that output files include the specified redactions.
        • Verify that output files exclude all the clues.
        • Verify that output files include the Bates#.

      • Open PDF and save as TIFF
        • Open FindTextTest.pdf.idso image file saved above.
        • Click File - Save as... menu.
        • Specify save as type as TIFF files and file name in the Save Document dialog.
        • Confirm that the saved file is output to specified locations with specified name.
        • Verify that output files include the specified redactions.
        • Verify that output files exclude all the clues.
        • Verify that output files include the Bates#.

    • Print
      • Print to a Printer
        • Open previously saved FindTextTest.tif.idso image file in local folder.
        • Uncheck all the checkboxes in the layers window to turn off all objects' visibility.
        • Click File - Page setup menu.
        • Confirm that Page setup dialog appears.
        • Change Paper size to A3 and orientation to Landscape.
        • Change left/right/bottom/top margins to 1 inch.
        • Close Page Setup dialog.
        • Click File - Print preview menu.
        • Confirm that Paper size and Orientation are correct.
        • Confirm that Zoom works.
        • Confirm that Two pages button works and close print preview dialog.
        • Click Print image button
        • Confirm that Print dialog appears.
        • Specify ID Shield virtual printer and click Print.
        • Confirm that ID Shield Office has disallowed to print to ID shield printer.Please select another printer message appears.
        • Click OK and return back to Print dialog.
        • Select a physical printer and uncheck the print to file checkbox.
        • Confirm that the printed image includes the specified redactions.
        • Confirm that the printed image includes the Bates number.
        • Confirm that the printed image doesn't include the Clues.
        • Check all the checkboxes in the layers window.
        • Close the image file.

    • Find and Redact Words / Patterns
      • Open FindTextTest.tif image file in local folder.
      • Add this, test and 111 as entries to the Find and redact - Words/patterns edit box.
      • Click the Find next button and confirm that the first word This is highlighted by green with a border.
      • Click the Redact button and confirm that the highlight changes to a redaction and that the word test is highlighted.
      • Click the Redact button again and confirm that the highlight changes to a redaction and that test within the word testing is highlighted.
      • Click the Find next button and confirm that the word this is highlighted.
      • Click the Reset button and confirm that the redactions are retained.
      • Zoom in on the redacted This text.
      • Click the Find next button and confirm that the redacted word This is highlighted again and that the window remains zoomed in.
      • Click the Find next button and confirm that the test redaction is highlighted.
      • Click the Find next button and confirm that test is highlighted.
      • Click the Find next button and confirm that this is highlighted.
      • Pan around in the image and confirm that the This redaction is still redacted. Also confirm that the test highlight that is part of testing is no longer highlighted and is NOT redacted.
      • Click the Redact all button and confirm that all the this, test and 111 strings are redacted including the item that had previously not been redacted.

      • Close and reopen the FindTextTest.tif test image.
      • Modify the test entry to \btest\b and remove the other entries.
      • Modify the dialog settings to check the Match case and Treat as regular expression checkboxes.
      • Click the Find next button and confirm that the word test is the first item highlighted.
      • Click the Redact button and confirm that the highlight changes to a redaction and that the word test is highlighted.
      • Confirm that test within testing was NOT highlighted or redacted.
      • Close image file without saving.
