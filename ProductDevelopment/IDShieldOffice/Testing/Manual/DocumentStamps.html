Document Stamp Testing Checklist

  • Configuration
    • Can specify a bitmap to be used as a document stamp
    • Can specify text to be used as a document stamp. Font style and size can also be specified

  • Functionality
    • User can move the stamp's location after placement
    • Resolution and color depth of stamp bitmap shall be scaled down to match document
    • Stamp bitmap shall be smaller than page size
    • Document stamp can be deleted after application
    • If document stamp extends beyond page bounds, only the visible portion shall be displayed on the page
    • Document stamp can be placed manually
      • Stamp location specified by mouse position
      • Position is anchored to one of nine points on bitmap or text stamp ( top, center, bottom ) + ( left, center, right )
      • Stamp is displayed during manual drag operation

      • Stamp location can be defined using a corner-based anchor point and a fixed offset
