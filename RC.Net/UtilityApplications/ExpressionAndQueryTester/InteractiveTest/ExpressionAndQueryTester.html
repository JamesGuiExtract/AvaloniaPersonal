ExpressionAndQueryTester - Testing Checklist

NOTE: Additional query and expression examples can be found in the Data queries.doc document in .\Engineering\RC.Net\DataEntry\Core\Requirements

  • User Interface

    • Confirm that the window caption is Expression And Query Tester.
    • Confirm that a VOA filename (data to test) edit box is provided.
    • Confirm that an associated browse button is provided.
    • Confirm that the edit box is updated with the selected VOA filename.
    • Confirm that another button is provided that brings up the VOA File Viewer.
    • Data Scorer
      • Confirm that a (default) Test RSD data scorer expression radio button is provided.
      • Confirm that an Attribute(s) to score edit box is provided.
      • Confirm that an RSD Filename edit box is provided.
      • Confirm that an associated browse button is provided.
      • [DotNetRC #861] Confirm that the Open dialog allows browsing for *.RSD or *.RSD.ETF files.
      • Confirm that the edit box is updated with the selected filename.
      • Confirm that the Test button is enabled only when all three of the following fields are populated.
        • VOA filename
        • RSD filename
        • Expression
    • Data Scorer
      • Confirm that a Test data query radio button is provided.
      • Confirm that a Root attribute edit box is provided.
      • Confirm that a read-only Data source edit box is provided.
      • Confirm presence of an associated tag selector button containing the following options.
        • Open data file
        • Copy connection data type to Clipboard
      • Confirm presence of an associated Configure connection button.
      • Confirm that clicking the button displays a Connection Properties dialog.
    • Confirm that a large, unlabeled edit box is provided for the user to define either a data scorer expression or a data query.

  • Data Scorer Performance

    • Setup
      • Set the VOA filename to AllText.voa.
      • Clear the Attribute(s) to score field.
      • Set the RSD Filename to TestCases.rsd.
    • Expressions
      • Confirm that #Wayne.count() yields:
        • Overall score = 6
        • 6 individual Wayne attributes
      • Confirm that #Short.count() yields:
        • Overall score = 13
        • 13 individual Short attributes
      • Confirm that #Short.sum() yields:
        • Overall score = 351
        • 13 individual Short attributes
      • Confirm that #Short.distinct().count() yields:
        • Overall score = 11
        • 11 individual Short attributes
      • Confirm that #Short.min() yields:
        • Overall score = 1
        • 13 individual Short attributes
      • Confirm that #Short.max() yields:
        • Overall score = 88
        • 13 individual Short attributes
      • Confirm that #Short.average() yields:
        • Overall score = 27
        • 13 individual Short attributes
      • Confirm that #Short.sort()[0] yields:
        • Overall score = 1
        • 13 individual Short attributes
      • Confirm that #Short.distinct().sort().reverse()[1] yields:
        • Overall score = 51
        • 13 individual Short attributes
    • Operations
      • Addition: Confirm that #Short.count() + #Wayne.count() yields:
        • Overall score = 19
        • 13 individual Short attributes
        • 6 individual Wayne attributes
      • Subtraction: Confirm that #Short.count() - #Wayne.count() yields:
        • Overall score = 7
        • 13 individual Short attributes
        • 6 individual Wayne attributes
      • Multiplication: Confirm that #Short.count() * #Wayne.count() yields:
        • Overall score = 78
        • 13 individual Short attributes
        • 6 individual Wayne attributes
      • Division: Confirm that #Short.max() / #Wayne.count() yields:
        • Overall score = 14
        • 13 individual Short attributes
        • 6 individual Wayne attributes
      • Modulus: Confirm that #Short.max() / #Short.count() yields:
        • Overall score = 10
        • 13 individual Short attributes
      • Exponent: Confirm that #Short.max() ^ #Short[6] yields:
        • Overall score = 681472
        • 13 individual Short attributes

      • Error: Confirm that #Short.max() ^ #Short[5] yields:
        • ELI33843: RSD data scorer failed exception
        • Confirm that an ELI21061: Value was either too large or too small for an Int32 exception is also included.

  • Data Query Performance

    • Setup
      • Set the VOA filename to All.voa.
      • Clear the Root attribute field.
      • Click the Configure connection button
        • Use the Change... button if necessary to set the Data source to Microsoft SQL Server (SqlClient).
        • Choose (local) or another appropriate Server name.
        • Choose Demo_IDShield or another appropriate database name. Note that the database should already have content.
        • Click the Test Connection button before confirming the settings.
        • Confirm that the Data source and Connection string fields include meaningful information.
    • Expressions and Queries
      • Confirm that <Attribute>/Short</Attribute> yields:
        • 13 individual attributes where each attribute is a two-character numeric string.
      • Confirm that <Expression><Attribute Type='int[]'>/Short</Attribute></Expression> yields:
        • 13 individual attributes where each attribute is a number. Note that zero-padding should no longer be seen.
      • Confirm that <SQL>SELECT TOP 4 Filename FROM FAMFile</SQL> yields:
        • 4 individual attributes where each attribute is a fully-qualified path to a Demo_IDShield test image (assuming that the Demo_IDShield database was used during setup.
      • Confirm that <Regex Pattern='[\d]+'>The answer to the ultimate question of life is 42</Regex> yields:
        • 42
