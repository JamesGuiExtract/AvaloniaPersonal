Pagination Utility - Testing Checklist

  1. Basic Operation

    • Confirm that PaginationUtility.exe exists in the CommonComponents folder for a FLEX Index installation.
    • Confirm that PaginationUtility.exe exists in the CommonComponents folder for an ID Shield installation.
    • Confirm that PaginationUtility.exe exists in the CommonComponents folder for a LabDE installation.
    • Confirm that PaginationUtility.exe requires a special license file.
      • Confirm that a meaningful exception is displayed if an appropriate license is not present.

  2. General User Interface

    • Prepare a set of images for testing. Files in K:\Common\Engineering\Sample Files\UW Transplant\Set003\Images\Rasterized\tifs will work well.
    • Modify Settings if needed to load the testing images.

    • Main and Image Window
      • Confirm that Pagination Utility is the window caption.
      • Confirm that the following menu items are defined
        • File - Restart
        • File - Exit
        • Tools - Settings
        • Help - About
      • Confirm that an Output document(s) toolbar button is defined immediately below the menu bar.
      • Confirm that an edit box is defined next to the toolbar button.
      • Confirm that a browse button is defined next to the edit box.
        • Confirm that the toolbar button, edit box, and browse button are each only enabled when an entire document's collection of image pages is selected.
        • Confirm that only the Output document(s) toolbar button is enabled if if the selected pages complete more than one output document.
      • Confirm that a label is visible next to the browse button.
        • Confirm that the label is only visible when 1 entire document's collection of image pages is selected.
        • Confirm that the label displays the total number of pages to be saved to the selected document.

      • Confirm that the following toolbar buttons are available for the Image Window. Also confirm proper functionality for each button and shortcut.
        • Zoom window (Z)
        • Pan (A)
        • Zoom in (F7, Ctrl++)
        • Zoom out (F8, Ctrl+-)
        • Zoom previous (Back, Alt+Left)
        • Zoom next (Alt+Right)
        • Fit to page (P)
        • Fit to width (W)
        • Rotate counterclockwise (Ctrl+Shift+R)
        • Rotate clockwise (Ctrl+R)

    • Context Menu
      • Select a single page that does not constitute all pages necessary for an output document.
      • Confirm that the following context menu items are available.
        • Cut item
        • Copy item
        • Delete item
        • Insert copied item
        • Insert document separator
        • Output document
      • Confirm that Insert copied item is enabled only if a single page has been copied to the Clipboard.
      • Confirm that Insert copied items is enabled only if multiple pages have been copied to the Clipboard.
      • Confirm that Output document is disabled.
      • Confirm that Append document separator is available and enabled when right-clicking outside of the thumbnail area.
        • Confirm that a document separator is added after the last displayed page.

    • Keyboard Navigation
      • Confirm that Up Arrow moves up one row in the list of pages.
      • Confirm that Up Arrow automatically moves to the bottom row of the list if starting from the top row.
      • Confirm that Down Arrow moves down one row in the list of pages.
      • Confirm that Down Arrow automatically moves to the top row of the list if starting from the bottom row.
      • Confirm that Left Arrow moves left one column in the list of pages.
      • Confirm that Left Arrow automatically moves to the last column on the previous row of the list if starting from the leftmost column.
      • Confirm that Left Arrow automatically moves to the last column on the last row of the list if starting from the leftmost column of the first row.
      • Confirm that Right Arrow moves right one column in the list of pages.
      • Confirm that Right Arrow automatically moves to the first column on the next row of the list if starting from the rightmost column.
      • Confirm that Right Arrow automatically moves to the first column on the first row of the list if starting from the last column of the last row.
      • Confirm that TAB moves right one column in the list of pages.
      • Confirm that SHIFT+TAB moves left one column in the list of pages.
      • Confirm that Ctrl+TAB selects all of the pages in the document associated with the selected page.
      • Confirm that a subsequent Ctrl+TAB selects all of the pages in the next document.
      • Confirm that PageUp moves backward through the list of pages.
      • Confirm that PageDown moves forward through the list of pages.

  3. Settings

    • Confirm that the Pagination Utility Settings dialog is displayed after clicking the Settings toolbar button.
    • Confirm that the Pagination Utility Settings dialog is displayed after clicking the Tools - Settings menu item.
    • Confirm that the dialog includes a Specify input directory edit box and associated browse button.
      • Confirm that the browse button displays a Browse For Folder dialog and replaces any current selection in the edit box.

    • Confirm that the dialog includes a spinner control to specify how many pages are to be kept in the input pane (if available).
      • Confirm that the maximum number of pages is 999.
    • Confirm that the dialog includes a Specify output directory edit box and associated browse button.
      • Confirm that the browse button displays a Browse For Folder dialog and replaces any current selection in the edit box.
    • Confirm that the dialog includes a Randomize output filename checkbox.
    • Confirm that the dialog includes a Preserve sub-folder structure in output directory checkbox.
      • Confirm that the checkbox is enabled only if the Include files from all subdirectories checkbox is checked.
    • Confirm that the dialog includes an After handling all pages of an input document group box.
      • Confirm that the group box contains a Move the document to the directory: radio button.
      • Coinfirm that the group box contains an associated edit box and browse button.
      • Confirm that the edit box and browse button are enabled only when the radio button is selected.
      • Confirm that the browse button displays a Browse For Folder dialog and replaces any current selection in the edit box.
      • Confirm that the group box contains a Delete the document radio button.
    • Confirm that the dialog includes an Export Settings button.
      • Confirm that a Save dialog is displayed.
      • Confirm that settings are saved in a config file.
    • Confirm that the dialog includes an Import Settings button.
      • Confirm that an Open dialog is displayed.
      • Confirm that settings are available in config files.
