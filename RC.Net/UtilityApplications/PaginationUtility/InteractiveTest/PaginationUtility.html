Pagination Utility - Testing Checklist

  1. Basic Operation

    • Confirm that Pagination Utility is available as a Start Menu item under FLEX Index - Utilities.
    • Confirm that Pagination Utility is available as a Start Menu item under ID Shield - Utilities.
    • Confirm that Pagination Utility is available as a Start Menu item under LabDE - Utilities.
    • Confirm that PaginationUtility.exe requires a special license file.
      • Confirm that a meaningful exception is displayed if an appropriate license is not present.
    • [DotNet #921] Confirm that PaginationUtility /? displays a meaningful usage message.

  2. General User Interface

    • Prepare a set of images for testing. Files in K:\Common\Engineering\Sample Files\UW Transplant\Set003\Images\Rasterized\tifs will work well.
    • Modify Settings if needed to load the testing images.

    • Main and Image Window
      • Confirm that Pagination Utility is the window caption.
      • Confirm that the following menu items are defined and work correctly.
        • File - Load next document
        • File - Output selected document(s)
        • File - Restart

        • Edit - Cut
        • Edit - Copy
        • Edit - Delete

      • Confirm that an Output document(s) (Ctrl+S) toolbar button is defined immediately below the menu bar.
      • Confirm that an edit box is defined next to the toolbar button.
      • Confirm that a browse button is defined next to the edit box.
        • Confirm that the toolbar button, edit box, and browse button are each only enabled when an entire document's collection of image pages is selected.
        • Confirm that only the Output document(s) toolbar button is enabled if if the selected pages complete more than one output document.
      • Confirm that a label is visible next to the browse button.
        • Confirm that the label displays the page number of the single, selected page if only one page is selected from a multi-page document.
        • Confirm that the label displays the total number of pages to be saved to the selected document if all pages are selected.

      • Confirm that the following toolbar buttons are available for the Image Window. Also confirm proper functionality for each button and shortcut.
        • Zoom window (Z)

      • Confirm that the following context menu items are available.
        • Cut
        • Copy
        • Delete
        • Insert copied item(s)
        • Toggle document separator
      • Confirm that Insert copied item is enabled only if a single page has been copied to the Clipboard.
      • Confirm that Insert copied items is enabled only if multiple pages have been copied to the Clipboard.

    • Keyboard Navigation
      • Confirm that Up Arrow moves up one row in the list of pages.
      • Confirm that Down Arrow moves down one row in the list of pages.
      • Confirm that SHIFT+Up Arrow moves up one row in the list of pages and selects each intervening page.
      • Confirm that SHIFT+Down Arrow moves down one row in the list of pages and selects each intervening page.
      • Confirm that Left Arrow moves left one column in the list of pages.
      • Confirm that Left Arrow automatically moves to the last column on the previous row of the list if starting from the leftmost column.
      • Confirm that Right Arrow moves right one column in the list of pages.
      • Confirm that Right Arrow automatically moves to the first column on the next row of the list if starting from the rightmost column.
      • Confirm that the arrow keys do not automatically wrap around when already at the top or bottom row.
      • Confirm that TAB moves right one column in the list of pages.
      • Confirm that SHIFT+TAB moves left one column in the list of pages.
      • Confirm that Ctrl+TAB selects all of the pages in the document associated with the selected page.
      • Confirm that a subsequent Ctrl+TAB selects all of the pages in the next document.
      • Confirm that PageUp moves backward through the list of pages.
      • Confirm that PageDown moves forward through the list of pages.
      • Confirm that ESCAPE unselects the current selection.
      • Confirm that SPACE toggles a document separator before the selected page.
