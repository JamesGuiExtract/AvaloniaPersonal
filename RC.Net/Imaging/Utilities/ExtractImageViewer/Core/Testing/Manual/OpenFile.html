Image Viewer Open Files Testing Checklist

    Manual Tests

    • [DotNet #63] Confirm that drag-and-drop of an image into the Image Viewer successfully opens the image.
    • [DotNet #45] Confirm that a wait cursor is displayed while the Image Viewer opens an image.
    • [DotNet #64] Confirm that specifying an image name on the command line successfully opens the image.
    • Confirm that clicking the left side of the Open Image toolbar button displays the Open Image File dialog.
    • Confirm that clicking the right side of the Open Image toolbar button displays a drop-down list of recently opened files.

    • [DotNet #49] Large image file and MRU list behavior
      • Open K:\Common\Engineering\Sample Files\TWG\Set001\moi%20sample[2].pdf.
      • Close the Image Viewer.
      • Open a new instance of the Image Viewer and confirm that the previous file is present in the MRU list.
      • Open a different image - perhaps one from the Redaction Demo.
      • Use the MRU list to open the first file.
      • Confirm that the image opens and that the MRU list menu closes.

    • [DotNet #109] Display previous image on Open failure
      • Open an image from the Redaction Demo.
      • Open a different image from the Redaction Demo.
      • Temporarily rename the first image.
      • Use the MRU list and attempt to open the first file.
      • Confirm that a useful error message is displayed and that the second image remains open.

    • MRU list from File menu
      • Open an image from the Redaction Demo.
      • Confirm that the new image has been added to the Recent files menu under the File menu.
      • Open a different image.
      • Use the Recent files menu to open the first file.
      • Confirm that the image opens.

    • MRU list from Open image toolbar button
      • Open an image from the Redaction Demo.
      • Confirm that the new image has been added to the MRU list drop-down menu.
      • Open a different image.
      • Use the MRU list drop-down menu to open the first file.
      • Confirm that the image opens.
