Image Viewer Miscellaneous Testing Checklist

    [DotNet #38] Mousewheel Scroll / Zoom - Manual Tests

    • Open an image from the Redaction Demo.
    • Zoom in several steps.
    • Confirm that scrolling the mousewheel up scrolls the image up.
    • Confirm that scrolling the mousewheel down scrolls the image down.

    • Confirm that scrolling the mousewheel up while pressing Shift scrolls the image left.
    • Confirm that scrolling the mousewheel down while pressing Shift scrolls the image right.

    • Confirm that scrolling the mousewheel up while pressing Control zooms the image in.
    • Confirm that scrolling the mousewheel down while pressing Control zooms the image out.

    • Click and drag to begin defining a zoom region.
    • Scroll the mousewheel while continuing to define the zoom region.
    • Confirm that the mousewheel has no effect.
    • Keep the image open for the following tests.

    [DotNet #69] Context Menu - Manual Tests

    • Zoom in several steps.
    • Right-click to display the context menu.
    • Confirm that Zoom window is available.
    • Select Zoom window and confirm that a zoom region can be defined and that the Image Viewer properly displays the new region.

    • Confirm that Pan is available.
    • Select Pan and confirm that it works properly.

    • Confirm that Zoom previous is available.
    • Select Zoom previous and confirm that it works properly.

    • Confirm that Zoom next is available.
    • Select Zoom next and confirm that it works properly.

    • Confirm that Highlighter is available.
    • Select Highlighter and confirm that a rectangular highlight can be created.
    • Select Highlighter and confirm that an angular highlight can be created.

    Fit Mode - Manual Tests

    • [DotNet #41] - Works for PDF images
      • Open a PDF image.
      • Zoom in several steps.
      • Click the Fit to page toolbar button.
      • Confirm that the entire image is now visible.

    Shortcuts - Manual Tests

    • Shortcuts disabled during interactive mouse events
      • Open an image from the Redaction Demo.
      • Zoom in several steps.
      • Click P as a shortcut to enable Pan mode.
      • Click Z as a shortcut to enable Zoom mode.
      • Click and drag to begin defining a zoom region.
      • Click P while continuing to define the zoom region.
      • Confirm that the shortcut key has no effect.

    Status Bar - Manual Tests

    • Status text
      • Confirm that no status text is provided without an open image.
      • Open an image and confirm that status text is provided for a user action, image resolution and mouse position.
      • Confirm that status text for user action does not change and is meaningful when selecting the following toolbar buttons.
        • Open image
        • Print image
        • Zoom in
        • Zoom out
        • Zoom previous
        • Zoom next
        • Fit to page
        • Fit to width
        • Rotate counterclockwise
        • Rotate clockwise
        • Go to first page
        • Go to previous page
        • Go to next page
        • Go to last page
      • Confirm that status text for user action does change and is meaningful when selecting the following toolbar buttons.
        • Zoom window
        • Pan
        • Angular highlight
        • Rectangular highlight
        • Select highlight

      • [DotNet #41] Registry setting controls display of position percentage
        • Use RegEdit to modify the following registry entry: HKCU\Software\Extract Systems\InputFunnel\InputReceivers\SpotRecIR\DisplayPercentageEnabled
        • With the setting at 0, confirm that the status bar does not provide percentage information.
        • With the setting at 1, confirm that the status bar does provide percentage information.

    Page Navigation - Manual Tests

    • *Invalid page number displays error message
      • Open a multi-page image from the Redaction Demo.
      • Select the View - Goto Page menu item.
      • Enter a page number beyond the end of the document.
      • Confirm that a meaningful error message is displayed.

    Pan Tool - Manual Tests

    • Tools menu
      • Open an image.
      • Zoom in several steps so that the image is not completely visible.
      • Select the Pan tool from the Tools menu.
      • Confirm that click and drag allows the user to pan.
    • Toolbar
      • Open an image.
      • Zoom in several steps so that the image is not completely visible.
      • Select the Pan tool from the Toolbar.
      • Confirm that click and drag allows the user to pan.
