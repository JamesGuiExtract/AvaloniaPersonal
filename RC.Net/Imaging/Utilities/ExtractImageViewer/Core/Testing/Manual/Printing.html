Image Viewer Printing Testing Checklist

    Basic Printing - Manual Tests

    • Open a multipage image - TestImage004.tif from the Redaction Demo.
    • Confirm that the current page can be printed.
    • [DotNet #58] Confirm that printing occurs the first time that Ok is clicked.
    • Confirm that a range of pages can be printed.
    • Confirm that the entire image can be printed.

    • Begin a print operation but Cancel before printing starts.
    • Confirm that printing does not occur.
    • Keep the image open for the following tests.

    Printing Highlights - Manual Tests

    • Create several highlights on the image.
    • [DotNet #56] Confirm that the highlights are printed as black redactions.
    • Rotate the image.
    • Print the rotated image and confirm that the printed image is also rotated.
    • Confirm that the highlights are also printed with appropriate rotation applied.
    • Close the image.

    Printing Annotations - Manual Tests

    • Open an image that contains annotations - FourPagesWithAnnoA.tif from I:\Common\Testing\Annotation_Images.
    • Print the image.
    • Confirm that the annotations are printed.
    • Rotate the image.
    • Print the rotated image and confirm that the printed annotations are also rotated.
    • Close the image.

    Printing Rotated View Perspectives - Basic

    • Open an image with an unusual view perspective - Image23_VP3.tif from I:\Common\Testing\Rotated_Images.
    • Print the image.
    • Confirm that the image is printed with the displayed view perspective.
    • Close the image.

    Printing Rotated View Perspectives - Special License

    • Create a special license file with RotatedImages_And_Annotation_Feature not licensed. This will require a temporary modification removing ID 2076 from the FlexIndexAndIDShieldCore package.
    • Open an image from the Redaction Demo.
    • Rotate the image.
    • Print the image.
    • Confirm that the image is printed with the displayed rotation.

    • Create several highlights on the image.
    • Print the image.
    • Confirm that the highlights are also printed with appropriate rotation applied.
    • Close the image.

    • Open an image that contains annotations - FourPagesWithAnnoA.tif from I:\Common\Testing\Annotation_Images.
    • Rotate the image.
    • Print the rotated image and confirm that the printed annotations are also rotated.
    • Close the image.
    • Delete the special license file and confirm that any modifications to network DAT files are corrected.
